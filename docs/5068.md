# CES 2018:真正的进步，真正的进步，真正的问题



CES 2018\. All photos by author unless noted.

CES 很大，也很重要。CES 2018 很有趣，但不是惊天动地的，但如果你认为它会惊天动地，那么今年你会失望，就像过去任何展览一样。

在经历了 3 天 87207 步之后，我提出了关于 CES 上展示的产品和技术方向的五点看法。

*   **声音**。所有平台和设备都在语音控制方面投入了大量资金——几乎每个展台都有语音控制。这主要是由亚马逊 Alexa 的势头和易于访问的 SDK(可以让平凡的事情变得更有趣)以及谷歌令人难以置信的快速追随者投资(显示了福音主义的力量)推动的。Siri 和 Cortana 都没有太多的存在。正如本文后面所描述的，现在还为时过早，但简而言之，现在选择一个赢家还为时过早，重要的是，如果一个赢家真的出现了，定义它可能意味着什么还为时过早。真正的问题是，语音，尤其是在消费者家庭中，是否会像 Windows、Android 或 HDMI 一样获得平台回报？
*   **家用电子产品**。在娱乐、运营和安全方面,“未来之家”似乎在可用性、合意性和实用性方面取得了重大进展。CES 就是这样发展的——一年前看起来很尴尬并且“为有钱的懒人准备”的东西，似乎一夜之间变得合理、实用，并且对每个人都可用。CES 是一个见证这种演变的绝佳场所——从旋钮到按钮到触摸屏的演变，到从断开连接到有线到无线的复杂演变，再到模拟到数字的大规模演变。在这一年，我认为可以公平地说，经营家居的领导权已经从日本和美国转移到了韩国(三星和 LG)，中国企业正努力成为第三大玩家(并拥有一个垄断的本土市场)。
*   **汽车**。在消费电子展上，汽车是个时髦的东西。老派的“车载娱乐”北大厅已经被来自汽车制造商、车载芯片制造商(如英伟达)和其他自动/电动交通工具的大规模创新展台所取代。从许多方面来说，这对于汽车来说都是一个奇怪的地方，因为与会者都不是业内人士，汽车制造商的主要声音似乎是“看，我们正在像科技公司一样创新”，而不是展示任何实际产品。这个领域没有太多新闻，有很多大公司的“创新谈话”，大部分都是没有内容的。
*   **没有电线。**CES 上没有电线。唯一存在的电线是从你的高速互联网到路由器。之后一切都是无线的。很高兴看到。或许 HDMI 是一个例外，但电视制造商不再谈论它，因为他们基本上希望你使用他们的应用程序/控制中心来访问互联网服务。当我想到以前专门用于电线、电线管理、连接器、插头、集线器、路由器、扩展器等的所有展位(平方英尺)时，这是一个关于“中断”的教训。也许这是该节目最大的“物理”变化。至于无线，Wi-Fi(和音频，蓝牙)是基础设施的一部分，甚至不是像过去那样是节目的一个独立部分(HDMI 仍然有一个专用区域)。
*   **但是在终端仍然有太多的技术…** 在过去 3-5 年的 CES 上，对我来说，一个始终如一的主题就是“为什么在边缘有这么多的复杂性”？你需要多少张 SIM 卡？有多少 ARM 处理器运行修改后的 Android 或 Linux 操作系统？有多少设备需要完整的用户界面？现在，有多少有源监听麦克风？这种情况继续存在。我认为，特别是在需要持续 10 年以上的汽车和家用设备中，边缘复杂性是交易的破坏者(在新的和现有的端点中)。我可能想要一个很酷的立式办公桌，但我肯定不想要一个运行内置 Android 的平板电脑。我喜欢语音控制设备，但我不希望它的麦克风和语音运行时间与灯光或安全系统中的另一个麦克风和语音运行时间不同。我希望我的车里有一个屏幕，但如果我不能使用我手机上的东西，或者如果我必须切换屏幕来改变气候控制，我就不需要。对我来说，这是 CE 制造商在努力创新**和**差异化时需要解决的最大“问题”。

所有这些都只是提醒我们，CES 几乎涵盖并触及了我们每天所做的一切。CES，要对产品人重要，不需要有下一波颠覆性的技术。CES“只是”需要展示日常生活如何在明年或后年变得更好。我能在不改造房子的情况下安装家庭安全系统吗？我能把我的门锁换成我能追踪回家的孩子的门锁吗？烹饪会有怎样的改变？这些平凡的事情都需要时间，并且每年都有进步——烹饪可以在没有火和热替代的情况下得到改善。我真的不喜欢人们评判会议(或者坦率地说，任何技术公告)的标准是新的东西是否是全新的，是否取代了一切，是否无缝地融入了当今的生活。大多数时候，创新并不是这样发生的。

这份报告只有我一个人。我只是从一个摊位走到另一个摊位，做笔记，拍照片，然后让所有的事情都沉寂几天。我只是一个人，不可能把每件事都做对，也不可能面面俱到。我绝对不会试图与那些覆盖所有新闻事件的优秀团队竞争，比如 The Verge、Mashable、Engadget 等等。我通过产品经理或技术经理的视角来看待 CES，试图了解广阔的前景是如何演变的。只是一个人的看法。

*仅供参考，我在这里提到产品并不是因为我在挑选最佳产品、推荐或投资它们。我提到它们是为了证明更大的观点。这里的一切都只是我，没有其他人参与，也没有议程。*

在对整体环境进行快速总结后，该报告从以下几个方面分析了展示的内容:

*   声音
*   “嘿谷歌”
*   电视和显示器
*   耳机和音频

当然是很多，但我今年用了很多照片，每个部分主要是关于集体观察。如果你想了解产品的细节，肯定有更好的地方——但我肯定花了时间在这些电台玩和听演示。这都是动手。

# CES 到底是什么样的？

CES 是拉斯维加斯最大的年度展会，也是全球排名前五的年度展会。所有真正的大项目都是飞机、汽车和建筑(CEBit 将它们首尾相连)。有趣的是，CES 的规模大约是世界移动通信大会的两倍。

如果你在展会前看看有影响力的人的推文，会有很多“今年不去了”和“CES 已经为我做了”之类的想法。一个好朋友(也是老朋友)甚至说“CES 是新的 COMDEX”，这是一种表明展会已经结束的方式。这让我在登机前感到非常沮丧——我的意思是谁会想去看一场糟糕的演出。

事实是，透视和大型年度贸易展一样古老。总的来说，商业展览很糟糕。我的意思是你走了几英里；你找不到任何人；人们爱出风头；在所有错误的地方都有香烟的烟雾；你累了，饿了，脱水了(在拉斯维加斯，嘴唇干裂是正常的)；很多摊位真的很挤，很恐怖。CES 只是一个大型贸易展会。哦，还有拉斯维加斯。

但是，只要几分钟，你就能看到激情和喧嚣这两样供应过剩的东西。

人们喜欢这种东西。他们热爱自己的工作。处于行业边缘的人带着他们的背包出现，并在他们的背包里塞满了小册子(仍然！)、按钮、笔等等，他们排队等着一杆进洞，坐在原型车里，或者只是走过巨大的有机发光二极管屏幕的走廊。业内人士与他们的合作伙伴、大公司的其他员工联系在一起，他们都有惊人的故事可以讲述，在展厅里与来自世界各地的人(从字面上)就他们的产品进行互动。

这是大厅里的人群。它是肩并肩的。CES 取消了在大厅使用手推车的能力(为了安全)，因为这些真的把事情搞砸了。

For many this is the most exciting place. On the right is the historically most crowded intersection which is Qualcomm and Intel (that positioning itself is kind of entertaining). 喧嚣无处不在。我认为那些持负面观点的人并不总是意识到代表他们企业的人所付出的努力。一家有 A 轮融资的小公司可能会在这个节目上下注，花 50 万美元买一个小摊位，派出最少的核心人员，飞往西南，从 Thrifty 租一辆车，用折叠床分享 Days Inn 房间。这很正常。

大公司可以很容易地派出 1000 多人(高管除外)，他们每天花 4-5 个小时在展位上回答问题，做演示，并担心展位“经理”如何评价他们在创造流量和充分利用数百万美元方面的表现。(哦，他们也有室友！)

销售和营销人员看着他们的预算资金被花掉，并试图找到分销商或与行业作家联系，他们可能会放弃一个关于他们所做的故事。作者知道这一点，但有时会忘记，一个可靠的故事和正确的信息可以决定 CES 上的投资成败。

虽然到处都是竞争，但大多数人并不认为大公司真的在竞争，而是认为他们过得很轻松。就我个人而言，没有什么比安排一个展位更累人的了(或者在我后来的职业生涯中，负责充分利用 1000 平方英尺的空间)。

谷歌几乎用“嘿谷歌”覆盖了拉斯维加斯(不仅仅是世博大厅)。到处都是。着陆标志是一回事，但与人类一起执行是另一回事。左边是非常明显的“嘿谷歌”口香糖吃角子老虎机(一个在北大厅，一个在南大厅)。这个东西有一条线，每小时一次，但更重要的是，在那里工作的谷歌员工的无尽热情超过了顶部——我的意思是，我见过许多公司覆盖 CES，但这是一个令人难以置信的努力。

虽然在幕后和大机器后面可能有 1000-1500 个“嘿，谷歌”的标志在摊位上(总共将近 4000 个摊位)。但作为一家公司，即使你与谷歌合作，你也必须做到这一点，你不能只是做出没有证据支持的声明。这需要一大批人来完成。下面你可以看到我看到的许多对话中的一个，一个穿着连身衣、拿着原色吸盘的谷歌人在讨论“嘿，谷歌”的执行。1995 年，我在国际消费电子展上到处张贴(并要求撤下)“由 Windows 95 驱动”的标志，因此我完全理解这需要付出多大的努力。





On the left is the “Hey Google” gumball machine where you won prizes for correctly answering brain teasers solved by asking Google staffed with Googlers in jumpsuits and Chucks. On the right is a Googler in jumpsuit inquiring about the “Hey Google” signage you can see in the booth.





如果我没有提到 CES 本身就是一个产品，并且是一个在大多数年里在众多变化中看到改进的产品，那我就失职了。作为一个老前辈，看到一些东西移动或改变真的很奇怪——今年三星比往年更靠近中央大厅，所以我很困惑。我不知道如果索尼或英特尔移动会发生什么！

今年有一个很棒的应用程序来配合这个节目。它有强大的搜索和导航功能。你可以输入一个展位号，然后得到步行方向(我猜是用定位信标)。您也可以通过各种其他属性进行搜索。这个应用程序还有很好的通知功能——比如当我站在 LG 的黑暗中时，我收到了停电的警告。





The app for CES was vastly improved over past years. Security is an incredible concern at any big event, especially this year at Las Vegas.





# 声音



Typical sign at a booth saying it works with everything.



虽然去年语音是新的，在许多摊位，它大多没有工作，感觉像一种方法，使旧的演示看到新鲜。今年，声音是一种策略。整个公司都在押注语音作为差异化，目前的两大平台公司亚马逊和谷歌都使出浑身解数。

LG 和三星是制造家用电器终端最多的两家公司，我将在本文稍后介绍。

对于其他公司来说，有两种策略在起作用:

*   通过添加麦克风、处理器等将一个或多个 Alexa 或 Assistant 嵌入到设备中。
*   使用 Wi-Fi 和设备上的现有软件，允许设备由一个或多个控制器产品控制(直接由扬声器或由另一个控制器如家庭安全系统或集线器控制)。

现在有很多挑战。让我从乐观的部分开始。对于家里的绝大多数东西来说，更换或增加基本的开/关功能现在是可行的(只要制造商允许)。这是一场胜利。这不是杰特森的家庭自动化，但它确实工作可靠。

当我到达我的酒店房间时，有一个 Alexa，我花了 10 分钟试图让房间暖和起来。我尝试了我的两页 Alexa 小抄上的一切，但没有运气。我放弃了恒温器(注意，操作酒店恒温器从来都不是一个可预测的步骤)。接下来我做了灯光、窗帘、设置闹钟等等。所有这些都完美无瑕——甚至能识别“窗帘”或“窗帘”等。虽然没有完全完成(像让窗帘停在 50%是不可能的)。这个房间相对较新，有一个设计非常好的单一控制台有线遥控系统。



我相信现在声音的一个核心问题是期望。有各种各样的现实世界的问题，从家里的客人到站在窗外大喊大叫进入你家处理的人，但是一个人确实很快习惯走进房间说“Alexa 请开灯”,当然如果你还可以随着音乐得到关于天气等问题的回答，这是一个净增加。

voice 真正令人失望的地方与几乎所有新产品都令人失望的地方是一样的——它做的没有你希望或想象的那么多。多年来，技术爱好者一直在尝试家庭自动化场景——将你的家“编程”为锁门、启动周界安全、关闭室内灯(卧室除外)、关闭电视、打开婴儿监视器等等，所有这些都是为了命令“就寝时间”。这是不会发生的，任何有这个设计点的人都会失败。这将会失败，就像微波炉按钮“重新加热”不起作用，或者语音应答系统问你“陈述你的问题”总是让你“请稍等，我给你接通接线员”。

我看好基本指挥控制的语音。除此之外，我们还处于非常早期的阶段，前面还有很多挫折。战略性的问题是语音是否会取代打字，就像一个编辑控件取代查询语法一样。用这种方式表达，声音仍然很遥远，但越来越好。询问语音是否会取代应用还是更多仍然是一个“大而难的问题”。

尽管如此乐观，仍有不少挑战:

*   嵌入第三方运行时是一种复杂的方法，很可能会使设备很快过时，而且谁想要更多的麦克风呢？我很不愿意购买那些声称需要麦克风和扬声器来完成烹饪、报时甚至是电视功能的设备。我认为 Assistant 和 Alexa 很有可能在相当长的一段时间内作为中心受益(另外我有一部手机！).我的观点是，端点应该专注于利用集线器的存在，并专注于将它们的 API 集成到集线器体验中。值得注意的是，Homekit(或许还有 Siri)是一个更复杂的集成，而不仅仅是一个软件 API，并对硬件提出了要求。
*   **如果你相信 hubs (Alexa，Assistant，iPhone，甚至 Bixby 和 Cloi 等。)是技术行业意义上的“平台”,那么整合所有这些平台可能是一个非常困难的策略。**如今，这些平台更多的是相似而非不同。但随着新平台的出现，情况总是如此。它们很可能会迅速发展并有所不同，维持与所有这些平台的连接将被证明是妥协的，就像任何跨平台挑战一样困难(但用于掩盖差异的表面积要小得多)。再一次，考虑一下 HomeKit/Siri 已经不仅仅是 API 中简单的“动词”描述符了。
*   某个地方确实需要一种商业模式。现在没有人靠卖集线器赚钱，集线器制造商也不卖终端。因此，问题是这些目标是什么。从外部来看，我们可以假设各种各样的事情，比如在冰箱上做广告，或者使用集线器来订购一些东西，然后作为订购订阅的一部分来赚钱。但今天，所有这些工作都是作为亏损领袖进行的。从历史上看，亏损的领导者不会转向其他业务，尤其是大公司。尽管 Android 和 Kindle 取得了巨大成功，但它们的最终经济效益仍存在许多问题(当然是独立的，但也是广度服务的一部分)。
*   这似乎不太可能是一场赢家通吃的 T2。主要公司的行为似乎表明这是一个被主宰的市场。关于家，我们知道的一件事是，市场通常需要标准化和商品化。在我看来，语音控制很容易成为下一个 HDMI 或 Wi-Fi(从各方面来看，好的和坏的)，而不是下一个 Windows 或 iOS。要控制的事物的空间过于分散和多样，以至于不能简单地(或容易地)集中在一个玩家身上，尤其是在早期已经有至少 4 种可能的技术解决方案的时候。事实上，两者都可以自由嵌入和连接到今天，这使得这种多样性更有可能继续下去。当然，还有家庭端点的替换周期长度的问题。
*   **小和大会有不同的操作。**大公司(尤其是三星和 LG，也许会想到苹果)将从索尼(以及之前的 RCA)那里吸取经验，试图在语音方面拥有专有的角度，同时也“支持”今天的这些大公司。具有讽刺意味的是，所有这些都会产生意想不到的副作用，使他们的产品(我们大多数人都会有)更难使用，更复杂。任何在鼎盛时期拥有索尼 AV 设备或索尼 PC 的人都知道，“专有”的东西总是让使用标准的东西(记忆棒)变得更加困难和不那么令人满意。).随着这些平台的分化，小公司将需要押注，但这些押注使它们成为伟大的产品和伟大的并购目标，但也带来了分销方面的挑战(百思买可能希望你整合一切，但谷歌可能会提供大量支持，成为助理专属)。这就像今天为移动设备开发一个应用程序一样。

# “嘿谷歌”

在深入电视之前，有必要先介绍一下谷歌在展会上的表现。我见过(也曾经见过)资源丰富的大公司来到 CES，让人们知道他们的存在。老实说，我从来没有见过谷歌用“嘿谷歌”完成的任何事情。尽管一个巨大的场馆因为暴雨引发的洪水而关闭，但现场的情况是“疯狂的”,无法逃脱。

单轨铁路，时装秀飞碟，甚至威特房子(哈哈， [GIK](https://jamesbondlocations.blogspot.com/2012/05/whyte-house-las-vegas-hilton.html) ，威拉德会怎么说？)正在炫耀“嘿谷歌”。在展览中，如前所述，有 100 到 1000 多个“嘿，谷歌”，当然还有数百名谷歌员工代表。

唯一能拿走的是声音是一个很大的赌注。今天这个赌注的执行是将助手功能构建到 CES 上的所有东西中，从汽车到炉子到电话到扬声器到时钟等等。这个节目的目标是展示谷歌如何融入所有这些设备。

这是一项巨大的努力。Alexa 在展台的展示区也有类似的存在(如果不是更多的话)，但没有超过谷歌的存在。这就是领导职位的样子——人们自愿表示支持。

正如一些评论家无疑会说的那样，苹果 Siri 和 HomeKit 以及微软 Cortana 都没有引人注意的存在(无论是第一党还是其他)。

因为我认为这还为时过早，也不太可能成为赢家通吃，所以这种存在如何随着时间的推移而演变将是重要的。

也就是说，执行过程令人印象深刻，但也很昂贵。

# 电视和显示器



Typical OLED panel striving to be more than a “TV” showing how i is basically a souped up screen saver. It shows the same pattern as the wall it is mounted on, a clock/weather, fireplace (not warm though), fake bookshelf, and photos. It is pretty nice I admit. But still who wants that running all the time?



2018 年的电视是有机发光二极管最终走向主流。与 LED 相比，它仍然很贵(LED 已经变得不可思议的便宜，任何去好市多的人都知道)。所有主要玩家都展示了超薄的大(高达 85 英寸)有机发光二极管屏幕。

这里有一件 CES 需要注意的事情。花哨的“尚未发货”的有机发光二极管电视都有集成底座，5 毫米屏幕就放在上面。这些基地是扬声器酒吧，并利用一些获得的深度，使一个后发射低音炮在面板背面。由于每个人都在展示这些产品，15 年后的壁炉壁挂架和 4 英寸凹进的壁式角落很可能不会成为下一次展示的合适尺寸。

而且基本上没有曲面电视，当然也没有 3D 电视。我试图想出一些像 3D 一样来来去去的东西，我能想到的可能就是 VR 耳机。

今年的电视有几个重要的战略点:

*   **有机发光二极管**。看到有机发光二极管是令人兴奋的。质量太疯狂了。有趣的是，我们现在几乎没有“展示”这种东西。仅仅在屏幕上显示图像就需要大量的计算。于是问题就变成了“什么是*本土*图片？”我真的相信，相对于信号来说，显示器上发生了太多的视频和音频处理，大多数都使事情变得更糟，但你需要一些。对于视听纯粹主义者来说，这是一场争论，就像电影人和数码人在相机里争论一样。但这很重要，因为我认为许多电视关掉所有这些运动和音频东西会变得更好。当然，对于 4K 或 8K 屏幕，缩放将是至关重要的。
*   **“AI”作为流行语。每个人都在谈论他们的电视具有智能。智能是为了用花哨的处理芯片改善声音和图像，也是为了指挥和控制电视。我认为称之为与谷歌助手 AI 集成是好的，但所有其他东西都是一种延伸。我把 AI 和所有显示像素和照明方案的命名放在同一个桶里。**
*   **特点**。电视是三星和 LG 的中枢，因此拥有所有的指挥和控制。它们都有超高的遥控器(在美国有网飞按钮)，但专注于电视内的语音控制和“智能中心”。老实说，所有这些东西似乎都分散了我使用电视的注意力。我怀疑对很多人来说，这只是“猜猜是怎么回事”的复杂形式，因为他们连接到有线电视或卫星电视盒以及 Roku 或 AppleTV 等第二来源。我担心软件更新、安全等问题。我不喜欢电视有麦克风，并担心网络连接，坦率地说，我可以没有所有的软件功能。
*   **中国**。对我来说，最大的消息是中国制造商正在取得进展。这已经酝酿了 15 年。我怀疑与韩国竞争的内部描述是激烈的，我们还没有看到这一点的最后。中国已经看到了韩国是如何从日本手中接管过来的，并将照此行事。不过，韩国制造商拥有的不仅仅是电视，因此像海尔和 TCL 这样的公司还有很多工作要做。中国产品的质量越来越高，不仅仅是 2 年前的 LED/LCD 面板。有第二梯队的中国制造商，他们不是很好。
*   **一线 v 二线。**一线玩家有自己的“控制中心”，LG 谈 WebOS。二线玩家还在做安卓电视。Chromecast 和 Miracast 一样内置于一系列电视中。值得注意的一点是，中国制造商非常“全力”使用谷歌平台制造电视，尽管谷歌在中国没有业务。这感觉就像手机又以同样的方式发生了。

在展示有机发光二极管以外的图像方面仍有创新。LG 有一台非常漂亮的 4K 专业投影仪，大约 18x6x6 英寸，垂直放置，便于携带。它甚至有内置电源和可伸缩的电线。

索尼继续推进其令人难以置信的短距投影仪。今年全尺寸的一个是可利用的(下面),真的很了不起。基本上，看起来像宜家电视柜的是一个 120 英寸的投影仪，你可以把它放在离墙 9.6 英寸的地方(即使靠着墙也是 80 英寸的画面)。支架中有扬声器和低音炮，加上电缆距离盒子背面只有几英寸(但被隐藏起来)。只需 30，000 美元，它就可以是您的了，并且正在发货。

索尼也在平板电视声音方面进行创新。这已经成为毫米有机发光二极管屏幕的一个大问题——扬声器放在哪里。索尼做的事情超级酷。他们将前置“扬声器”内置在面板的前面，因此基本上面板提供高音和中音，并直接向您开火(同样，更多的处理使这一切看起来像有 4 或 6 个扬声器)。面板背面有一个后置低音炮(本身有点疯狂，因为它只有 1 英寸深)。看起来很疯狂。去年它出现在一台电视上，并在所有型号中传播。

色彩深度的世界继续在 HDR10 和杜比视界等等之间混淆。我只是“忽略”所有这些东西，因为我知道在短时间内每个人都会支持它们，然后最终所有的内容都会在一个里面。这种战斗在电视上屡见不鲜。

最后开始计数到 8K。是的，这是完全愚蠢的，因为没有 8K 的内容，但很快就会有了。在此期间，预计每个拥有 8K 的人都会谈论他们如何拥有扩大规模的魔力，而没有 8K 的人会谈论如何没有内容。

# 耳机和音频

我们可以感谢苹果 AirPods 为 Jawbone 开创的无线入耳式耳机注入了生命。有几十个耳塞制造商都在炫耀类似的电池充电盒设计。

主要竞争优势:

*   合身。一些制造商添加了不同的机制来调整和/或固定耳塞进入您的耳朵。和有线耳塞一样，人们普遍认为它们不会脱落。我也不相信他们。

所有这些都在继续努力提供与苹果匹配的配对体验，当然还有设备切换。他们被蓝牙的复杂性束缚住了手脚，更不用说普遍的脆弱性了。我认为 Android 手机将继续与这一类别的售后产品斗争。

CES 上最令人惊讶的时代变化之一是韩国制造商如何从显像管电视开始，崛起并主导家电市场。除非你去完全“厨师的厨房”(一组制造商最近合并)，否则阅读这篇文章的人下一次购买的电器可能是三星或 LG。就竞争而言，最令人兴奋的事情是中国现在对战胜韩国有多重视。

关于家用电器的一些有趣之处:

*   **轮毂**。LG 和三星希望我们考虑他们的中枢——既是硬件设备，也是语音助手。LG 有 *Cloi* (nee，Chloe)，三星有 *Bixby。与此同时，这些公司也作为 P & Ls 运行，因此一大堆设备也在不同程度上支持 Alexa 和谷歌。会变得很混乱。这种类型的战略举措一直困扰着索尼——每个人都想在竞争中获胜，但高管们正试图实施专有的跨公司战略。这种中心战略的一部分给这些设备带来了太多的复杂性，因此需要注意。我想我会购买这些公司的扬声器，只要他们的心似乎在 Alexa/谷歌的整合上。但是你永远不知道…*
*   **声音**。关于话筒的去向，有各种各样的解决方案。存在这样的解决方案，其中冰箱是主集线器或电视，或者可以有一个或多个扬声器。当然，LG 也演示了 Cloi 的物理表现，它在舞台上的首次亮相很粗糙。再说一次，我的观点是，在我可能用了十年的冰箱或电视上安装麦克风并不是正确的做法。
*   **创新**。经常被忽视的是电器的大量创新。这就是“厌倦 CES”让我生气的地方。现实是，连通性和竞争的结合给基本家电带来了一些超级酷的变化。第二个抽屉的迷你洗衣机/烘干机或“小容量”洗碗机都超级酷。此外，电器使用更少的水和电。也有新的清洗衣服的方法。虽然我们可以模仿所有的食谱屏幕，跟踪冰箱里的食物等等，但还是有一些用处的。诀窍在于，安定下来需要时间，所以我将成为一个谨慎的买家，尽管我是早期采用者。
*   **连通套件。**三星和 LG 都想卖给你家里所有的电器。这极具挑战性。首先，大多数电器套件都是在房子改造/新建时购买的，然后更换一次一个出现故障(因为维修比更换花费更多，与你交谈 2004 年的 Miele 洗碗机)。有趣的是，三星和 LG 几乎像所有的西尔斯一样覆盖了你。他们有高端的厨房(取暖器，煤气，电，都有专业设计的新的领先材料)和洗衣房。在 Toto 和 Kohler 的地盘上的浴室里也没有玩家。

说得难听一点，我们把通用电气(GE)、惠而浦(Whirlpool)和其他公司称为“第二梯队”(这在数字上可能不真实，但在未来的广度和规模上是真实的)。提醒一下，通用电气现在在中国归海尔所有。这些产品正在与 Alexa/Google 进行轻量级集成，而不是过度需要“全包”它们的完整套件。你最终会得到一些不错的创新，而不必购买一整套可能过时的硬件。

我最喜欢的例子是洗衣机/烘干机的漩涡系列。它们只是低科技的无线网络应用。当然，你可以使用扬声器通过他们提供的集成来控制这些，或者你可以使用他们的应用程序。他们提供的东西非常有用。例如，惠而浦(Whirlpool)的新(也是他们的第一款)one drum w/d(冷凝器类型，就像他们在欧洲从 Asko 获得的一样)有一个可容纳 40 桶洗涤剂的容器。你只需要把衣服放进去洗干净，然后最终你会收到一个电话通知来补充。您不必每次都打开并检查，他们也不必在设备上构建复杂的 UI 来检查。

通用电气公司创造了一个超范围的通风口，带有一个屏幕来搭配他们的烤箱/炉灶。有趣的是使用那个位置而不是冰箱。在美国，几乎每个人都有微波炉，所以思考厨房的演变有点有趣。仍然不热衷于在屏幕上建造一个在厨房重建前可能不会被替换的油烟机。

# 汽车和运输

尽管汽车占据了展览空间的 10%,但它似乎占据了展览“带宽”的 50%。就国际消费电子展而言，这是我最困惑的一个领域——所以把我说的任何负面的话都当成困惑吧。

当然我非常清楚地看好[电动](/my-new-electric-car-is-really-like-a-phone-69ddd0be70eb)和[自主](https://www.recode.net/2014/7/8/11628624/auto-autonomy-cars-are-racing-toward-disruption)。毫无疑问你会收到我的来信。我也不知道时间表，但我相信这是一场大规模的文化、城市规划和经济转变，将会造成巨大的动荡，我们最终会在所有权、交通、道路、城市规划等所有方面与今天完全不同。

也就是说，CES 和汽车只是一个奇怪的组合。我相信我们在展会上看到的大部分都是大型汽车公司和主要供应链供应商的定位，而不是真正制造汽车和改变事物。与会者不是合适的人，汽车公司真的不像典型的消费电子展人员那样在消费电子展上做生意。这是一场好戏。这是令人沮丧的艰难跋涉。我敢肯定，在规模庞大的底特律汽车展(正在举行)上，事情会有很大的不同。

一个例子是福特，当然不是把他们单独挑出来，但他们基本上有一个整天都在进行的“愿景”演示，基本上花了很多时间说“交通很糟糕”(你不能停车，交通，拥有卡很难，等等。).是的，没有争论。福特上周宣布了一批新的 GCE 皮卡和一款新的野马车，支持了这一点。

在很大程度上，所有的“汽车”展台都是未来自动驾驶汽车或电动汽车或两者兼而有之的模型。这些模型的重点不是修复交通工具或提高电池寿命或任何东西，而是简单地展示“驾驶舱”设计——字面意思是舒适座椅、娱乐系统和杯架的模型。在极端情况下，贝尔直升机公司有一个驾驶舱，但他们不会“透露”该设备如何或是否会飞行，但你可以排队等待，体验一次虚拟现实之旅，在这些由神秘空气推进提供动力的舒适座椅上飞行会是什么样子。说真的。

有一吨的“最后一英里”运输展示，如踏板车，电动自行车，折叠自行车，等等。这些都很有趣，因为它们是消费者世界不可或缺的体验。我最喜欢的一个是 OJOElectric，它是与福特官方授权的合作伙伴。花 2550 美元，你就可以得到一辆 50 英里范围的踏板车，有篮子，侧镜，20 英里的速度，远程电源，自行车道友好，和一个光滑的车载充电器(可伸缩的电线)。我在为我自己考虑这件事！

电动汽车在一定程度上得到了展示，但主要是在零部件层面(电池或驱动系统，来自供应链展台)。我喜欢的一辆电动汽车是 AEV 科技公司的汽车，如下图所示。这是一辆邻里 NEV 或“城市交通工具”。我住在西雅图市区，不禁想到使用这种工业电动汽车的服务车辆可能带来的污染和噪音减少。该公司位于德克萨斯州的奥斯汀，该市(就面积而言)现在正在使用它们。

ClearMotion 是一款超级狂野的“运动取消”减震器。它对运动的作用就像噪音消除对噪音的作用一样，在几分之一秒内减少道路输入。下面的演示非常酷，这不仅仅是因为它是在 Surface Studio 上进行的，它还可以让你展示汽车如何应对任何路面或波士顿地区的道路。你可以看到汽车下面的道路成形了，但车身却纹丝不动。这是一个超级酷的、软件支持的创新。

# 增强现实/平视显示器(AR/HUD)

公平地说，虚拟现实被几乎每一个展位和主要供应商所抛弃。我不会说它就像 3D 电视一样，但我不知道明年是否会有任何发展将它带回来。

增强现实引起了巨大的轰动，但不是你想的那样。它不是游戏或工业或任何前瞻性的东西，而是当谈到未来的驾驶体验时，它遍布整个节目。最令人兴奋的演示是车载屏幕或基于手机的 ar，很少有演示展示眼镜/护目镜。

令人难以置信的是，平视显示器(HUD)已经存在了几十年。当然，软件、激光和 LED 都取得了巨大的进步。许多厂商展示了售后市场和新车 HUD 系统。像 VR 护目镜，有真的低端的，也有让人印象极其深刻的高端的。只要我必须开车，我就喜欢这一点，因为 AR 叠加导航消除了驾驶时的主要压力来源——当你不知道你要去哪里时，看向别处是非常可怕的。

到目前为止，最令人印象深刻的是一家使用“全息”成像的瑞士公司 WayRay。他们对互联服务等有着远大的愿景。

在零售领域，AR 或类似 AR 的产品有许多创造性的用途。正如所料，这不是今年才出现的东西，而是过去几年中展览的一部分，但产品现在真的开始工作了。

我最喜欢的 AR 演示是 YouCam，它可以显示各种化妆品的使用前/后。这是一个渠道行动，也涉及到与化妆品公司的合作，因此它不是一个合成的观点，而是与可用的产品直接相关。他们也有一个应用程序，你可以体验家庭购物，但店内设备非常酷。

相当多的产品希望创造性地使用您的手机来增强购物。一个有趣的例子是向品牌提供技术来开发其产品的 3D 图像，然后让你通过扫描二维码在手机上快速查看完整的产品。这需要一个应用程序，所以我认为会有一些引导的挑战，但在应用程序内的体验是好的。视图是 AR 视图，而不仅仅是产品镜头。

# 移动的

CES 不是一个移动展会，这里永远不会有任何关于移动的重大公告(可以说是 COMDEX 的消亡导致 CES 感觉像一个 PC 展会)。当然，最有趣的事情是移动设备如何取代电脑成为整个展会的标志和集成点。平板电脑是展台上的主要交互方式，而触摸是目前主要的交互模式。

移动领域最大的新闻仍然是来自亚洲的供应商和/或希望用手机或手机硬件做事的供应商更喜欢他们“黑”Android 的能力。高端供应商都在炫耀他们的苹果连接。对我来说，看看所有正在进行的 Android 工作，欣赏创新，同时深切关注中期的健壮性和可维护性并不难。我们都已经看到这种情况在家庭电脑中发生，没有理由认为 Android 会有所不同。

华为 Mate10(和 Pro)将在美国市场的展会上首次亮相。然后在最后一分钟，美国电话电报公司退出了分销，这导致了华为首席执行官的尴尬主题演讲。我确实玩过这款手机，就运行 Android 的手机而言，它是一款顶级机型。它有一个徕卡相机镜头，快速充电，一流的谷歌眼镜。对 Android 用户来说非常好。

# 个人电脑

PC 市场正处于一波强劲的整合浪潮中:制造商的整合和用例的整合。在消费电子展上，你看不到个人电脑的任何新的或创造性的用途。虚拟现实演示都是在电脑上进行的(无论是否连接)，3D 打印机都是由电脑驱动的。还有很多其他的，但我会说他们是长尾。

也就是说，戴尔和联想尤其展示了令人兴奋的新产品。我的观点是，戴尔目前在制造非常坚固、以工作为中心的个人电脑方面处于领先地位。下面你可以看到最新的戴尔敞篷车，它与 Surface Book 2 或联想 Yoga 竞争。其中一部分是一种新型键盘，它使用悬浮磁性开关(超级棒)。15 英寸运行全内核处理器。下面您可以看到戴尔的所有可扩展性(和无线)。我认为在完整的无线设置中存在一定程度的缺陷，在这成为核心的 Windows 场景之前，我会保持谨慎。很多事情都会出错。

# Qi 无线充电

无线充电是非常适合 CES 的事情之一，也是反对者需要注意的事情。Qi 是一种全行业的努力，在达到临界质量之前需要几个周期(几年，产品周期),然后是我们日常生活中使用的一系列好产品。

我认为，今年，iPhone X 为 Qi 世界提供了动力，使其行动起来，创造出一些坚实的产品。可能有太多或不同的效用或质量，但至少在这一点上它们都将工作(不像我的雪佛兰 Bolt 中的充电器不符合标准)。

有各种形式的收费:摇篮，平板电脑，闹钟，车载(平板电脑和摇篮)，等等。如果我仍然使用外部显示器，我肯定会想要一个内置的显示器，就像下面的华硕，因为这正是我的手机会高兴的地方。

在手机的许多外部应急电池中，LinearFlux Hypercharger X [sic]上有一个 Qi 充电板。它为你的手机提供了一种非永久性的粘性，这样两者就可以保持在一起(但手机会粘在你的口袋里)。充电器本身需要一个插头来充电，但你可以同时在它上面给你的手机充电，所以总是少一根电缆。

来自英国的 Airsound Technology 是一款网状多室扬声器，可与所有音频源配合使用，内置 Qi 坞站(以及 USB-C 母插头)。

# 健康

在过去三年左右的时间里，健康状况一直在飞速发展，但却令人失望。遥测设备越来越多，它们最适合在医生办公室使用，但也适合在家里使用。我感觉很多这些都是领先于诊断价值的技术。

当然，医疗保健领域正在进行大规模创新，但今年的 CES 上没有。总的来说，仍然有大量的伪科学和过度测量。对于正确的患者，有几年前不存在的神奇设备，几年后，这种测量和机器学习的结合将开始新一轮的预防性医疗保健。从这个意义上说，消费电子展是必要和重要的，但现在还不是时候。将会出现的公司将会具备测量和连接的能力，并将其与最先进的软件相结合。

有一家公司对我来说很突出，那就是 RightEye，它使用的眼球追踪技术对我们这些做过软件产品可用性测试的人来说很熟悉。它将跟踪功能用于帮助眼科专家诊断各种潜在问题。它可以做非常有趣的事情，比如轻松地将阅读障碍与视力问题分开，或者为帕金森或 MS 提供早期线索。你基本上只需坐在设备前面(由第三方制造，因为这家公司的价值在于软件)并进行一些测试。然后软件分析结果。它是先进的硬件、先进的软件和如何进行测试的见解的组合。这些测试只是增加了诊断中的其他因素。超级酷。

传感器无处不在，不断有许多产品将加速器、GPS、温度等嵌入头盔、袜子、鞋子、胸罩、衬衫、工具等。有一种独特的骑马产品可以在你摔倒时发出移动警报。一家公司展示了一个完整的警察遥测装置(它是背心下的一块板)，如果发生武器发射或一名警察被击中(就像机械战警或外星人)，它会向指挥部发出警报。他们的愿景是在指挥中心数据流中增加人的因素。

CES 上摆满了用于家庭安全的摄像头和许多婴儿监视器。由 Wearless Tech(有点混乱)发行的 Clarity Cacooncam 有一个用于婴儿床的独特的高架摄像头，但它也可以检测生命体征(通过热(。有一个监控应用程序和所有。只是不同的视角。

虽然还有其他家庭健身贸易展，但 CES 见证了 Peleton 跑步机的首次亮相。这是一台多么好的跑步机啊。这是一种新型跑步机，它使用更小的部分，而不是连续的踏板，以获得更多的弹跳。有一个 32”(！)连接到众所周知的 Peleton 系统的屏幕。它不仅仅是为了跑步，所以预计课程会有重量和其他锻炼。不便宜也不小，但是很酷！这是 SaaS！

两种健康产品上了“我不敢相信我刚刚看到了”的名单。一个是名副其实的“胎儿摄像头”(M1 型号)。它实际上是一个家庭超声波仪，可以为你的宝宝制作 3D 照片，与朋友和家人分享(真的！).我就不多说了。
