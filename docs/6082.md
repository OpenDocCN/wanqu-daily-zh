# 精通技术的父母的崛起|海象

我 可以理解当你的母亲突然占据你的电脑屏幕，要求你打电话给她时，远离成人世界的目光和干扰，和你的朋友一起欢笑是多么令人不安。作为一名青少年，当你拒绝接电话、发短信、发邮件和玩游戏时，妈妈知道如何远程关闭你的技术，这是什么感觉？请记住，当我的孩子不知道为什么他们的应用程序不见了的时候，我会帮我修理他们的手机。我是为他们建立 Tumblr 上传定格视频的人，我也教他们怎么做。当他们的第一份工作要求他们学习最新的软件时，我就是他们要找的人。

这是一个常见的刻板印象:如果你需要开发一项新技术，你会把它交给家里最年轻的成员。近二十年来，媒体一直在发表关于成年人对技术明显无知的文章。2000 年，*经济学家*声称“家庭技术大师”更有可能成为青少年，而不是一家之主也许他们应该和母亲核实一下，因为在我的房子里从来没有这样过，我和我的丈夫以及 16 岁的孩子住在一起。

在典型的一天，我会玩*魔兽世界*，将新软件集成到我们无数的机器上，提醒每个人将他们的文件备份到外部硬盘上，并解释新应用程序的酷因素。几年前，我甚至给我的孩子买了一个树莓派，我认为这是制造电脑的一个很酷的入门。我的孩子并不像我一样对软件感兴趣，所以我最终自己编写了它，然后把它发给蒙特利尔的另一位家长，让她和她的孩子做同样的事情。我确信，我不是唯一一个对技术了解更多(显然也更感兴趣)的书呆子家长。事实上，我不认为我的孩子缺乏技术知识在年轻人中是独一无二的。

我 t 并不总是这样。像许多 80 年代成长起来的孩子一样，我比我的父母更了解技术，并且会带着一种居高临下的叹息，可靠地履行我作为家庭技术天才的角色。从 7 岁开始，我设置了录像机来录制我母亲最喜欢的肥皂剧*我们生活的日子*，到了 12 岁，我已经是那个尽职尽责地给大盒子电话应答机编程的人了。我记得面对我的技术优势时我妈妈的承诺:有一天，我的孩子会以同样的方式取代我。对她来说，这可能是一种防御机制，或者这可能是她实际上相信的，但我开始在 DOS 中编程的那一天，就是我的孩子——离出生还有 14 年——开始在我妈妈的预言中失去立足点的那一天。

技术至上的假设告诉我们，90 年代中后期出生的一代人(有时被称为 Z 世代，有时被称为后千禧一代)伴随着互联网、手机和笔记本电脑长大，因此对新技术有“直觉”。据推测，这一代人是直觉型的，而我，作为 X 一代的一员，却不能。但是大量的研究反驳了年轻人总体上比我们这些老年人更加数字化的观点。

大约有同样多的人上网，这被认为是千禧一代的领域。例如，根据皮尤研究中心最近的一项研究，97%的千禧一代使用互联网；96%的 x 世代也是如此。即使当我们用智能手机连接时，使用上也没有太大的区别。不到 30%的千禧一代主要在移动设备上使用互联网，而 x 世代的这一比例为 18%。我们几乎和“数字原住民”一样紧密相连

作为一个十几岁孩子的母亲，作为一个给新千年一代教一年级新闻学的人，我还可以告诉你，我这一代人(为了与技术互动，在成长过程中学习编程的一代人)与今天这一代人(不需要学习编程的一代人)截然不同。像我的许多 70 年代出生的同龄人一样，在网站模板出现之前，我就学会了如何编程。当时，像 WordPress 或 Squarespace 这样的内容管理系统还不存在。在没有这一切的情况下，我学会了如何自己做事。到我十八九岁的时候，我已经编写了一些小游戏与我的朋友分享，我每天晚上都在留言板上与其他志同道合的书呆子聊从漫画书到政治的一切。

诚然，古老的技术——像旋转式电话和盒式录音机——因为一个原因而从知识中消失了:我们不再需要它们了。但我确实想知道，当我们不再对事物如何运作感到好奇时，我们还会失去什么。当我们每天都在使用技术，却不知道它是如何工作的时候，会发生什么呢？

O 除了发展自己的社交媒体每天早上查看自己的 Instagram 照片以确保获得正确的赞数，孩子们现在都在构建什么呢？他们试图回答的不可知问题是什么？我们所有假定的技术进步似乎都消除了技术使用者体验中的两个关键特征:耐心和好奇心。在许多方面，这些可能是相同的特质。解决问题需要时间，理解技术背后的结构需要时间，在追逐中迷失方向也需要好奇心。如果你一直在寻找其他人来解决问题(另一个应用程序、另一个软件或你的妈妈)——更糟糕的是，你已经找到了——那么你就失去了解决问题的批判性思维。

想想这件事的潜在后果:我第一次给我的新闻学学生布置任务，把他们写的故事上传到他们学校创建的博客上，我发现许多人不知道如何做某些基本的事情。有些人甚至不知道他们需要登录哪个账户才能访问他们的博客。对大多数人来说，上传照片或音频文件需要更多的指导。

作为一名教师，我花了很多时间在课堂内外解决简单的技术问题。其中一些挑战在于糟糕的批判性思维，还有一些可以追溯到对互联网和计算机如何工作缺乏基本的了解。随便问一个青少年，如果没有 YouTube、Instagram 或 Vevo，他们会怎么把视频放到网上，你可能会一脸困惑。他们甚至会嘲笑这种想法，即他们至少应该对如何做这件事感到好奇。但是你认为互联网是怎么建立起来的呢，年轻的学徒？因为我们试图回答不可知的问题。

我不是第一个注意到年轻一代被预言的优越性与我们现在的现实相差甚远的人。2017 年，期刊*Teaching and Teacher Education*发表了一篇名为“数字原住民和多任务者的神话”的论文，作者在论文中得出结论，数字原住民并不存在。事实上，我们所有人——年轻人和老年人——都可以提高我们的数字技能。一年前， ECDL 基金会，一个“致力于提高劳动力、教育和社会的数字能力标准”的国际组织的一项研究得出结论，所有年龄组的很大一部分人，包括伴随着计算机长大的一代，缺乏基本的计算机技能。更令人担忧的是，2013 年的一项研究评估了来自世界各地 21 个教育系统的 6 万名八年级学生的计算机和信息素养技能，发现即使在表现最好的地区之一安大略省，也只有 5%的学生排名最高，这需要在网上搜索信息时使用批判性思维。

像我这样的人，在< blink >标签(如果你不知道这些是什么，请认为你是幸运的)还是一个东西的时候就建立了网站，对编程语言如何工作有一种本能，更重要的是，对编写它们的人如何思考有一种本能。如果 Z 世代似乎有这种本能，那是因为我们都在为他们开发应用*。那不是本能。这是一种后天养成的习惯——就像一个孩子听到振动声，会本能地伸手去拿手机。在其研究结论中，ECDL T4 基金会呼吁在我们的学校系统中进行更多的数字素养教育。我完全同意。我们不能仅仅因为今天的孩子和青少年出生在一个每个人都可以在互联网上出名的时代，就认为他们理解机器的影响和力量。我们不能假设他们知道如何使用它们或创造新的。这让我们回到了开始:如果你不接电话，是你妈妈打来的，准备好接受技术接管——因为你可能伴随着它长大，但它是你妈妈的答案。*
