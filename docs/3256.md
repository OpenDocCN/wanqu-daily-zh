# 在算法时代，人类看门人回来了|技术|守护者

G 瑞格林登也许不是一个家喻户晓的名字，但他改变了我们与文化互动的方式，永远地改造了零售。上世纪 90 年代末，林登是亚马逊的一名工程师，他致力于解决一个奇怪的问题:如何在没有人工干预的情况下推荐书籍。在那之前，亚马逊依赖于每年撰写数百篇评论的编辑。这是一个既费钱又耗时的过程。

事实证明，自动推荐比任何人预期的都要棘手。林顿破解了。他想到了“个性化”，矛盾的是，这意味着不看个人的购买历史，而只看产品之间的相关性。不管你过去买了什么，[亚马逊](https://www.theguardian.com/technology/amazon)意识到，如果产品 A 经常与产品 B 一起被购买，这意味着几乎所有购买产品 A 的人也会想要产品 B。亚马逊测试了结果，看看哪种方法卖出了更多的书。毫不奇怪:编辑们很快就在寻找新的工作。人类出来了；机器进来了。一些估计表明，亚马逊三分之一的销售额来自这些推荐。从那以后，算法的崛起就一直没有停止过。现在书，文章，音乐，电影，更不用说节假日，衣服，都是机器建议的。

去年出版了 100 万本英文新书。至少从古希腊开始，人们就相信有太多的东西要读；现在他们可能是对的。当然，这还不包括所有自我出版的作品、大量的新闻或互联网的庞大。无论用什么标准衡量，我们的阅读材料都多得惊人。

我们拥有的越多，我们就越依赖算法和自动推荐系统。因此，算法推荐、机器学习、人工智能和大数据不可阻挡地向文化领域进军。

然而，这并不是故事的结尾。例如，搜索告诉我们想知道什么，但如果我们还不知道自己想要什么，就帮不上忙。在算法时代，人类的创造力和敏感性远没有消失，而是有了新的价值。是的，我们拥有的越多，就越需要自动化。但是，我们也越来越想要有见识的和独特的选择。人类回来了。

这就是为什么，尽管拥有世界上最强大的图书推荐引擎，亚马逊还是收购了基于个人书评的网站 Goodreads。这就是为什么像 Canopy.co 这样的网站在亚马逊上蓬勃发展。Canopy 知道许多亚马逊最好的商品隐藏在平庸的泥沼中。Canopy 的创始人都是设计师，他们每天从数以千计的参赛作品中筛选出优秀的产品。

这就是为什么出版商不断推出新版本，以允许更多样化和个性化的书单，也是为什么书店再次繁荣，尽管我们可以在网上找到任何想要的书。我们去浏览他们的桌子。在日本，他们谈论“读书太多”的不舒服感。他们也有自己的解决方案:东京银座的一家[书店，一次只卖一本书。](https://www.theguardian.com/books/2015/dec/23/japanese-bookshop-stocks-only-one-book-at-a-time)

这种对策展的复兴兴趣不仅仅发生在出版业。在 Spotify 上，你可以听 3000 万首歌曲，其中 20%从未被播放过。为了帮助管理这个庞大的目录，Spotify 花费了 1 亿美元(7700 万英镑)收购了一家名为 [the Echo Nest](https://www.theguardian.com/technology/2014/mar/06/spotify-echo-nest-streaming-music-deal) 的公司，该公司开创了一种被称为音频指纹的技术，可以自动对歌曲进行分类。然而，与此同时， [Spotify](https://www.theguardian.com/business/2016/may/24/spotify-revenues-surge-80-to-more-than-13bn) 大规模扩大了播放列表制作者的范围，音乐专家正迅速成为新的 DJ。

网飞的电视和电影比我们想要的还要多。作为利用数据科学管理文化的早期先驱，它甚至发起了一项价值 100 万美元的竞赛，让研究团队改进其算法——然后，尽管有奖金，但没有实施任何改变，因为它们被认为不够好。然而，网飞也训练观众详尽地标记其内容。他们做出了机器无法做出的判断:结局是伤感的吗？小胡子在电影里重要不重要？

脸书陷入了一系列关于其新闻监管的争议，从直播杀戮，到剪辑越战的标志性照片，再到政治偏见的指控。它最近试图通过解雇人类编辑来使这一过程变得平稳……却发现新闻反馈退化为一大堆虚假和有争议的新闻故事。

苹果的新闻和音乐应用程序充分利用了他们的人力资源，甚至从新闻编辑室和电台雇佣了著名的名字。Twitter 对其【T2 时刻】产品进行了大量投资。虽然没有受到普遍欢迎，但这表明 Twitter 希望更好地策划。三星的新闻 app 分为你想知道的和需要知道的；前者由算法选择，后者由编辑选择。大型科技公司正在疯狂招聘传统专家。

我们还有多余的东西。西欧家庭平均拥有 1 万件物品，美国家庭更多。但为了应对这种情况，我们没有求助于应用程序，而是求助于近藤方法，这是一种非常流行的家庭整理技术，它依赖于我们的个人历史。在零售业的上层，重新强调专家选择是时尚精品店[开业典礼](https://www.theguardian.com/fashion/fashion-blog/2012/jul/23/opening-ceremony-london)和“未来超市”Eataly 等各种商店成功的背后原因。就像我们的媒体一样，我们已经从一个大规模行业选择的时代过渡到精细选择的时代。

策展可能是一个笨拙的，有时被诽谤的词，但它的拉丁词根 *curare* (照顾)，抓住了这种不可替代的人情味。我们想要惊喜。我们需要专业知识、独特的审美判断、明确的时间和精力支出。我们喜欢另一个人的品味和信任的个人关系的混乱现实。我们不仅仅想要相关性——我们想要一个为什么，一个叙述，这是机器无法提供的。即使我们将策展定义为选择和安排，这也不会只留给算法。与许多经历技术颠覆的行业(从无人驾驶汽车到自动化会计)不同，文化领域将永远重视人类的选择，即独特的视角。

在机器学习的世界里，这是艺术和人文学科反击的地方。这是新一代的工作。信息过载及其技术驱动的反应是我们这个时代的重大变革之一。但是在今天的饱和(以及那些摇摇欲坠的新书堆)中，知识和主观判断比以往任何时候都更有价值。用一位硅谷投资者的话说，“软件吃掉了世界”。嗯，软件是吃不了人策的。与神话相反，传统的看门人角色将继续存在。

我们将看到的是混合体:人类和机器的丰富融合，可以处理巨大的数据集，同时远远超出狭窄的范围。我们现在拥有如此多的东西——无论是书籍、歌曲、电影还是艺术品(更不用说数据了)——以至于我们无法独自管理所有这些东西。我们需要一种“算法文化”。然而，我们也比以往任何时候都更需要一些东西:人类的品味。
