# 产品病毒式传播的两种类型

每当介绍或讲授增长时，我都会讨论“产品病毒式传播”，因为我相信这是产品提高知名度和获得新用户的最佳方式之一。它是免费的，并且可以扩展。

我将产品病毒式传播定义为产品中的用户流和特性，这些用户流和特性**自然地**导致现有用户向他们的网络(朋友、同事、家人等)传播对产品的认知。).

我先说产品病毒不是什么:

*   激励或非激励推荐流程
*   一对
*   口口相传

产品病毒式传播的例子包括:

*   **Dropbox 文件夹共享:**你正在和团队成员一起做一个项目，你需要共享文件，所以你试图让他们加入进来。



Invite friends, coworkers or group members to join so you can share files with them via syncing folders.



*   **脸书照片标记:**你天生就想在照片上标记朋友。过去，如果你的朋友不在脸书，他们会收到一封电子邮件，表明他们被标记了，这将提示他们加入脸书，如果他们还没有加入的话。
*   **WhatsApp 群发消息:**你创建了一个 WhatsApp 群供你的密友聊天，所以你自然会试图让你所有的密友都加入这个应用程序，这样他们就可以参与进来。

在检查了各种例子后，我发现产品病毒可以进一步分为两类:

*   ***拉动产品病毒式传播(PPV):*** *产品病毒式传播现有用户需要其网络中的人加入，以从某个功能中获得价值。*
*   ***分销产品病毒式传播(DPV):*** *产品病毒式传播现有用户向他们的网络传播对产品的认知。*

以上面的例子为例，使用 WhatsApp 的群发消息和 Dropbox 文件夹共享功能，你需要某些人加入才能从该功能中获得主要价值，而使用脸书的照片标签功能，你不需要。收到照片中被标记的通知的人只是以一种非常上下文相关的方式意识到脸书。因此，WhatsApp 和 Dropbox 的功能将归入 PPV，而脸书的照片标签功能将归入 DPV。

通过 PPV，你的用户积极地试图说服他们网络中的人使用该产品来为自己谋利，而通过 DPV，你的用户只是让他们网络中的人知道该产品，并不一定从特定人的加入中获得价值。

PPV 的更多示例:

*   **Slack:** 你必须有同事加入你的渠道，才能获得能够与公司其他人高效沟通的价值。
*   **Splitwise:** 你需要你的朋友加入应用程序，这样你就可以输入与他们共享的账单、欠款和贷款。

*   Skype: 您需要您的朋友加入视频会议。

更多 DPV 的例子:

*   **Instagram 交叉发布:**你天生也想在脸书和推特上分享 Instagram 照片，所以他们为此提供了一种内嵌的无缝体验。当你在这些社交网络上的朋友看到 Instagram 链接或点击 Instagram 页面时，他们就会知道 Instagram。

*   **Nike+ Running 应用程序，跑步分享:**当你通过应用程序追踪到一次精彩的跑步时，你自然会想把它分享给你的社交网络。Nike+提供了与脸书等社交网络的独特集成，在共享的卡片中显示您的速度和跑步地图。当您的朋友看到您的活动时，他们会注意到 Nike+ Running 应用程序。

在这两种类型的产品病毒之间，PPV 在转换、激活和保留潜在新用户方面更有效，因为现有用户正在积极说服某人加入并使用该产品。

根据定义，拥有 PPV 也意味着你的产品有[网络效应](http://versionone.vc/network-effects/)，因为用户从更多用户的加入中获得价值。然而，反过来是不正确的，如果你的产品有网络效应，这并不一定意味着你有 PPV。例如，Reddit 有网络效应，但你不需要你的网络中的人加入才能从中获益，所以它没有 PPV。

DPV 通常为潜在的新用户提供上下文感知，这有助于增加他们的转化、激活和保留。然而，DPV 对这些漏斗下端的影响是 PPV 无法比拟的。尽管如此，DPV 在许多情况下扩大了受众范围，产生了更多的漏斗顶端认知度。

DPV 和 PPV 并不互相排斥，有些产品同时拥有这两者。例如，脸书像 DPV 一样有照片标签功能，但它也有活动和群组等功能，用户*需要*特定的人加入，从而产生 PPV。

当你在产品中设计病毒时，这两个因素都应该考虑。是否以及如何实现将取决于您的具体产品。为了成功地创造这些体验，你需要更深入地了解你的用户以及他们与非用户互动的相关情况。

这并不容易，一如既往地，产品的市场适应性仍然是增长之前的关键，但如果你能够让 PPV 和 DPV 以可扩展的方式工作，它应该会产生一些严重的增长。

*PPV 和 DPV 是我用来正式表达我对此的想法的术语，它对我有效，但我很乐意听到任何其他人看到或用来描述这些概念的类似术语和框架。
