# 红色、黄色、绿色:把个人签到带到办公室



两年前，我在 [Reboot.io](http://reboot.io) 从乡亲们那里得知了“个人签到”的事情。随后，我在 FullContact 召开的会议上介绍了这个概念。

现在，我们在员工会议、日常会议、场外会议和董事会议上签到。我甚至在合作伙伴会议、供应商会议和候选人面试期间签到！

我发现个人签到符合 FullContact 的“与人充分沟通，与人为善”的文化。

那么什么是“个人入住”这是一种简单但极其有效的方式来记录你做得如何，并与你的团队分享。

在繁忙的现代生活中，我们常常不会停下来问自己这些问题:

我的身体感觉如何？

我的心态如何？

我的呼吸怎么样？

我过得怎么样，真的？

签到是提问和回答这些问题的方式，其工作原理如下:

作为领导，我开始。这是至关重要的。通过以身作则，它更有影响力，我可以用自己的弱点来定调。

我首先指出我是红色、黄色还是绿色。

红色意味着我处于高度敏感的，蜥蜴脑的，情绪状态。这可能会令人难过。这可能是愤怒。这可能是情绪化的。这也可能意味着我的身体一团糟，或者我生病了或者受伤了。

黄色意味着我有点紧张。我的身体可能累了，或者可能有某种程度的情绪紧张。也许我的想法有点混乱，我分心了。

绿色意味着我很冷静。我的身体感觉很好。我觉得休息好了。我感觉我可以清晰地思考。

在颜色指示器之后，我接着解释颜色背后的原因。这是最重要的部分。它迫使我真正审视为什么我的身体和思想会处于现在的状态。

通常，我发现原因既有专业的，也有个人的。

我可能会对生活中的一大堆事情感到焦虑。

我可能会很平静，因为前一天晚上我花时间和我的孩子一起玩。

我可能累了，因为我旅行太多了。

我可能会难过，因为我家里有人正在经历痛苦和损失。

我可能会很兴奋，因为野马队进了超级碗。

原因多种多样，每个人都有自己的独特之处。

毕竟，正如我的朋友布拉德·菲尔德曾经说过的，“我们只是每天来工作的大袋化学品。”

在我完成后，其他人会四处走动，记录他们做得如何。

在这个过程中，我们保证不去评判对方。

这与你是什么颜色无关，这是一种给你自己和你的团队打分的方式。从根本上说，这是关于**同理心**和更好地理解你的队友，以便更好地一起工作。

它并不打算成为一个大的集体治疗会议。它旨在成为一种工具，供我们用来记录我们做得如何，了解根本原因，把它放在一边，开始工作！

我发现，虽然一开始有些尴尬，但团队会自然而然地适应它，并最终接受它。

这并不适合每一种公司文化，但我鼓励你尝试一周，看看它是否适合你！
