# 为什么我们应该致力于建立一个永久的公司，而不仅仅是安德鲁森的独角兽

“独角兽公司。”这是影响我们在湾区谈话的最新术语，以至于《华尔街日报》和《财富》都有[巧妙的信息图表](http://graphics.wsj.com/billion-dollar-club/)和[顶级公司名单](http://fortune.com/unicorns/)。在推介中，企业家被要求解释他们的新公司将如何成为下一个独角兽创业公司，科技媒体经常辩论一个热门的新团队是否会建立下一个独角兽。然而，这个词对企业家来说是再无意义不过的目标了。

毕竟，独角兽创业公司的定义是什么？只有一个达到 1B 的估值？谁在乎呢。我希望我们回到“十亿美元创业公司”而不是独角兽，以反映这个术语的真实性质，而不是一个矫情的外表。

这是终极的虚荣心指标，因为股东价值的 1B 美元仅仅是一个滞后指标，表明我们已经为这个世界创造了一些有用的东西。这本身绝不应该成为创办公司的目标。所以我们都不要再说独角兽了。我打电话给独角兽峰。让我们把重点放在建立有影响力的、持久的公司的投入上，在这些公司中，创造财富是出色工作的副作用。

相反，让我们来谈谈如何建立我们的**永久公司**。

当我第一次创业时，作为一个住在西雅图的注意力不集中的年轻技术人员，我对自称为“连续创业者”的人有一种非理性的崇拜，这些人在职业生涯中创建并出售了一批初创公司，即使它们很快就被卖掉了。创办多家公司的多样性似乎非常令人兴奋，尤其是当你年轻且缺乏目标的时候。然而，我在这个行业呆的时间越长，我就越钦佩那些创办和经营公司多年、几十年甚至一辈子的人。沃伦·巴菲特、理查德·布兰森、杰夫·贝索斯、马克·扎克伯格和其他人都属于这一阵营。

这些人已经开始并建立了他们的永久公司。这些公司也非常成功，但更重要的是，作为企业家，他们已经找到了毕生的工作。

毕竟，我们许多技术人员崇拜史蒂夫·乔布斯，因为他对设计的感觉和他的远见。有些人甚至模仿他的时尚。但你知道什么很难模仿吗？事实上，他十几岁时就开始从事硬件/软件产品的开发，并在接下来的 40 年里以这些想法为基础，直到他耗尽了时间。我们中有多少人能像那样对我们的工作奉献一生？

平衡
The forever company 是一家专注于企业家的公司，旨在平衡成为独角兽的财务目标。希望我们可以两者兼而有之！当然，我们都希望我们的公司有价值，并产生重大影响，但尽管独角兽关注的是企业家精神的产出，但永久公司的目标始于投入和正确的意图。

这与生活方式公司不同。贝佐斯将亚马逊作为他永远的公司，但这肯定不只是为了支持他的生活方式，而是为了产生比这大得多的影响。亚马逊在成为上市公司的过程中获得了数百万的风险投资资金，以充分抓住这个机会。当对一种生活方式的渴望与目标的“永远”部分相冲突时，就有另一种不同的问题了。那些对产品投资不足的人会给更聪明/更大/更有资金的竞争对手带来将他们挤出市场的危险，这是一家不会永远存在的生活方式公司！这种区别很微妙，但很重要。

这个术语不是我创造的。这种想法只能在深夜与我的姐姐和姐夫艾达和 T2 的深入交谈中产生，他们也在科技行业工作。有一天，他们顺便提到这是他们自己的一个有价值的目标，这个词真的引起了我的共鸣。很少有东西像它一样被卡住了，我希望它也能粘住我所有的读者。

**当永远的公司扩大规模时，当他们不扩大规模时** 有时永远的公司扩大规模，成为数十亿美元的公司。在许多情况下，一个永远的公司和一只独角兽是一样的，当市场很大，团队很有才华，还有一些好运气。这些是我们都想在硅谷创办并投资的公司。这些公司很容易被接受。

但有时，一个永远的公司只是意味着一生致力于一些可能永远不会变大的事情。这也是巨大的荣誉，我不禁钦佩那些追求这一目标的人。到目前为止，我们都已经看到了[二郎梦见寿司](https://en.wikipedia.org/wiki/Jiro_Dreams_of_Sushi)，他对寿司的热情和技巧令人难以置信。更接近科技的人可能是像 IDEO 的[大卫](http://www.ideo.com/people/david-kelley)T4 凯利这样的人，他在几十年前创立了自己的公司，虽然他们永远不会成为独角兽，但我想他一定会为他们在过去 25 年多的时间里所做的工作感到非常自豪。

最后，我想给你们留下我最近看到的一个关于 Jiro 的很棒的采访。他说只有到了 50 岁才感觉自己像个大师。关于手工与自动化的讨论很吸引人，也很吸引年轻一代的职业道德。我敢肯定，每个从事软件设计或工程的人都会有同感。
