# 揭秘数据货币化

公司已经发现，数据可以用于日常运营，以降低成本和增加收入。



今天运营的每一家公司都是数据公司。大多数公司都可以访问有关其供应链、运营、战略合作伙伴、客户和竞争对手的一系列数据。然而，大多数公司都把钱留在桌面上，只有十二分之一的公司最大限度地将数据货币化。数据本身是有价值的，但是来自数据的*洞察力*大大增加了这种价值。这些见解可用于指导各种活动，如客户细分、需求和流失预测、定价优化、保留营销和成本管理，并且在对外销售时还可以获得更高的利润。

实现数据货币化有两条主要途径。第一种是内部的，侧重于利用数据来改善公司的运营、生产力以及产品和服务，并与客户进行持续的个性化对话。第二条路径是外部的，包括通过向客户和合作伙伴提供数据来创造新的收入流。

这些途径并不相互排斥，有些公司两者兼而有之，如威瑞森、德国电信和西班牙电信等电信公司。他们通过使用数据优化运营和客户服务实现了内部盈利，并且他们还通过提供以下服务，在 B2B 客户和合作伙伴的各种使用案例中利用匿名和聚合的数据:

*   [针对零售商和旅游业的地理定位和地理围栏](https://geomarketing.com/geomarketing-101-what-is-geofencing)。
*   广告公司、政府机构、公共交通公司、城市规划者和医疗保健组织的交通流量和密度规划。
*   金融机构和信用卡公司的欺诈检测。
*   面向品牌和数字广告商的智能定位和点击流洞察。
*   零售店的位置、布局和员工规划。
*   面向各种公司的物联网(IoT)应用。

成功的数据货币化需要谨慎的方法，专注于符合公司整体业务战略的最高价值机会。

### 为数据货币化做准备

John Deere 是一家通过数据为自己和农民客户创造新收入来源和价值的公司。他们通过与康奈尔大学的合作，使用康奈尔大学的数据平台 [Ag-Analytics](https://ag-analytics.org/) 与 John Deere 的运营中心同步，以访问和分析农场数据。农民可以获得分析工具，如作物保险的估值器以及产量和风险管理的预测。该平台整合了公共数据源，包括土壤类型和天气。
