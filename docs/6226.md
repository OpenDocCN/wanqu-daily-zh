# 我们如何建立 Figma 设计团队

注意:这篇文章是不久前写的，作为一家快速发展的公司，我们的团队一直在不断发展壮大。要了解我们最新的团队成员，请访问 Figma 社区上的[文件。](/community/file/814575098768004426)

就在不到一年前，我们只有一个全职的核心产品设计师，我们中的一些人兼职做特色工作。很难跟上所有的工程师，我们知道我们必须快速招聘，以保持高质量的设计。但我们也想确保它是经过深思熟虑的。你加入一个小团队的每一个人都会对你的工作方式产生重大影响——如果你在一艘小船上划船，你会比在游轮上更注意到每一个划手。

快进一年。我们现在有一个由八名设计师组成的团队，一个更有经验、更成熟的合作评论文化，以及更清晰的目标和所有权范围。作为一个团队，我们仍然是新手，还有很多需要改进的地方，但我们很自豪能够有基础来共同构建设计工具的未来。

在 Figma，我们相信设计最好在户外进行。在花了一些时间来确保内部的透明文化之后，我们意识到我们想要与更广泛的设计社区分享我们的方法。为了把我们的钱用在我们的嘴上，我们认为这将是一个有趣的机会，提供一个深入了解我们如何建立我们的团队。

当我们第一次开始面试设计师时，我们遭受了不明确的角色期望，面试问题中的不一致以及关于候选人的不集中的辩论。有很大的改进空间，但作为一个早期的创业公司，我们都忙于戴不同的帽子，以至于一开始放慢脚步重新思考这个过程感觉有点困难。

幸运的是，Niko 参加了 [Joel Califa](https://twitter.com/notdetails) 举办的关于建筑设计团队的精彩而及时的讲座，这给了我们一个新的工具来深思熟虑地指导我们的过程。正如 Joel 所解释的，它分为三个阶段:

1.  **学习:**尽可能的看每一个职位描述。向已经建立团队的人学习(由其他领导提问)。了解哪些技能或特质对你的团队有价值。
2.  分析:利用这种理解来建立一个全面的技能列表。筛选出公司真正需要的技能。筛选出设计团队应该负责的技能。然后是每个设计师都需要知道的基本技能。
3.  **进化:**你是否意识到你的团队存在新的差距？公司或团队的需求有变化吗？你现在需要什么？可能的话，在内部培养员工。

按照这个指南，我们独立地写下了我们对下一个团队成员的期望，然后开始在 Figma 中合并和巩固这个列表以建立共识。它看起来像这样:

然后，我们对面试的文化部分进行了类似的练习，以确保我们带来的人非常适合 Figma，符合公司的核心价值观(大胆、包容、有趣)。



我们最近参加了一个多样性和包容性研讨会，解释了人们在面试中如何无意识地扭曲标准以适应他们现有的偏见。这适用于软技能和硬技能——例如，如果你认为优秀的设计师很有主见，你可能会寻找它，即使你作为一个团队同意它对这个角色并不重要。为了帮助解决这个问题，我们列出了我们想要的品质，并将其直接用于 Lever(我们的招聘和雇佣工具)的面试反馈表中，以促进有意义的重点反馈，而不仅仅是一个开放式的文本字段。



We added the requirements directly into our feedback tool, Lever, for clearer assessments 



我们也根据现有的标准来判断自己。我们希望这样做可以揭示我们团队中的新差距，从而找到完善我们所缺失的东西的方法，但实际上并没有太多的惊喜。尽管如此，它仍然是有价值的，因为我们不想在我们不能首先评价自己的事情上评价别人。这也是一次令人谦卑的经历，确保了我们都能实事求是地看待自己的优势和劣势。

当然，仅仅是招聘活动不会神奇地把你需要的人放进你的团队——要找到合适的人选，还需要做大量的工作，比如寻找合适的人选、四处打听、与人会面。但至少现在我们知道了我们实际上需要什么来正确地进行搜索。对我们帮助很大的一个技巧是与我们出色的招聘人员卡玲密切合作，让她融入设计团队，参加我们的评论和团队会议，这样她就可以帮助我们聘用合适的人才，并以第一手经验更准确地讲述我们的日常工作。有兴趣和你的团队一起运作吗？试用[这个模板](/community/file/766883810351845384)。



首先，Lucyé(Lucy)在网上向我们提出了申请，当我们看到她的作品集时，我们被她优美的动作和视觉探索惊呆了。她曾在多个领域工作过，从制作网站、在代理机构工作到为第一台手持数控机床[设计界面。她是我们见过的最好奇的人之一，总是希望了解更多关于事物为什么是这样的——我们很幸运有她在船上。](https://www.shapertools.com/)

然后是 Marcin——一位在 Google、Medium 和 Code for America 工作过的 UI 老手。从[将吃豆人复制成谷歌涂鸦](https://www.google.com/doodles/30th-anniversary-of-pac-man)(一直到匹配街机游戏中的 bug)，到关心[的小细节，如媒体上的下划线](https://medium.design/crafting-link-underlines-on-medium-7c03a9274f9)和[引号](https://medium.design/quotation-marks-c8993b54417c)，到构建一个[定制城市规划应用程序](https://medium.com/@mwichary/youre-a-street-designer-you-just-dont-know-it-yet-b5e83620e428)，我们知道拥有一个对高质量工艺、工具和排版充满热情的人对于 Figma 这样的设计工具来说是完美的。

很难找到比一个花了 10 年时间思考和构建创造性工具的人更合适的人选了。Bradee 的大部分任期是在 Adobe 度过的，大部分时间是在 Photoshop 上。她热衷于开放式设计和创造工具，帮助我们创造一个更加包容的世界。她喜欢新颖的互动，曾在 WWDC 与苹果公司一起展示 Photoshop touchbar。我们喜欢她在工作内外的创造性表达——她最近设计并实现了一个定制的万圣节动画角色，在一个带有摄像头和语音装置的大投影屏幕上与路人互动。

最后，作为团队的最新成员，我们很幸运地找到了 Tori 作为我们的第一位创意总监，他将帮助我们发展 Figma 品牌。她以前是一名自由职业艺术总监和设计师，曾与苹果、Etsy、谷歌和许多其他公司的客户合作。Tori 拥有出色的领导方法，认真倾听并理解团队的目标和动机，共同打造产品。作为一个非常关心授权他人的人，我们知道她会成为我们沟通设计团队的优秀管理者和领导者。她还制作史诗般的渐变效果。
