# 谷歌如何工作

# 谷歌的早期

谷歌早期的排名算法 PageRank 的想法源于爱因斯坦。拉里·佩奇和谢尔盖·布林是斯坦福大学的学生，他们注意到科学研究经常引用著名论文，如相对论。这些引用几乎就像投票一样——你的作品被引用得越多，它就越重要。如果他们下载每一篇科学论文并查看参考文献，他们可以从理论上决定哪些论文是最重要的，并对它们进行排序。

他们意识到，由于链接，互联网可以以类似的方式进行分析和排名，只是他们可以使用链接而不是引用。因此，他们开始尝试“下载”(或抓取)整个互联网，找出哪些网站链接最多。理论上，链接最多的网站就是最好的网站。如果你搜索“大学”，他们可以看到关于“大学”的网页，并对其进行排名。

# 今日谷歌

谷歌今天的工作方式基本相同，尽管更加复杂和微妙。例如，并非所有链接都具有相同的权重。来自权威网站的链接(从一个网站有多少链接指向它来看)比来自非权威网站的链接更有价值。《纽约时报》的一个链接可能比不太权威的网站的一万个链接更有价值。

说到底，谷歌的目的是为你在搜索栏中输入的单词找到“最好的”(或最受欢迎的)网页。

所有这一切意味着我们需要让谷歌清楚我们的页面是关于什么的，然后清楚地表明我们是受欢迎的。如果我们这样做，我们就赢了。为了做到这一点，我们将遵循一个非常简单的过程，这个过程每一次都比你想象的要省力。

# 玩弄系统

谷歌是一家非常聪明的公司。他们编写的算法的复杂程度令人难以置信；请记住，目前有一些由谷歌算法驱动的汽车在硅谷自动驾驶。

有些东西短期内有效，但谷歌很聪明，而且越来越聪明。打败谷歌的难度与日俱增，而谷歌关闭垃圾网站的速度也与日俱增。大多数人甚至坚持不了一周，你所做的一切就会消失，你的工作也会蒸发。那不是你应该做事情的方式。

我们将专注于在互联网上建立公平，而不是基于互联网的搅动和燃烧。因此，如果你看到一些高薪 SEO 顾问告诉你使用软件和旋转内容来生成链接，或者当你看到一些黑帽匠攻击系统时，就知道这不值得。我们会建立权威，快速获得流量，但我们会以一种不会在未来消失或削弱你的网站的方式去做。

## 长尾理论

我们需要熟悉一个叫做“长尾”的概念

如果我们用“流行度”作为 Y 轴，等级顺序作为 X 轴来绘制大多数事物的“流行度”,我们会得到类似幂律图的东西:



有一些大的点击获得了大多数的关注，在几次点击之后，图表急剧下降。长尾理论认为，随着我们作为一个社会变得越来越多样化，上图的黄色一端将永远延伸并变得更高。

想想亚马逊。他们可能有一些畅销产品，但他们的大部分零售收入来自各种各样的东西，这些东西在任何地方都不如他们的畅销产品那样常见。同样，如果我们对过去 10 年播放的歌曲的受欢迎程度进行排名，将会有一些热门歌曲获得大多数播放，而大量歌曲只有少数播放。那些不太受欢迎的产品和歌曲就是我们所说的*长尾*。

在 SEO 中，这很重要，因为至少在开始时，我们会追求长尾关键词——非常精确的，意图驱动的关键词，竞争较低，我们知道可以获胜，然后我们会逐渐向左移动。

我们的网站不会在一开始就超越极具竞争力的关键词，但是通过更加具体，我们可以开始用更少的努力赢得非常有针对性的流量。

我们要找的关键词我们称之为“长尾关键词”

# 页面搜索引擎优化

既然我们已经选择了关键词，我们需要确保谷歌知道我们的网站是关于什么的。这就像确保正确的关键字出现在正确的位置一样简单。这大部分与 html 标签有关，html 标签构成了网页的结构。如果你不知道 html 或者不明白它是如何工作的，只要把这个列表交给一个开发者，他们应该能帮助你。

这里有一个简单的清单，你可以遵循，看看你的内容是否得到优化。

# 页面搜索引擎优化清单

**☐** 你的关键词在< title >标签中，最好是在标签的前面(或者靠近前面)

**☐** 你的关键词接近于<标题>标签的开头(理想情况下是第一个单词)

**☐** 标题标签包含的可视字符少于 65 个(可选，但推荐)

**☐** 你的关键词在第一个< h1 >标签中(你的页面有一个< h1 >标签)

**☐** 如果你的页面包含额外的标题标签(< h2 >、< h3 >等)，你的关键词或同义词会出现在大多数的标题标签中

**☐** 页面上的任何图像都有一个包含你选择的关键字的< alt >标签

☐ 你的关键词在元描述里(而且有元描述)

☐:这一页至少有 300 字的文字

**☐** 你的关键词出现在网址中(如果不是主页)

**☐** 你的关键词出现在文案的第一段

☐:你的关键词(或者同义词——谷歌现在可以识别)会在页面的其他地方出现

**☐** 你的关键词密度在 0.5%到 2.5%之间

☐页面包含了其他页面的 dofollow 链接(这只是意味着你没有使用 nofollow 链接到其他页面)

**☐** 该页面是原创内容，并非取自其他页面，与你网站上的其他页面也不相同

如果你把这些都准备好了，那么从页面的角度来看，你应该已经准备好了。除非你在一个竞争非常激烈的领域，否则你很可能是你所选择的关键词的最佳优化页面。

我们现在剩下的就是离页优化了。

# 离页搜索引擎优化

离页搜索引擎优化只是说*链接*的一种花哨方式。(有时我们称它们为*反向链接*，但这其实是一回事。)

谷歌将网络上的每个链接视为加权投票。如果你链接到某个东西，在谷歌看来，你是在说，“这值得一查。”你越合法，你的选票就越有分量。

# 链接果汁

SEO 用一种怪异的方式来描述这个投票过程；他们称之为“链接果汁”如果一个权威网站，比如维基百科，链接到你，他们会给你“链接果汁”

但是 link juice 并不仅仅适用于站点对站点——如果你的主页非常权威，并且链接到你站点的其他页面，那么它也会传递 link juice。因此，我们的链接结构变得非常重要。

## 检查链接果汁

有很多工具可以让你检查有多少链接指向一个站点，以及这些页面的权威性。不幸的是，没有一个是完美的——知道什么链接指向你的站点的唯一方法是抓取那些页面。

谷歌每天都会抓取一些最受欢迎的页面，但是他们不希望你操纵它们，所以他们更新索引的速度非常慢。

也就是说，你至少可以在谷歌搜索控制台(以前称为网站管理员工具)中查看谷歌索引的样本。一旦你导航到你的网站，在左侧选择“搜索流量”，然后“链接到你的网站。”关于这是否真的向你展示了谷歌知道的所有链接(我 99%确信这只是一个样本)，有一场激烈的争论，但它至少是一个有代表性的样本。

要查看您的所有链接，请点击“谁与您的链接最多”下的“更多”，然后“下载此表”这似乎只是下载了谷歌所知的一个样本。你也可以选择“下载最新链接”,它比其他选项提供更多的最新链接。

不幸的是，这并没有让我们看到多少链接的价值，也没有告诉我们哪些链接已经掉了，或者这些链接是从哪里来的。

要使用这些工具，有很多种:如果你有预算，我会选择 ahrefs.com，因为他们有最大的索引，其次是 Moz 的 Open Site Explorer(大部分数据可以通过免费帐户获得，如果没有，那么它比 ahrefs 略便宜)，最后是 SEMrush，它对我们需要的大多数目的都是免费的。MajesticSEO 使用了“信任流”和“引用流”的组合，它也能很好地给你一个整体健康状况和指向你站点的链接数量的概念。

所有这些都使用不同的内部指标来确定链接的“权威”，但使用它们来比较苹果可能是有益的。

# 链接结构

HTML 链接看起来像这样:

[http://www.somesite.com](”<a)" title = " keyword ">锚文本< /a >

其中[http://www.somesite.com](http://www.somesite.com)是链接指引你去的地方，标题很大程度上是时间流逝的遗迹，链接的文字——想想你点击的蓝色文字——被称为“锚文本”

除了一个页面的链接流量，锚文本的相关性也很重要。

一般来说，只要有可能，你就应该使用你的关键词作为内部链接的锚文本。外部链接(来自其他网站)不应该对锚文本进行大量优化。如果你 90%的链接都有相同的锚文本，谷歌可以抛出一个红旗，假设你在做一些可疑的事情。

如果你曾经创建过链接(就像我们以后会展示给你的)，我只会使用一些通用的东西，比如网站名称，“这里”或者完整的 URL。

# Robots.txt、disavow、nofollow 和其他细节

大多数 SEO 都是管理可能出错的东西。有很多这样的东西，但是我们会检查一下什么可以覆盖 99%的需求，如果有什么真的很疯狂，你可以谷歌一下。

**Robots.txt**

几乎每个网站在 url.com/robots.txt 都有一个页面——甚至谷歌也有。

这只是一个纯文本文件，让你告诉搜索引擎爬虫什么爬行和不爬行。大多数都很善于倾听，除了 Bingbot，不管你告诉它什么，它都会做它想做的事情。(我多半是开玩笑的。)

如果你不想让谷歌抓取一个页面(可能是你不想索引的登录页面，登陆页面等。)你可以在 robots.txt 中说 disallow: /somepage 来“禁止”它。

如果你添加一个尾随/到它(例如 disallow: /somepage/)它也将禁止所有的子页面。

从技术上讲，您可以为不同的机器人(或用户代理)指定不同的规则，但是如果您不需要单独的爬行规则，那么用“User-agent: *”开始您的文件是最简单的。

**否认**

谷歌将惩罚垃圾网站，不幸的是，这导致了一些不良行为者的不良行为。比方说，你想干掉一个竞争对手。你可以发送一堆明显的垃圾链接到他们的网站，让他们受到惩罚。这被称为“负面搜索引擎优化”，这是在高度竞争的关键词中经常发生的事情。谷歌一般会试图假装这件事没有发生过。

然而，在这种情况下，你可以在搜索控制台中“否认”链接，这相当于说，“嘿，谷歌，不要算这个。”我希望你永远也不会用到它，但是如果你雇佣了(或者已经雇佣了)一个糟糕的 SEO 或者被竞争对手攻击，这就是你如何对抗它。

**不跟随**

链接可以有一个名为“nofollow”的属性，如下所示:

[http://www.somesite.com](”<a)" title = " keyword " rel = " no follow ">锚文本< /a >。

如果你想链接到某人，但你不想把它算作投票(你不想传递链接汁)，或者你支持用户生成的内容，并希望阻止垃圾邮件发送者，你可以使用 nofollow 链接。谷歌表示，它对这些链接的价值打了折扣。我不相信他们会大打折扣，但其他 SEO 也是如此，如果没有别的，他们似乎可以阻止垃圾邮件发送者。

**重定向**

如果你要改变一个网址，但你不希望它的链接消失，你可以使用 301 重定向。A 301 将通过大部分链接果汁。

**规范的网址**

如果你有两个几乎相同的页面，你可以加上类似<link rel="”canonical" href="”<a" class="au le" target="_blank">https://www.someurl.com/somepage>这样的话来说“嘿，把这一页当成那一页，但是我不想把它 301。"

就这样，我们准备好建立我们的第一个链接。

# 链接建筑

链接建设是 SEO 真正开始起作用的地方，也是很多人最终陷入痛苦的地方。

建立链接的最好方法就是不建立链接。我过去曾为一些公司工作过，这些公司不需要你去要求，它们只是从媒体、客户博客、他们令人敬畏的博客帖子等渠道源源而来。如果这是一种选择(我们将通过几种方法使其更有可能)，你就在一个很好的地方。

如果没有，至少在开始时，我们将手动创建几个。

我们将以合法的方式创造它们，而不是在印度雇人来做。这是导致灾难的一个因素，我甚至都数不清我见过多少次这种情况。

# Web 2.0s

建立高质量链接最简单的方法是 SEO 所谓的“web 2.0s”，这只是“社交网站”或让你发布内容的网站的一种说法。现在在推特上发一个链接到深渊不会对你有任何影响，但是会影响个人资料、状态页面等等。一定要有分量。如果它们来自一个被认为是链接的热门域名。

一些最简单的是:

YouTube(视频的描述——它必须有实际的浏览量)

Wordpress(是的，你必须实际创建一个博客)

博客作者(此处相同)

Tumblr

基于投票的网站(黑客新闻，增长黑客，Inbound.org 等)。)

如果没有别的，你可以从那里开始，得到半打到十几个链接。你可以在网上找到很多“web 2.0s”，但是请记住，如果你想在博客平台上有所建树，你必须*真的*有所建树。这需要大量的内容和时间，但是你必须用正确的方法去做。

我们通常在这里保留一个更大的 Web 2.0s 列表。有些可能已经过时了，但是不管怎样，你应该只构建半打到十几个 Web 2.0s。

# 过期的域

另一种获得链接果汁的方法是购买过期域名。这个比较难做，但是有很多选择，比如 expireddomains.net。(谷歌“过期域名”，你会发现许多网站在监控它们。)

您会想要购买一个已经过期的域名，并使用存档将它尽可能恢复到原始状态。这些网站可能有一些链接果汁传递，你可以把它传递给自己。

# 链接交叉点

另一种查找链接位置的方法是使用链接交叉工具。这些网站会找到链接到“竞争对手 a”和“竞争对手 b”的网站，但不会链接到你。理论上，如果他们链接到你的两个竞争对手，他们应该愿意链接到你。Moz、Ahrefs、LunaMetrics 和其他公司都有运行良好的链接交集工具。

如果你做了这两件事，将会发生一些变化:首先，你会注意到人们点击你的链接*，因为你是一个喜欢贡献*的有思想的人。第二，人们会尊重你的公司*,因为你是一个有思想的人，喜欢做出贡献。*
