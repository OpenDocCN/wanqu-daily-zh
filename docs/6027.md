# 强技术和弱技术

> 原文:[http://cdixon.org/2019/01/08/strong-and-weak-technologies/?UTM _ source = Wanqu . co&UTM _ campaign = Wanqu+Daily&UTM _ medium = website](http://cdixon.org/2019/01/08/strong-and-weak-technologies/?utm_source=wanqu.co&utm_campaign=Wanqu+Daily&utm_medium=website)

> 在 2007 年的一次[媒体旅行](https://www.businessinsider.com/steve-jobs-reaction-first-iphone-2015-9)中，史蒂夫·乔布斯向记者展示了这款设备，有一次一名记者批评了 iPhone 的触摸屏键盘。
> 
> 记者说:“这不管用。”。
> 
> 乔布斯停了一会儿，歪着头。记者说，他或她一直打错字，按键对他或她的拇指来说太小了。
> 
> 乔布斯笑了笑，然后回答道:“你的拇指会学习的。”

当 iPhone 在 2007 年推出时，它让竞争对手感到困惑，因为它不是为这个世界而生的。无线网络太慢了。智能手机用户只知道如何使用物理键盘。没有软件开发商为触摸屏手机开发应用程序。它经常掉线。

但是 iPhone 是如此非凡的设备——粉丝们称之为“耶稣手机”——以至于全世界都适应了它。运营商增加了无线容量。开发者发明了新的应用和界面。用户学会了如何在触摸屏上快速打字。苹果不断发布更好的版本，修复问题，增加新功能。

智能手机是更广泛的历史模式的一个很好的例子:技术通常成对出现，一种强形式和一种弱形式。以下是一些例子:

| **强** | **虚弱** |
| --- | --- |
| 公共互联网 | 私有内部网 |
| 消费者网络 | 互动电视 |
| 众包百科全书(维基百科) | 专家策划的百科全书(例如，Nupedia、Encarta) |
| 众包视频(YouTube) | 媒体公司的视频技术(例如 RealPlayer) |
| 互联网视频聊天(Skype) | 网络电话(例如 Vonage) |
| 流媒体音乐(Spotify) | MP3 下载(例如 iTunes) |
| 带完整操作系统和应用商店的触摸屏智能手机(iPhone) | 带物理键盘的有限应用智能手机(如黑莓) |
| 全电动汽车(特斯拉) | 混合动力车 |
| 加密货币驱动的无许可区块链 | 许可/私人区块链 |
| 公共云 | 私有云/混合云 |
| 基于应用的媒体公司(如网飞) | 有线电视公司提供的视频点播 |
| 虚拟现实 | 增强现实 |
| 电子竞技 | 通过互联网传送的传统体育 |

强大的技术抓住了技术爱好者的想象力。这就是为什么许多重要的技术都是从周末爱好开始的。狂热分子用他们的时间投票，与大多数商业世界不同，他们有长远的眼光。他们从基本原则出发，充分利用现有资源来设计他们应该存在的技术。有时候，这些狂热分子经营着大公司，在这种情况下，他们通常像史蒂夫·乔布斯(Steve Jobs)一样，是有着庄重和远见的创始人，能够进行大规模的长期投资。

主流技术世界注意到了这种兴奋，想要加入进来，但不愿意一路走下去，拥抱强大的技术。对他们来说，强大的技术似乎是奇怪的、玩具般的、不严肃的、昂贵的、有时甚至是危险的某种组合。所以他们接受了弱形式，一个妥协的版本，看起来更熟悉、更高效、更严肃、更安全。

强大的技术通常按照 Perez/Gartner 炒作周期发展:

[![researchmethodology illustration hype cycle](../Images/ab9ea717d5bbe31a5f51838dc986f529.png "researchmethodology illustration hype cycle")](/static/153f5948062ac4cc6f9e4f79607be64b/58232/researchmethodology-illustration-hype-cycle.jpg)

在幻灭的低谷期，投资强技术的企业家和其他人有时会失去信心，将注意力转向弱技术，因为弱技术似乎更接近主流采用。这通常是一个错误。

也就是说，弱形式的技术也可以成功。比如很有可能增强现实会很重要，在网上看传统体育会很流行等等。

但是最终定义新时代的是强大的技术。乔治·萧伯纳关于人的话也适用于技术:

> 理智的人使自己适应世界:不理智的人坚持试图使世界适应自己。因此，所有的进步都取决于不讲道理的人。

弱技术适应当前存在的世界。强大的技术让世界适应自己。进步依赖于强大的技术。你的拇指会学习。
