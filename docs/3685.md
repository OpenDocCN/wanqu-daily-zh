**成长黑客的崛起**
“[成长黑客](http://startup-marketing.com/where-are-all-the-growth-hackers/)的新头衔正在将自己融入硅谷的文化，强调编码和技术能力现在是成为一名优秀营销人员的重要组成部分。增长黑客是营销人员和编码人员的混合体，他们着眼于传统的问题“我如何为我的产品获得客户？”以及带有 A/B 测试、登录页面、病毒因子、电子邮件送达率和开放图表的答案。在此基础上，他们将直接营销的原则分层，强调定量测量、通过电子表格的场景建模和大量的数据库查询。如果一家初创公司处于产品/市场前期，成长型黑客可以确保病毒式传播嵌入到产品的核心。在产品/市场匹配之后，他们可以帮助提高已经在工作的产品的分数。

这不仅仅是一个角色，整个营销团队都被打乱了。成长型黑客不是一群向他们汇报的非技术营销人员，而是领导工程师团队的工程师。将您的产品整合并优化到一个大平台的过程需要模糊营销、产品和工程之间的界限，以便他们共同努力，打造产品市场本身。像邮件送达率、页面加载次数和脸书登录这样的项目不再是技术或设计决策，而是赢得市场的进攻性武器。

**赌注是巨大的，因为“超级平台”可以接触到 1 亿多消费者** 这些技能是无价的，可以改变新产品的轨迹。有史以来第一次，新产品有可能在短短几年内从零增长到数千万用户。很好的例子包括 Pinterest、Zynga、Groupon、Instagram、Dropbox。每周都有令人难以置信的新产品出现。这些拥有数百万用户的产品建立在新的开放平台之上，而这些平台又拥有数亿用户——尤其是脸书和苹果。1995 年，网络只有 1600 万拨号用户，而今天，超过 20 亿人访问互联网。除了这些前所未有的数字，消费者使用超级病毒式的沟通平台，迅速加快新产品的扩散——不仅市场更大，而且发展速度也更快。

在这个时代之前，市场营销学科依赖于能够接触到数千万人的唯一沟通渠道——报纸、电视、会议和零售商店等渠道。为了与这些沟通渠道沟通，你需要借助人们——广告公司、公共关系、主题演讲和业务拓展。如今，传统的沟通渠道已经支离破碎，过时了。传播你的产品的最快方式是通过使用 API，而不是 MBA，在一个平台上发布。业务开发现在是以 API 为中心，而不是以人为中心。

虽然公关和媒体曾经是获取客户的驱动力，但现在它是一个滞后的指标，表明你的脸书整合正在发挥作用。营销副总裁的角色，长期以来被认为是一个非技术性的角色，正在迅速消失，取而代之的是一种新型的营销人员/编码人员的混合体。

**Airbnb，案例研究** 让我们用 Airbnb 的案例来说明这种心态。首先，回忆一下[傻逼点击率定律](http://andrewchenblog.com/2012/04/05/the-law-of-shitty-clickthroughs/):

> 久而久之，所有的营销策略都会导致糟糕的点击率。

这个定律的反过来是，如果你是第一个进入市场的人，或者也可以说是第一个进入营销渠道的人，你可以获得很高的点击率和转换率，因为它新颖且缺乏竞争。这对知道自己在做什么的成长团队来说是一个极具吸引力的机会——他们可以在一个大平台上进行相当困难的集成，并期望在早期获得优势。

Airbnb 正是这样做的，它与 Craigslist 进行了出色的整合。他们选择了一个拥有数千万用户、自动化工具相对较少的平台，并创造了一个分享 Airbnb 房源的绝佳体验。它简单而深入地集成到产品中，是我多年来见过的最令人印象深刻的特别集成之一。当然，传统的营销人员不会想到这一点，或者知道这甚至是可能的——相反，它需要一个有营销头脑的工程师来剖析产品，并建立一个如此平滑的集成。

看起来很简单，对吧？令人印象深刻的是，这是在没有公共 Craigslist API 的情况下完成的！事实证明，为了完成这样的整合，你必须仔细观察 Craigslist。请注意，对我来说，对已经工作的东西进行逆向工程要比提出参考实现容易 100 倍——因此，我对这种集成印象深刻。

**逆向工程“向 Craigslist 发帖”**
你要做的第一件事就是看看 Craigslist 是如何允许用户向网站发帖的。如果没有 API，你必须编写一个脚本来抓取 Craigslist 并与它的表单交互，以预填充你想要的所有信息。

玩 Craigslist 时，你能注意到的第一件事是，当你发帖时，你会得到一个保存你所有信息的唯一网址。所以如果你去 https://post.craigslist.org，你会被重定向到一个不同的网址，看起来像 https://post.craigslist.org/k/HLjRsQyQ4RGu6gFwMi3iXg/StmM3?的 T2s =类型。原来这个 URL 是唯一的，所有进入这个列表的信息都与这个 URL 相关联，而不是与您的 Craigslist cookie 相关联。这与大多数网站的做法不同，后者将大量信息保存在 cookie 和/或服务器端，然后取出。这种将 Craigslist 数据和 URL 关联起来的独特方式意味着，您可以构建一个机器人来访问 Craigslist，获取一个唯一的 URL，填写列表信息，然后将 URL 传递给用户，以进行发布的最后一步。这成为整合的基础。

与此同时，机器人需要知道处理所有表单的信息——除了填写 Craigslist 类别(这很简单)，你还需要知道选择哪个地理区域。为此，你必须访问他们服务的每个市场上的每个 Craigslist，并收集每个地区的名称和代码。幸运的是，你可以从 Craiglist 侧面板的链接开始——事实证明，Craigslist 有 100 多个不同的版本。

![](../Images/b1ed62ff0853132ee83d26380af960bb.png "Screen Shot 2012-04-26 at 7.10.34 PM")

如果你稍微挖掘一下，你会发现某些地理市场比其他市场更详细。在一些，像旧金山湾区，有分区(南湾，半岛等。)和社区(伯纳尔，太平洋高地)，而在其他市场只有分区，或者只有市场。所以你必须将所有这些整合到你的界面中。

然后是列表本身的问题——默认情况下，Craigslist 会给你一个匿名的电子邮件地址，你可以用它来和潜在客户交流。如果你想把他们带到你的网站，你必须注意到你可以关闭显示电子邮件，而只提供“联系我”链接。或者，你可以填写一个特殊的电子邮件地址，比如**listing-29372@domain.com**，它会自动将询问转给正确的人，这可以通过使用 [Mailgun](http://mailgun.net) 或 [Sendgrid](http://sendgrid.com) 等服务来完成。

最后，你会希望列表看起来不错——原来 Craigslist 只有[支持有限的 HTML](http://www.craigslist.org/about/help/html_in_craigslist_postings/) ,所以你需要努力让你的列表在这些限制下工作良好。

完成整合只是开始，一旦开始，你就必须对其进行优化。一旦开始在 Craigslist 上分享他们的列表，完成率是多少？你如何改变流程、行动号召、表格中的步骤来提高这个百分比？同样，当人们从 Craigslist 登陆时，你如何确定他们可能会完成交易？他们需要特殊的信息吗？

跟踪所有这一切需要额外的工作，用唯一的网址进行点击跟踪，Craigslist 列表上的 1×1 gif，以及更多的细节。

长话短说，这种融合不是小事。有许多小细节需要注意，如果最初的集成需要一些非常聪明的人花很多时间来完善，我不会感到惊讶。

没有一个传统的营销人员会想到这一点
说实话，一个传统的营销人员甚至不会想到上面的整合——要实现它需要太多的技术细节。因此，这只能是一个负责从 Craigslist 获取更多用户的工程师的想法。谁知道 Airbnb 从这种整合中获得了多少价值，但在我看来，这令人印象深刻。它利用低竞争、大容量的营销渠道，并在产品中深入构建营销功能。最重要的是，这对所有参与者来说都是一个双赢的局面——既包括通过挖掘预建需求出租房屋的人，也包括租房者，他们可以看到更好的房源，有更好的照片和描述。

这只是一个案例研究，但通过这种类型的整合，新产品不仅能够在功能上竞争，还能在分销策略上竞争。这样，两个相同的产品可以有 100 倍不同的结果，仅仅基于它们与 Craigslist/Twitter/脸书的整合程度。这是一个令人惊奇的时代，一批新的有创造力的技术营销人员正在出现。看这个趋势。

所以总结一下:

*   有史以来第一次，像脸书和苹果这样的超级平台独一无二地提供了对数千万用户的访问
*   营销学科正在从以人为中心转向以 API 为中心的活动
*   成长型黑客体现了在平台时代蓬勃发展所需的营销人员和编码人员的混合体
*   Airbnb 有一个惊人的 Craigslist 整合

祝你们好运，成长黑客！
