# Slack 的产品路线图从 Twitter、Google Docs 获得灵感

> 原文:[http://venturebeat . com/2014/09/26/slacks-product-roadmap-looks-to-Twitter-Google-docs-for-inspiration/？UTM _ source = Wanqu . co&UTM _ campaign = Wanqu+Daily&UTM _ medium = website](http://venturebeat.com/2014/09/26/slacks-product-roadmap-looks-to-twitter-google-docs-for-inspiration/?utm_source=wanqu.co&utm_campaign=Wanqu+Daily&utm_medium=website)

越来越多的以企业为中心的软件公司，如 Dropbox、Convo、Google 和微软，正在开发允许人们协作的产品。

现在，Slack，一个面向企业的对话平台，正在通过最近购买 Spaces 的工具来建立自己的协作功能，这种工具可以让人们同时远程处理同一份文件。

Slack 成立于 2009 年，但直到今年 1 月才正式推出。该公司自成立以来已经筹集了 6000 万美元，但 Spaces 是它的第一笔收购。虽然它主要是一个通信工具，但它的口号“一切都在一个地方”表明了更高的愿望。

Slack 拥有与 Twitter、Dropbox、Google Docs 以及“数十种”其他服务整合的能力。现在有了 Spaces，它正慢慢走向与谷歌文档和微软 365 在数字文档领域的竞争。

Slack 的一名代表在谈到最近的收购时说:“这是制定计划和规格、进行产品审查、共享文件和创建交付时间表的完美工具，这只是几个例子而已。”。从很多方面来说，这只是 Slack 的最新合作。

Slack 今年有了很大的增长。据《华尔街日报》报道，该公司大约有 20 万用户，其中 6 万是付费用户。但是面对像谷歌(拥有谷歌文档)和微软(拥有微软 365)这样的巨头是一个相当高的要求。事实上，作为一个交流平台，该公司已经与同样属于微软的 Convo 和 Yammer 展开了激烈的竞争。

为了获得更好的知名度，Slack 可能会从谷歌文档(Google Docs)和推特(Twitter)等时尚产品中汲取灵感，将自己定位为更现代的公司。Slack 告诉我们轻量级消息线程(类似于 Twitter 所做的)，这将允许你回复/评论/引用消息，计划很快开发。尽管线索化对话并不新鲜。Convo 和 Yammer 都提供线程对话和文档支持。

但是空间的获得不仅仅是文档支持，它还是一个文档平台。

还有一点需要记住:今年早些时候，Slack 筹集了 4200 万美元的资金。因此，这可能只是帮助 Slack 建立“一切在一个地方”平台的一系列收购的第一步。
