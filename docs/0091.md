## 应用哲学

Appium 的理念是测试原生应用不需要包含 SDK 或者重新编译你的应用。并且您应该能够使用您喜欢的测试实践、框架和工具。Appium 是一个开源项目，已经做出了设计和工具决策，以鼓励一个充满活力的贡献社区。
