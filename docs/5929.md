# 为什么所有网站看起来都一样？

## 互联网缺乏想象力，所以我让我的学生重新设计它



今天的互联网平淡无奇。一切看起来都一样:通用的字体，没有可言的布局，可互换的页面，缺乏富有表现力的视觉语言。即使是微排版也是一团糟。
