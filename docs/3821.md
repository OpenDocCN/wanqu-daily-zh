# 为什么这项工作还没有被机器处理？风险投资

每当我告诉人们我从事人工智能工作时，他们往往会感到有必要接过全世界人类工作者的衣钵。确切的论点往往各不相同，但通常可以归结为某种形式的“机器无法完成人的工作，因为它们带来了任何机器都无法企及的‘人类接触点’。”

在参加完其中一场辩论后，我在从国外返回纽约的旅途中，有幸亲身体验了一些备受赞誉的“人情味”。第一次遇到的是一个为航空公司工作的代理人。当我去办理登机手续时，她告诉我，因为我至少提前一个小时没有出现，从技术上讲，我已经错过了我的航班，必须买一张新票。举个例子，我仅仅过了安检线几分钟，我乘坐的机场通常有 0 到 5 分钟不等的安检线。这么说吧，如果整个机场是一个停车场，他们就不会费心去命名区域了。

我抗议，但他们声称他们的系统不会让他们打印我的机票，即使他们想。

对团队人情味来说不是个好的开始。

当机器仍然是发号施令的人时，人类代理人的意义何在？我向指挥系统提出了我的呼吁，但看起来真正的老板——他们的系统——在这个问题上寸步不让。我买了一张新机票，通过了安检，然后为了消磨时间，我看着他们登上了我原来的航班。

我在主要机场的连接彻底陷入混乱，我决定在候机楼的一家商店买一份价格过高的礼物来安慰自己。这家商店很大，所以他们有两个收银台。然而，这位显然一直渴望与人接触的店员离开了岗位，与商店另一边的其他店员交谈。我冒险穿过商店，把钱给了他们，感觉自己闯进了他们的私人客厅，要求得到服务。我不是那种大惊小怪的人，所以我等她讲完故事后才向我致谢。

下一个我的绝技是移民代理。这位特工的举止就像一个老派侦探，他知道你有罪，只是在等你犯错。幸运的是，在一些粗暴的质问之后，我们都如愿以偿，我离开了他的国家。

在 JFK 着陆后，我受到了自动化移民代理的欢迎。它没有审问我，只是扫描了我的护照，做了一些快速的计算，然后似乎用炭笔在我的移民机票上画了一张相当吸引人的照片。说真的，不管他们在那台机器里设置了什么过滤器，都必须在 Instagram 上。

当我走出航站楼时，我经过了捷蓝航空的自助值机，这是一家我喜欢的航空公司，它的系统从来没有因为我只早到了 55 分钟而对我做出判断。

在我最终到家之前，我决定从 CVS 购买一个价格更合理的东西，那里的自助结账机抑制住了与邻近机器聊天的冲动，而是急切地等待着为我服务。

值得注意的是，我对机器更愉快的体验并不是独一无二的。细致入微的企业在最近的一项研究中发现，67%的人更喜欢自助服务机。

此外，实施这些机器可以节省大量成本。航空公司通过人工代理办理登机手续的费用约为每次 3 美元；有了移动值机设备，这一成本降至 0.14 美元。据国际航空运输协会(International Air Transport Association)称，转向全自动值机流程可以为航空业每年节省 16 亿美元。这就提出了一个问题:为什么我们没有？

自动化系统不仅能为公司节省成本，实际上还能帮助他们赚更多的钱。麦当劳的一项实验发现，使用自助服务亭的顾客更有可能加大订单量，[导致每单平均增加 0.3 美元](http://www.cbc.ca/news/business/marketplace-are-you-being-served-1.3422736)。看起来从人类那里点餐的顾客更在意员工的判断，这让我感到震惊。在麦当劳吃饭的人还在乎别人怎么看他们？我一直认为这是不顾一切的餐馆选择。

如果客户更喜欢它，它省钱，甚至更赚钱，为什么它还没有普及呢？

我不愿意做第一个行动者——让下一个人来解决问题。然而，当它在一家公司运行良好时，为什么没有被同一垂直行业的所有竞争对手迅速采用呢？走过航空公司办理登机手续的柜台是对当前状况荒谬性的一种视觉表现。一些航空公司有值机机器，但旁边的航空公司明显没有。

很难从心理上理清这个圈子，直到我们看到 33%不喜欢自助服务机的人(上面 Nuance 研究的剩余部分)的人口统计数据。最能说明问题的指标是年龄。简而言之，这是社会上 33%的人最有可能打电话给他们的孙子，询问推特是什么意思。

对我们其他人来说不幸的是，这 33%通常也是大多数大公司董事会成员的投票多数。这更清楚地说明了某些公司有选择地不愿意在他们的工作流程中使用机器。坦率地说，如果我想购买的每一项技术都需要得到祖父的批准，我会用打字机写这篇文章，然后用旋转式电话打给我的编辑。

那么，问题应该是，当自助服务机在每一个可衡量的赚钱指标上都更好的时候，卢德派公司没有违反他们对股东的信托义务吗？

或许是时候让股东们开始要求更多的措施来确保公司跟上最新的技术趋势了。

股东应该能够审计公司的技术吗？在这种情况下，专业团队可以对实施自动化系统进行成本效益分析。是否应该为更年轻、更懂技术、能够帮助这些组织进入 21 世纪的成员设立强制性的董事会席位？毕竟，每年 16 亿美元似乎是一个相当大的疏忽，航空公司远不是唯一的违规者。像技术实现这样重要的事情不应该由一群天生厌恶技术的人随心所欲地去做。

我很乐意讨论那些暗地里渴望有一个友好的服务员为你加油的人。欢迎他们就人际接触点的重要性发表个人意见。然而，当这些信念优先于研究结果和公司利润——最重要的是，我的旅行经历——时，是时候采取更严厉的措施了。随着人工智能/机器人革命继续发展，进步不应该是一个见仁见智的问题。
