# 青少年最热门的聊天应用是谷歌文档——大西洋

> 原文:[https://www . theatlantic . com/technology/archive/2019/03/hotspot-chat-app-tenes-Google-docs/584857/？UTM _ source = Wanqu . co&UTM _ campaign = Wanqu+Daily&UTM _ medium = website](https://www.theatlantic.com/technology/archive/2019/03/hottest-chat-app-teens-google-docs/584857/?utm_source=wanqu.co&utm_campaign=Wanqu+Daily&utm_medium=website)



当斯凯勒学校的孩子们想在课堂上告诉朋友一些事情时，他们不会在一张小纸上草草写下一张纸条，然后扔到房间的另一头。他们使用谷歌文档。

15 岁的斯凯勒说:“我们真的不再传递物理笔记了。”和这个故事中的其他学生一样，斯凯勒用的是假名。

随着越来越多的笔记本电脑进入初中和高中，教育工作者[正在使用谷歌文档](https://www.nytimes.com/2017/05/13/technology/google-education-chromebooks-schools.html)进行协作练习，并帮助学生跟上课程计划。然而，学生们却用它来组织老师背后的对话。

青少年告诉我，只要他们需要放下手机，他们就会使用谷歌文档聊天，但他们知道他们的朋友会在电脑上。有时他们会使用该服务的实时聊天功能，默认情况下这种功能不会打开，很多老师甚至不知道这种功能的存在。或者他们会利用谷歌允许用户突出显示某些短语或单词，然后通过右边的弹出框对它们进行评论的事实:他们会克隆一个老师共享的谷歌文档，然后在评论中聊天，所以在不经意的观众看来，他们只是在课程计划上做笔记。如果教师走近仔细查看，他们可以单击解决按钮，整个线程将消失。

如果这个项目不是一个合作项目，孩子们将会创建一个共享文档，在那里他们将会一行一行地聊天，看起来像一段文字。“人们会做一个新的页面，用不同的字体交谈，这样你就能知道谁是谁，”斯凯勒说。“我有一个非常好的朋友，我们在不同的教室。所以我们会给对方发电子邮件，聊一些正在发生的事情。”在课程结束时，他们会删除一个文档或解决所有的评论。很少有人会像上一代人保存朋友的纸质笔记那样保存它们。

通过谷歌文档聊天不仅仅是愚弄老师；它也欺骗父母。当大家晚上登录做作业的时候，Google Docs 的聊天就活了起来。几组孩子将在一份文件上合作，而他们的父母认为他们在做一个学校项目。正如 Reddit 上的一个帖子[在二月份透露的那样，](https://www.reddit.com/r/teenagers/comments/ag2cx5/they_blocked_google_hangouts_at_my_sister/)通过 Google Docs 聊天也是规避父母社交媒体禁令的一个好方法。

就像过去的纸质笔记一样，大多数谷歌文档的聊天内容都很乏味。孩子们用它来谈论一天在学校发生的事情，计划舞会，闲聊，调情，互相欺负。在[的一篇博文](http://www.bark.us/blog/kids-using-google-docs/)中，Bark，一款父母可以用来监控孩子手机使用的应用，警告父母不要让孩子在谷歌文档中联合起来攻击其他孩子:“他们合作在一个共享的谷歌文档中写下刻薄或伤害性的东西。在其他情况下，孩子们创建私人的数字“燃烧书”，并邀请其他人贡献，而不包括被戏弄的孩子，”帖子写道。但是青少年说这种类型的使用很少。“人们只是谈论老师，或者说，谈论他们的日子。这将是最无聊的事情，但这是唯一能让彼此传递信息的方法，”斯凯勒说。

教室里不使用谷歌文档的孩子会求助于他们必须使用的任何协作学习软件来交流。例如，微软 Word 的在线版本具有与谷歌文档相似的功能，可以以同样的方式加以利用。16 岁的内森(Nathan)来自费城郊区，他告诉我，他和他的朋友们“发现有一个‘协作空间’，你可以在那里上传文档，与全班同学分享，”他在学校最喜欢的笔记软件 OneNote 中使用。"所以我们只是用荧光笔工具画些照片，然后上传到共享文件夹中."内森说，他和他的同学去年发现了群发信息功能，非常兴奋，因为“看起来你很有效率。”但是，他补充说，“缺点是你没有在做你应该做的事情，所以你什么也做不了。”

虽然谷歌文档聊天可能是中学生最热门的交流工具，但大多数青少年一到大学就放弃了它。20 岁的斯凯来自波士顿郊外，她说想到谷歌文档聊天就让她怀旧。“在谷歌文档上聊天很容易让人想起我们年轻的时候，”她说。还有纸质笔记？“从五年级开始，我就没给任何人发过短信，”她说。
