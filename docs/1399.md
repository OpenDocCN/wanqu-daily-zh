# 跟随你的激情，跟随你的直觉，是吗？

这就是你们看到的——四家令人敬畏的初创公司在上市当天被套牢了。虽然我们可以很容易地回顾和嘲笑这些评论，但还有数百家其他创业公司没有这么容易。

这里肯定有一个教训。你需要提醒自己的是——这里最重要的是——只要你追随你的激情，相信你的直觉，努力工作，最终可能会证明手段是正确的。忽略那些讨厌你的人。你愿意吗？跟随你的心，跟随你的激情。关注你的产品和你的品牌倡导者。
