# “以后再看”应用程序的秘密力量

*由蒂亚戈的* [的*福特实验室的*的](http://fortelabs.co)

2014 年底，我收到了一封电子邮件，通知我在这一年中，我已经在“稍后阅读”应用程序口袋中阅读了超过 100 万个单词。



这个数字本身并不令人印象深刻，考虑到我们每天摄入的信息相当于 34 千兆字节，10 万字，或者 T2 174 份报纸，这取决于你问谁。

在我看来，这个数字的意义在于它代表了 22 本书的阅读量，如果没有一个合适的系统，这是不可能发生的。

我们已经养成了一个习惯，用浏览 Twitter、Instagram 和脸书的方式来填充我们一天中的上百个随机空间。但是那些目光已经慢慢变成了凝视，而且这些凝视已经占据了我们醒着的大部分时间。

最终结果是，每年花 127 个小时在 Instagram 上的人(全球平均水平)抱怨她“没有时间”阅读。

事实是，阅读能力正在成为世界竞争优势的来源。

我说的不是基本素养。变得极其稀缺的(因此也是宝贵的)是身体、情感、注意力和精神上的能力，能够安静地坐着并持续一段时间集中注意力。

《哈佛商业评论》最近的一篇文章给这种新的神经现象起了一个名字:注意力缺陷特征。基本上，术语 ADD 和 ADHD 正在退出使用，因为实际上整个人群都符合诊断标准。这不再是一种状况，而是一种*特质*——现代生活固有的、不可避免的体验，其特征是“注意力分散、内心狂热和急躁”

# 一个系统中最高的杠杆点是在入口——最初的假设和范例通知它的发展

我曾经写过如何使用 Evernote 作为一个通用的参考文件系统，不仅是为了保持条理，也是为了激发创造力。

但是在创建任何工作流时，我都没有解决一个关键问题:信息如何进入系统，以及从哪里进入系统？工作流的输出质量从根本上受到其输入质量的限制。垃圾进，垃圾出。

我们可以讨论很多方法来提高你所消费的信息的质量。但我现在想把重点放在两个稍后阅读应用程序可以帮助的方面:

1.  越来越多地消费长篇内容(可能更具实质性)
2.  更好的过滤

# #1 |增加长格式内容的消费

为了消化好的想法，首先你必须消化许多想法。

这是 Tim Ferriss 和其他人提出的“信息饮食”建议中的根本缺陷:强过滤器对较大的初始流量最有效。把你的朋友作为你新想法的主要过滤器，确保你仍然是房间里最笨的人，对谈话没有任何贡献。

问题是，我们的整个数字世界都倾向于可抓取的低级信息——照片、推文、状态、快照、订阅、卡片等等。为了对抗潮流，你必须重新设计你的环境——你必须创造启示。

> 启示:一个物体和一个有机体之间的关系，通过一系列的刺激，给这个有机体提供了一个执行动作的机会。

让我们看看阅读长格式内容的 4 个主要障碍，以及阅读长格式内容的应用程序用来克服它们的启示:

## 3.异步读取

在我们支离破碎的时代背景下，这是最难理解的阅读障碍之一。

反复开始做一件事却没有完成，这是非常非常令人不满意的。这就是我们整天在工作中所经历的，不断被一连串的“紧急事件”打断在紧张的一天中，我们最不希望看到的就是连读一篇文章都失败。

*[的](http://users.wfu.edu/masicaej/MasicampoBaumeister2011JPSP.pdf) [2015 年修订版](http://www.amazon.com/Getting-Things-Done-Stress-Free-Productivity-ebook/dp/B00KWG9M2E)引用了罗伊·鲍梅斯特博士的作品，他表明“未完成的任务占据了大脑的空间，从而限制了清晰度和专注度。”无法完成一篇长文章(更不用说一本书)的认知失调风险让我们甚至无法开始写它。*

*稍后阅读应用程序解决这一问题的方法是，简单地将你的进度保存在给定的文章中，允许你在不同的时间或不同的设备上重新开始，并在你完成后明确地将项目标记为“已读”。*

# ***进步陷阱和范式***

***世界上的信息量是一个[进步陷阱](https://en.wikipedia.org/wiki/Progress_trap)。要读的东西太多和太少一样有局限性。***

***正如无与伦比的文卡特什·拉奥[所写的](http://breakingsmart.com/season-1/)，我们正从一个**容器**(公司、部门、学期、包裹、筒仓)的世界，转向一个**流**(社交网络、信息反馈、繁荣城市的主要街道、推特)的世界。问题和机遇都不愿意被整齐划一的小盒子包围。有太多东西需要消化。太多了，甚至无法猜测你所不知道的。***

***随着世界变化的步伐加快，我们加倍重视所有首先造成问题的方法——更多的规划、更多的预测、更多的控制和风险管理。我们剩下的是无人信任的庞大机构，它们既脆弱又大到不能倒，在社会经济金字塔的每一层都造成了不稳定性。***

***相反，以一种我们进行得越快越好的方式来解决问题(和探索机会)会是什么样子？***

***我不能公正地对待上面链接的饶的博客系列(它分为 20 个部分——可能想留到以后；)，但他建议的第一步是“让自己尽可能多地接触不同的信息流。”***

***当你沉浸在溪流中时，水流越快，你接触到的新奇观点和想法就越多。你发展了一个[对立的头脑](http://rogerlmartin.com/lets-read/the-opposable-mind)——有能力在任何问题上用不同的观点来玩杂耍和游戏，而不是通过一个镜头来看问题。***

***越来越多的是，在你的个人成长旅程中唯一重要的指标将是 **ROL:学习率**。近年来，我们听到很多关于实践学习和实践实验的重要性。我们明白了。埋头于一本书本身不会让你有所收获。***

***但是钟摆在那个方向摆得太远了。是的，你可以*太*行动导向。与有效的执行相比，想法虽然便宜，但仍然比我们花时间做的许多其他事情更有价值。***

***还有一种方法可以学得更快:吸收并借鉴他人的想法。当然，你不会理解他们的经历给他们的每一个心照不宣的教训，但你可以吸收其中的许多教训，在很短的时间内，你自己就会犯下每一个错误。***

***想法是高杠杆代理。当排列在高度交叉引用的网络中时，它们变得更加明显。我们拥有的唯一能够按需创建和访问这些网络的工具是人脑。***

***我之前撒谎了。有一种形式的杠杆作用甚至比最初的假设和范例更强大，它通知了一个系统的开发:超越范例的能力。***

***在 Donella Meadows 关于复杂系统的开创性文章中，我说得再好不过了:***

> ***坚持范例的人(这意味着我们所有人)会看到一个巨大的可能性，即他们认为的一切肯定都是无意义的，并迅速向相反的方向发展。当然，在任何世界观都没有确定性的观念或经验中，没有力量，没有控制，没有理解，甚至没有存在的理由，更不用说行动了。但是，事实上，每一个设法接受这个想法的人，无论是一时还是一生，都发现它是激进赋权的基础。如果没有合适的范式，你可以选择任何有助于实现你目标的范式。***
> 
> ***正是在这个掌握范式的空间里，人们摆脱了毒瘾，生活在持续的欢乐中，推翻了帝国，被关起来或被烧死在火刑柱上或被钉死在十字架上或被枪杀，并且**产生了持续数千年的影响**。***
> 
> ***最后，掌握似乎与推动杠杆点关系不大，而与战略性地、深刻地、疯狂地放手关系更大。***

***阅读是我们能想到的最接近他人想法的东西。这很长，有时很沉闷，但这项工作需要将你自己包裹在另一个人的范式中。这是疯狂放下自己的第一步。***

***想法的神奇之处在于，它不需要任何时间就能改变你的范式。它及时发生，但不需要时间，就像一个内部维度的虫洞，你大脑中的一个纠缠粒子在一个比宇宙更大的鸿沟上镜像它的孪生粒子——两个心灵之间的鸿沟。***
