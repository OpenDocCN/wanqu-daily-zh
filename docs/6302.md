# 消费者创业在社会距离世界中的机会和风险——消费者注意力框架。

社交距离极大地改变了我们打发时间的方式。随着所有面对面的聚会被取消，人们突然有了更多的空闲时间。新的和现有的消费互联网公司都在竞相争取这个时间。

作为消费品的投资者和爱好者，我对这个窗口所代表的机会感到无比兴奋。但我也问过自己，是什么让一家消费者创业公司不仅在社交距离遥远的时期蓬勃发展，而且在之后依然繁荣？这是我的想法，也是给消费者创业者的一些建议。

# 介绍岩石、沙子和水的框架以引起注意

我们总是说消费者的注意力是有限的，但它是如何划分的呢？

我将消费者在一天中的注意力划分为三种不同类型的“事件”或模式:跨越更大的连续时间块的事件(“岩石”)、利用这些时间块之间或期间的注意力间隙的微事件(“沙子”)以及可以覆盖其他两个事件的事物(“水”)。

每天，我们用这三者的某种组合来填满我们的 24 小时(+罕见的气泡)，不断地在我们的选择之间分配时间，以最大化我们的多巴胺/血清素等，同时履行我们对工作、家庭和健康的责任。

岩石吃角子老虎是最难安排的，因此也是最难赚的。真的要物有所值。一些岩石(如电视)更容易适应，因为它们是多孔岩石，使多任务处理更容易(如在节目的慢速部分浏览 Twitter)。其他的(比如 VR)更像是钻石，不给其他东西留空间，使得取舍更加困难。

找到一天中的沙子时间要容易得多——30 秒的抖音比 30 分钟的堡垒之夜更容易找到。

文字、照片、视频和游戏生活在石头和沙子的世界里，而音频是我认为的水。水不同于岩石和沙子。水不需要特别注意，它可以渗透到一些岩石和所有的沙子中。*例如,“不和谐”是游戏石头的绝佳水源。

现在，水是音频(音乐是迄今为止最流行的)**，尽管你可以想象未来 ar 是水。

不是每个使用产品的人都需要处于相同的模式。例如，谈话广播对听众来说可能是水，但对说话者来说需要像石头一样的专注度。

你可以用这个框架来集思广益消费者的想法。例如:

尝试进行一项“摇滚”活动，并思考如何让它成为一项“沙滩”活动。如书籍→博客→微博、电影/电视→短视频等。

或者尝试做一个“岩石”活动，仔细想想“水”在上面会是什么样子。如健身→ Aaptiv。博弈→不和。等等。

# 社交距离粉碎了线下的岩石，为消费者创业公司创造了机会之窗。

如果一个普通消费者的杯子曾经装满了石头、沙子和水的混合物，那这种混合物已经改变了。社交距离粉碎了线下的岩石，消费电子产品正在填补这一空白。你和朋友一起参加网飞派对，而不是去看电影。你[家庭聚会](http://www.houseparty.com)或 Facetime 或……上帝帮助你……[Zoom](http://zoom.us)与朋友在你安全的家中聚会，而不是去餐馆或朋友家。大多数(如果不是全部的话)现有社交应用的参与度都达到了创纪录的水平。

换句话说，摇滚老虎机(包括所有摇滚老虎机中最难的:现场，同步摇滚老虎机)比以往任何时候都更容易赚。

但如果你听到消费投资者担心这些早期消费公司的势头可能是虚假的，那是因为缺乏线下岩石是一种暂时的状态。然后呢？

让我们都希望在现实生活中，人们可以尽早地与其他人共度时光，而不会害怕或犹豫。当这种情况发生时，赢得 15 分钟以上的会议将变得更具竞争力，特别是对于那些数字化翻译亲身体验的应用程序。你如何确保一旦我们在这个社会距离炼狱的另一边，你会坚持下去？

# 抓住这个时刻，但是寻找机会优雅地退化成一次沙滩或水上的经历。

如果你是一名消费者创始人，正在思考当“原地避难所”结束时，该如何定位自己，我有两条建议供你思考:

首先，利用这个机会为你的成功做好准备，为你的网络效应创造必要的动力和规模。如果你的良性循环([我的参与度等级第三级](https://medium.com/@sarahtavel/the-hierarchy-of-engagement-expanded-648329d60804))在运转，你将最有可能赢得未来与线下竞争对手的摇滚烘焙比赛。换句话说，津津有味地抓住这个窗口！

对我来说，这可能意味着一件亵渎神明的事情，我以后可能会后悔说:如果你相信你有产品/市场契合度(通过每周队列保留率接近 30%以上的范围来证明)，我会考虑在收购上花一点钱来加速增长是否会帮助你利用这个窗口。不要误解我——它不可能成为增长的主要手段。这肯定是一个暂时的策略，你会看到有机增长继续成为你的主要增长载体，但在这个世界上，这是一个利用时间的时刻，我很乐意打破神圣的规则，如果它是在诚实和自律的情况下完成的。

第二，如果你开始的时候仅仅是一次“摇滚”体验，那就尽情享受吧，但最终这些摇滚机会将会越来越难获得。

当不可避免的事情发生时，为了让自己处于成功的位置，想办法优雅地降级到“沙”或“水”的体验。这将让用户与你互动，即使他们不能承诺“摇滚”时间。

我们离担心这个还有一段路要走，但是你会想要保持警惕。例如，Instagram Live 更像是一种“岩石”体验，但 Instagram 的其余部分是沙子(通常是砂岩！).

—

尽管社交距离限制了我们在现实世界中共度时光的能力，但它并没有消除我们作为人类相互联系、感受归属感以及在[中寻找他们可以参与的论坛](https://medium.com/@sarahtavel/the-era-of-participatory-social-2e43741f8152)的需求。

这段时间会出现什么岩石或者沙子？什么水域？我们可能会看到一种新的水域出现，这种水域具有跨越某些类型岩石的特性。

如果你正在建立的东西，你认为利用了这个窗口，在社交距离之后，不仅会生存，而且会茁壮成长，不要害羞。目的地管理系统是开放的。

*有趣的是，正如我的朋友[马来·甘地](https://twitter.com/malayhgandhi?lang=en)指出的，水是独一无二的，因为它是正和的(而不是石头和沙子之间吸引注意力的零和游戏)。

* *恕我把这个比喻说得太长，有时水(如有声书籍和播客)更像一种粘性液体——它需要一种非常多孔的岩石，如通勤或锻炼，以便获得时间，否则它更像一块岩石本身，必须与更坚固的岩石和沙子争夺注意力。
