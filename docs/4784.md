# 思考就是工作

创业初期，你除了时间什么都没有。当你有大量时间时，你会做很多看起来不像工作的事情。例如，这是我最近刚找到的一段视频，讲述了我们开始工作一年后，除了时间什么都没有的日子。



我把一个傻瓜相机贴在一根高尔夫球杆上，并在自己面前旋转。**我在尝试一种新的视频拍摄方式，而** **偶然发明了自拍杆。当你除了时间什么都没有的时候，就会发生这种事。

唉，我们从来没想过要把自拍杆商业化！但是，有了这么多空闲时间，我们*确实*想出了如何制造产品、寻找客户、推销自己、建立文化，以及做所有其他你需要做的事情来创建一个企业。因此，不要让 2007 年日历上的所有空白欺骗了你。和 2016 年的日历一样，这一天也很充实，充满了**思考公司的非结构化时间。很难将开放式思维视为工作，因为很多开放式思维不会带来具体的变化和进步。然而，最重要和最有影响力的想法来自开放式思维。****

# 提高工作效率以获得更多时间

面临的挑战是如何创造足够的时间来思考，即使这些时间可能最终会让你觉得毫无收获。最初，我的策略是使用能帮助我更有效利用时间的策略和工具。这意味着在我的日历上放上不开会的时间，生活在零收件箱中，并结合使用每周日记、任务列表自动化和虚拟助手，这样我就可以掌握重要的事情，同时希望有时间思考。

虽然这个*帮助我保持高效，但思考的时间总是第一时间离开。这意味着思考的时间以其他方式渗透到我的生活中:我会在晚上醒来思考生意，改变与妻子亚历山德拉的谈话——通常，我在家的时间越来越少。我不得不接受这样一个事实，我的生活会有点不平衡，我总是需要工作来找出如何更有效率，有时间思考，并为家人和朋友节省时间。但是事实证明,*是一种创造思考时间和保持高效的方法。我在和我的朋友本的一次晚餐中学会了这一重要的课程。**

# *生产力是人的问题*

*去年夏天，我和 Mailchimp 的首席执行官 Ben Chestnut 共进晚餐。如果你不知道 MailChimp，他们非常受欢迎，非常成功，有一个鼓舞人心的品牌，通常已经建立了一个[神奇的业务](https://www.nytimes.com/2016/10/06/technology/mailchimp-and-the-un-silicon-valley-way-to-make-it-as-a-start-up.html?_r=0)。本和我正在聊天，分享故事，这意味着我向他提出了无数个问题。最后，我问本，他如何保持足够的工作效率，以便有时间思考。他回头看着我，脸上带着一丝困惑。

他说，“人们总是问我如何变得更有效率，**但这从来不是效率的问题:这是人的问题。”本看着我的眼睛告诉我，要么是我授权不够，要么是我身边没有合适的人。

我傻眼了。我从未想过生产力是一个人的问题。***

*我一直认为我很擅长授权。这些年来，我寻求了很多帮助。我为完全没有我的干预而发生的产品、事件和内容感到自豪。我从来没有想过通过授权我可以有更多的时间去思考，我认为我目前参与的事情是我应该参与的事情。

那次谈话之后，我开始以不同的方式思考我的生产力。我真的需要参加会议吗？还是我真的*没有给人们足够的所有权，让他们慢了下来*？我身边有没有可以在没有我的情况下管理事情的人？我是否因为参与过多而拖累了公司？我接受了本的建议，开始委派更多的任务。不仅仅是委派解决方案，而是真正委派需要解决的问题。如果我需要参加会议，那是因为我没有很好地委派，或者我没有合适的人。把生产力看作一个人的问题，有趣的是，如果你有合适的人，你可以比你想象的更快地改善事情。接受本的建议三周后，我的时间表从满满当当变成了 50%的空闲时间。突然，我发现自己有很多时间去思考。*

# *不要感到内疚*

*如果你发现自己有时间思考，很容易感到内疚。我们有一种敬畏喧嚣的科技文化。疯狂的工作时间和付出是常态。挑战在于，在扩展时，思考时间变得更加重要，也更加难以获得。

不要重蹈我的覆辙。

问问自己:你有足够的时间发明自拍杆吗？或者更重要的是，你有足够的时间意识到你做到了吗？如果不是，无论你在旅程的哪个阶段，你可能都需要更多的时间去思考。*
