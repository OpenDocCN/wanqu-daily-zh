# 知道什么时候该继续前进

## 有时候离开你成功的创业公司对你来说是最好的事情

下周，我将离开我在世界上最昂贵的城市之一的 PlayHaven 的全职工作——这是我的选择。我的薪水很高，公司在发展，我和一个了不起的朋友团队一起工作。我还是会问自己，“我他妈的在干嘛？”

下面是我的故事…

T2】

大约三年前，我搬到旧金山，加入了 PlayHaven，成为#10。硅谷一直吸引着这位土生土长的俄勒冈人，有机会在蓬勃发展的移动游戏领域加入有远见的杨安迪，让我毫不犹豫地做出了决定。但老实说，我不知道我在做什么。很快我发现，团队也没有。

我们的产品很有吸引力，但收入和采用率几乎停滞不前。我们需要改变一些事情。头五个月是我一生中最激动人心也最害怕的时光，因为我们的现金储备越来越少，我们在争论下一步该做什么。裁员在即。

裁员前不久，我和安迪喝着啤酒聊了聊。他告诉我接下来会发生什么，并说我可能会在几周内失业。我做了什么？我从舒适的家乡搬到这个陌生的城市，加入一家濒临倒闭的初创公司？“我想我会挺过去，然后搬回家，”我想。

但我活了下来。几周后，公司缩小了规模，当我的朋友和同事收拾行李时，我悲伤地和他们说再见。还剩六个人。

T2】

快进三个月，情况开始好转。我们开始了一个新的，鼓舞人心的愿景。团队很兴奋。

我们埋头苦干，几乎废弃了所有现有的代码来重建产品。作为唯一的产品经理，我和一个由五名工程师组成的小团队一起工作。这在很大程度上是一种协作努力。一定是。在回答支持请求、偶尔发送营销电子邮件和其他各种义务之间，我负责定义用户体验，为新的仪表板创建线框。我做过类似的工作，但没有这么大的规模。我必须在工作中快速学习。我们取得了巨大的进步，这要归功于一个出色的、充满渴望的工程团队，但随着时间的流逝，收入几乎为零。我们设定的目标是在 2011 年 7 月的休闲连接上进行私人发布。

前一个月是忙碌和激动人心的。经历了痛苦难熬的漫漫长夜，但激情是一种能让你度过难关的强效药。在资源匮乏的情况下，我(一个非技术人员)匆忙将产品投入使用，为我们正在构建的仪表板设置了一个本地环境，并开始消除 bug。每次 GitHub 提交都会带来多巴胺的激增和对进展的满足感。

那是凌晨 2 点，一个大客户的重要演示的前一天晚上。在没有 QA 团队(甚至没有一个 QA 人员)的情况下，我对产品钻研到深夜，发现并有时修复更多的 bug。这是非常不科学的，但我们没有时间进行“过程”。我们只需要把事情做完。最终，产品以可用的状态交付，我们得到了第一个大客户。这是件大事，我们庆祝了我们的第一次重大胜利。

T2】

从我们卑微的开始，团队已经发展到 85 人。我们已经获得了几千名客户，创造了数百万美元的收入。我们在一个不断增长的市场中运营，有着惊人的愿景(直到今天我都坚信这一点)。事情进展顺利。

那么为什么要留下一个好东西呢？

正如我在之前[分享的，我的首要任务是学习。在过去的三年里，我已经长了*吨*，尤其是在](http://ryanhoover.me/post/49179929021/be-selfish)[不适](http://ryanhoover.me/post/42414716750/fear-is-healthy)的时候，那是我受到挑战最大的时候。

我在 PlayHaven 的时光教会了我很多。发现人们真正想要什么的方法。构建*右*(最小)功能的重要性。将想法推向市场的过程。如何扩大公司规模？雇佣合适的人和建立一致的文化的重要性。还有更多。

但是我已经准备好迎接下一个挑战，渴望加速我的学习。我的朋友内森·巴肖最近告诉我:

> 当你回顾六个月后的自己，并且不为自己的天真感到尴尬时，就有问题了。这意味着你没有学习，没有成长。

随着公司的发展，我的角色已经转移到更具体的关注领域。我们还聘请了一位 UX 设计师，来减轻我过去处理的许多责任和(有趣的)挑战。简而言之，我的角色从我的多面手出身变得更加专业化。

但老实说，我已经对我们所处的市场和行业失去了热情。我试图弥补这一点，三个月前休了一个(早该休的)两周的假，但激情并没有回来。这些年来，我对电子游戏的兴趣越来越小。我从小就痴迷于在星际争霸中微操作陆战队员，在激烈的团队堡垒经典比赛中断开连接后说脏话，并在光晕上大声咕哝到凌晨。今天，我很少在我的 iPhone 上玩哪怕是休闲的两分钟分心游戏。相反，我投资于自己，阅读、写作、兼职做项目/实验。

今天，我有很大的野心去制造一个我能消费的产品。如果你关注我的文章，你可能会注意到我从来不写我工作的行业:游戏。这很能说明问题。

此外，我不是游戏开发者，也没有动力成为一名游戏开发者。为自己开发产品的人拥有巨大的优势，因为他们是自己的用户。他们不需要“进入用户的大脑”来理解问题。他们坚持不懈的能力因解决自身问题的动力而增强。

T2】

尽管有这些感觉，这个决定远非告诉我的朋友和老板那么简单和困难。我几乎说不出话来，“我要走了。”我哭了。这仍然很难接受，但在内心深处，我知道这对我是正确的。

我感谢给我的机会，感谢朋友的支持，感谢企业家群体的指导(你知道你是谁)。我要感谢很多人。

T2】

当我在 PlayHaven 度过最后几天并思考我的下一次冒险时，我在评估新机会时会提到这些问题:

*   这真的是我想做的吗？
*   我对产品、市场和愿景有(强烈的)热情吗？
*   我会接受挑战、学习并充分发挥潜力吗？

我希望其他人可以从我的经历中学习，并感到鼓舞，找到他们的激情，反思他们的个人目标，并给自己一个机会。
