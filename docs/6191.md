# 缺齿简历——我简历中缺失的工作

> 原文:[https://hunter walk . com/2019/06/24/gap-齿牙-resumes-the-missing-job-from-my-cv/？UTM _ source = Wanqu . co&UTM _ campaign = Wanqu+Daily&UTM _ medium = website](https://hunterwalk.com/2019/06/24/gap-toothed-resumes-the-missing-job-from-my-cv/?utm_source=wanqu.co&utm_campaign=Wanqu+Daily&utm_medium=website)



我的简历上有一个小洞。现在，20 多年后，你需要眯着眼睛才能看到它，但它仍然在那里。我接受这份工作是因为我在读研时需要钱，而且，老实说，因为也许创业公司可以在它和 Web 1.0 泡沫破裂之前上市。必要性和贪婪的结合压倒了我更好的判断。

这是一个 B2B 企业软件商店，看起来更像是一个试图将服务收入转化为可扩展产品的开发团队。员工基础是一个维恩图，包括(a)知道这个地方有些不对劲的人——主要是领导层，但希望参与解决问题,( b)骗子,( c)职业生涯刚开始时认真的专业人士，他们可能认为这就是商业。

当你年轻的时候，你把简历上的每一份工作都抓得紧紧的——列出你开始工作的月份和结束工作的月份。“没有缝隙！”我记得职业中心告诉我。仿佛一个无缝的路径是一个光荣的白色微笑的牙科等价物，几个尴尬的职业转换成了一个拳击手的手套，打在你的下巴上，在你的手上留下血淋淋的牙齿，在你的脸上留下一个坑坑洼洼的笑容。所以我在这份工作上多待了一会儿。负净值遇到负自我价值。

我不记得我第一次把公司从我的简历中删除是什么时候。我走开后过了一段时间了吗？或者我只是对几年后的人生轨迹有足够的信心，以至于没有人问我从毕业到开始在林登实验室工作之间有几个月的时间？让我想起这条[推文](https://twitter.com/LeonHWolf/status/717111637317173249?s=20):
