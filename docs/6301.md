# 为什么你的第一个客户价值 1000 万美元——埃里克·乔根森

> 原文：<https://www.ejorgenson.com/blog/why-your-first-customer-is-worth-10000000?utm_source=wanqu.co&utm_campaign=Wanqu+Daily&utm_medium=website>

创始人和早期员工可以比 50+的员工多赚几个数量级。这是有充分理由的。早期员工不是旅行者，他们是制图员。

“做不可扩展的事情”是保罗·格拉厄姆给创始人和早期员工的建议...

> 创始人在开始时最常见的不可扩展的事情就是手动招募用户。几乎所有的创业公司都必须如此。你不能等着用户来找你。你必须走出去得到它们。”

虽然我一直*相信*这一点，但我并不真正*理解*。

毕竟第一个客户是最不赚钱的！他们比任何一个客户付出了更多的努力，更多的关注，更多的支持和更多的努力！数百小时的创始人和早期员工的工作进入建设，赚取和保持第一批客户。这些客户在单位经济(CAC / LTV)和会计基础上都是无利可图的...甚至很可能是士气的基础。

有这样一个时刻，当你有几个早期客户时，你会因为得到这几个客户而感到遍体鳞伤、疲惫不堪...你想知道这是否会变得更容易。

如果你花了太多时间想象所有的客户都会像第一个客户一样具有挑战性、缓慢且昂贵——“好吧，我们还不如现在就关掉公司。”

然而。

让我*理解*“做不可扩展的事情”的数学方法是当我不再看单位经济学，而开始看企业价值时——像投资者一样思考。

> “我是一个更好的投资者，因为我是一个商人，我也是一个更好的商人，因为我是一个投资者。”——沃伦·巴菲特

杰森·卡拉卡尼斯在谢恩·帕里什的播客中说，“当一家公司从 0 个客户变成 1 个客户时，它就从不可投资变成了可投资。”

就在那时，我突然想到了“做不可扩展的事情”。

当一家公司从 0 个客户变成 1 个客户时,**预计的企业价值**变化是...接近无限增长%。如果客户数为 0，公司价值为 0 美元。

当你没有任何客户的时候，给你的业务增加最大价值的事情就是增加你的第一个客户。

当一个年轻的公司增加一个客户时，他们不仅仅是将一个客户的 LTV 增加到他们的企业价值中。他们也增加了赢得其他有类似问题的客户的可能性。他们获得了一些增加的可信度，一些增加的规模，以及可以继续的经验教训。

**想一想早期销售对预计企业价值的影响**，很明显赢得并保持客户第一是公司最重要的工作。你的第一个客户可能对你的企业价值有数百万美元**。它可能会让你从不可投资变成可投资。**
