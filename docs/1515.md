# 创业清单

如果你想使用[清单宣言](http://atulgawande.com/book/the-checklist-manifesto/)的方法，这里有我推荐给任何企业家的 3 个清单。我可以花时间想出我自己的清单，但是没有必要重新发明轮子。即使在评估和与许多初创公司合作之后，我仍然经常使用这些工具。就像所有的清单和[简单规则](http://www.amazon.com/Simple-Rules-Thrive-Complex-World/dp/0544409906)一样，写下来很容易，但执行起来很难。

# 如何从彼得·泰尔开始:

点击此处查看他的 7 个你应该问的问题:

1.  工程问题:你能创造突破性的技术而不是渐进的改进吗？
2.  时机问题:现在是创业的好时机吗？
3.  垄断问题:你是从一个小市场的大份额开始吗？
4.  人们会问:你有合适的团队吗？
5.  分销问题:你有办法不仅创造产品，还能交付产品吗？
6.  持久性问题:你的市场地位在未来 10 年或 20 年内还能保持吗？
7.  秘密问题:你是否发现了别人看不到的独特机会？
