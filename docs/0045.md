# 啊哈！让保罗·格拉厄姆的 Y 组合子成为可能的时刻

1995 年夏天，保罗·格拉厄姆在收音机里听到一个宣传网上商务无限可能性的故事，当时网上商务几乎不存在。促销活动来自网景公司，该公司试图在首次公开募股前夕吸引人们对其业务的兴趣。这个故事听起来很有希望，但又很模糊。当时，格雷厄姆正处于十字路口。从哈佛大学获得计算机科学博士学位后，他陷入了一种模式:他会在软件行业找到一些兼职咨询工作；然后，攒够了钱，他会辞掉工作，把时间投入到他真正热爱的事情上——艺术和绘画——直到钱花光，然后他再去抢另一份工作。现在 30 岁的他厌倦了这种模式，他讨厌咨询。通过为互联网开发东西来快速赚大钱的前景突然变得非常诱人。

他打电话给他在哈佛的老编程伙伴罗伯特·莫里斯，让他对合作创业的想法感兴趣，尽管格雷厄姆不知道他们将从哪里开始，也不知道他们将开发什么；最终，他们决定尝试编写一个软件，让一个企业能够创建一个网上商店。一旦他们清楚了这个概念，他们不得不面对一个巨大的障碍。在那个时候，一个程序要足够流行，它必须是为 Windows 编写的。作为完美的黑客，他们厌恶 Windows 的一切，也从未费心去学习如何为它开发应用程序。他们更喜欢用 Lisp 编写程序，并在开源操作系统 Unix 上运行。

他们决定推迟不可避免的事情，无论如何还是为 Unix 编写了程序。稍后将其转换为 Windows 将很容易，但当他们考虑这样做时，他们意识到这将导致可怕的后果——一旦程序在 Windows 中启动，他们将不得不与用户打交道，并根据他们的反馈完善程序。这意味着他们将被迫在 Windows 中思考和编程几个月，甚至几年。这种前景太可怕了，他们认真考虑过放弃。

一天早上，格雷厄姆醒来时想到，他们或许可以通过点击链接来控制服务器上的软件。当他意识到这些词的含义时，他突然坐直了——创建一个程序来建立一个在线商店的可能性，这个在线商店将在网络服务器上运行。人们可以通过 Netscape 下载和使用它，点击网页上的各种链接来设置它。这意味着他和莫里斯将绕过通常的途径，编写一个用户可以下载到他们桌面上的程序。这将消除你涉足 Windows 的需求。没有任何东西像这样，但它似乎是如此显而易见的解决方案。他兴奋地向莫里斯解释了他的顿悟，他们同意试一试。几天之内，他们就完成了第一个版本，而且功能非常好。显然，web 应用程序的概念是可行的。

在接下来的几个星期里，他们改进了软件，并找到了自己的天使投资人，他提供了 10%的业务股份，初始投资为 10，000 美元。一开始，很难让商人对这个概念感兴趣。他们的应用服务器提供商是第一个在互联网上运行的创业程序，处于在线商务的最前沿。然而，慢慢地，它开始起飞了。

事实证明，格雷厄姆和莫里斯之所以想到这个新奇的主意，很大程度上是因为他们讨厌窗户，这个主意具有各种意想不到的优势。他们直接在互联网上工作，可以产生一系列连续的软件新版本，并立即进行测试。他们可以直接与消费者互动，获得对他们程序的即时反馈，并在几天内改进程序，而不是像桌面软件那样需要几个月。由于没有经营企业的经验，他们不会考虑雇佣销售人员来做推销工作；相反，他们自己给潜在客户打电话。但由于他们是事实上的销售人员，他们也是第一个听到消费者抱怨或建议的人，这让他们对该计划的弱点以及如何改进有了真实的感受。因为它是如此独特，而且是从左场出来的，所以他们没有需要担心的竞争对手；没有人能窃取这个想法，因为他们是唯一足够疯狂去尝试的人。

很自然，他们在这个过程中犯了一些错误，但是这个想法太强大了，以至于不能失败(尽管格雷厄姆注意到公司几次差点就失败了)。1998 年，他们将名为 Viaweb 的公司卖给了雅虎！五千万美元。

随着岁月的流逝，格雷厄姆回顾这段经历，他对自己和莫里斯所经历的过程感到震惊。这使他想起了历史上许多其他的发明，如微型计算机。使微型计算机成为可能的微处理器最初是为控制交通灯和自动售货机而开发的。它们从来就不是用来驱动计算机的。第一批尝试这样做的企业家遭到了嘲笑；他们创造的计算机看起来名不副实——它们是如此之小，能做的事情如此之少。但是他们抓住了足够多的人，为他们节省了时间，慢慢地，这个想法开始流行起来。同样的故事也发生在晶体管上，晶体管在 20 世纪 30 年代和 40 年代被开发并用于军事电子设备。直到 20 世纪 50 年代初，一些人才有了将这项技术应用于大众晶体管收音机的想法，很快就发现了历史上最受欢迎的电子设备。

在所有这些案例中，有趣的是导致这些发明的特殊过程:一般来说，发明者有机会接触到可用的技术；然后他们会想到这项技术可以用于其他目的；最后，他们会尝试不同的原型，直到找到合适的。允许这一过程的是发明者愿意用不同的眼光看待日常事物，并想象它们的新用途。对于那些被僵化的看待方式所困的人来说，对旧应用的熟悉催眠了他们，使他们看不到它的其他可能性。归根结底，拥有一个灵活、适应性强的头脑——这往往足以让一个成功的发明家或企业家与众不同。

在 Viaweb 上赚了一大笔钱后，格雷厄姆突发奇想，想为互联网写文章——这是他相当独特的博客形式。这些文章使他成为各地年轻黑客和程序员中的名人。2005 年，哈佛大学计算机科学系的本科生邀请他做了一次演讲。他没有通过分析各种编程语言来让他们和他自己感到无聊，而是决定讨论技术创业公司本身的想法——为什么有些成功，为什么有些失败。讲座非常成功，格雷厄姆的想法非常有启发性，以至于学生们开始围绕他们自己的创业想法向他提问。当他听的时候，他能感觉到他们的一些概念不是很离谱，但是他们非常需要塑造和指导。

格雷厄姆一直打算尝试投资他人的想法。他曾是他项目中一位天使投资者的受益者，通过帮助他人来回报这种恩惠是正确的。问题是从哪里开始。大多数天使投资者在开始投资之前都有一些相关的经验，他们倾向于从小规模开始尝试。格雷厄姆没有这样的商业经验。基于这一弱点，他想到了一个乍一看似乎很荒谬的想法——他将同时向 10 家初创公司投资 15，000 美元。他会通过刊登招聘广告并在申请者中选择最佳者来找到这十个潜在客户。在几个月的时间里，他会指导这些新手，帮助他们开始实施他们的想法。为此，他会从任何成功的初创企业中抽取 10%。这就像是科技创始人的学徒制度，但它真的有另一个目的——它可以作为他在投资业务中的速成班。他将是一个糟糕的第一投资者，而他的学生将是糟糕的企业家，这让他们成为完美的一对。

然而，他再次招募罗伯特莫里斯加入他的业务。然而，在训练的几个星期后，他和莫里斯意识到他们实际上是在研究一些强大的东西。由于他们在 Viaweb 上的经验，他们能够给出清晰有效的建议。他们指导的创业想法看起来很有前途。也许他们采用的这种快速学习方法本身就是一种有趣的模式。大多数投资者一年只处理几个创业公司；他们忙于自己的事业，无法处理更多的事情。但是，如果格雷厄姆和创始合伙人杰西卡·利文斯顿专门致力于这种学徒制度会怎么样呢？他们可以大规模生产这项服务。他们可以资助数百家而不是数十家这样的创业公司。在这个过程中，他们会突飞猛进地学习，这种指数级增长的知识会导致越来越多的成功创业。

如果它真的起飞了，他们不仅会发财，而且还会对经济产生决定性的影响，让成千上万精明的企业家进入这个系统。他们将新公司命名为 Y Combinator，并认为这是他们改变世界经济格局的终极手段。

他们用一路走来学到的所有原则来指导他们的学徒——寻找现有技术的新应用和未被满足的需求的好处；与客户保持尽可能密切的关系的重要性；保持想法尽可能简单和现实的需要；创造优质产品和通过工艺取胜的价值，而不是专注于赚钱。

随着学徒们的学习，他们也在学习。奇怪的是，他们发现真正造就成功企业家的不是他们想法的本质，或者他们上过的大学，而是他们的实际性格——他们愿意调整自己的想法，利用他们最初没有想到的可能性。这正是格雷厄姆在自己和其他发明家身上发现的特质——思维的流动性。另一个重要的性格特征是极其坚韧。

多年来，Y Combinator 以自己的方式不断发展，并以惊人的速度持续增长。Y Combinator 旗下公司的总价值目前为 137 亿美元，显然还有进一步增长的潜力。

教训很简单——真正的创造力是由我们精神的开放性和适应性构成的。当我们看到或经历某件事时，我们必须能够从几个角度来看待它，看到明显的可能性之外的其他可能性。我们想象我们周围的物体可以被用于不同的目的。我们不会因为纯粹的固执而坚持我们最初的想法，或者因为我们的自我与它的正确性联系在一起。相反，我们随着当下呈现在我们面前的事物前进，探索和利用不同的分支和突发事件。因此，我们设法将羽毛变成飞行材料。不同之处不在于大脑最初的创造力，而在于我们看待世界的方式，以及我们重新构建所见事物的流畅性。创造力和适应性是不可分的。

你刚刚读到的来自《纽约时报》畅销书排行榜，罗伯特·格林的《掌握》[](http://www.amazon.com/exec/obidos/ASIN/014312417X/ryanholnet-20)**，现在已经出了平装本。在这本书中，罗伯特·格林研究了当代和历史上大师们的生活——从拿破仑到企业家保罗·格拉厄姆，从自闭历程到查尔斯·达尔文——并揭示了他们的秘密。你会跟随* [*走向精通*](http://www.youtube.com/watch?v=qCwJeDnEw2M) *之路吗？**
