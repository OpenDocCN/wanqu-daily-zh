# “机器人”是个错误的名字..以及为什么认为自己很傻的人是错的。



现在，一些人看到像[天气猫](http://poncho.is)和[在笨重的人工智能用户界面上订购鲜花](http://recode.net/2016/04/12/facebook-messenger-1800flowers-spring-commerce-bots/)这样的例子，这是可以的，不是很好，他们认为机器人是愚蠢的。我相信这是大规模平台背后的微应用的开始，这将导致更多的关注和接触初创公司和更高兴的用户。

## 你在移动应用中真正得到了什么？



Shyp 是一次令人愉快的经历。你需要退货或运送任何东西到任何地方吗？按下你手机上的一个按钮，一个快递员就会立刻取走你的物品，而且价格很低。但是实际的移动产品体验如何呢？你拍下一件物品的照片(相机)，告诉他们你在哪里(位置)，提供你的信用卡，如果有问题就联系客户服务部。

所有这些功能都已经内置在 Messenger 中。

## 写一次，(最后)到处跑？

当我们从网络上的多种浏览器转移到手机上的多种本地移动应用程序时，团队必须为不同的手机编写不同版本的应用程序，为不同尺寸的手机进行测试，并且非常痛苦(配置文件 ftw！)提交到多个应用商店。通常团队一开始会忽略某些平台(Android？)，失去了接触数百万用户的机会。

如果 Shyp 今天就开始，他们可以简单地在脸书上构建一个微型应用程序，节省无数的开发时间。SaaS 软件的爆炸式增长(从 AWS 到 MixPanel 到 SendGrid，等等)让初创公司可以在更短的时间内完成更多的事情，同样，不必构建多个丰富的移动应用程序也可以节省时间。团队可以专注于他们的核心价值和成长。

在一个微应用的世界里，你在脸书平台上建立一个体验，然后接触到 1B 人。

## 初创公司可以更加关注增长

无论是调整社交体验还是优化付费用户获取，初创公司都需要专注于向数百万人提供愉快的体验。很多团队从哪里获取用户？脸书。目前用户体验如何？点击一个原生广告，它会通过 3 个链接追踪器将你带到一个移动网页，或者直接带到一个未经优化的应用商店页面。点击安装，可能输入你的密码，等待应用程序下载，打开它，重新登录或创建一个帐户，也许只有这样才能提供一些乐趣。就这样**可怕地**破碎了。

在一个微型应用程序的世界里，点击一个 Messenger 应用程序后，你的[丰富的个人资料](/lightspeed-venture-partners/5-observations-from-f8-on-messenger-bots-7edcd5bac14f)已经在那里了(个人信息、朋友、信用卡、地址、衣服尺寸等等)，你可以立即取悦用户。这种转化的提高仅仅意味着更多的用户。

## 更丰富的用户界面/UX...即将推出。



Who remembers FBML? This is ugly, but you get the point.



如果你相信将会有微型应用，那么用户界面/UX 将会大大改善。正如我在[之前的一篇文章](/lightspeed-venture-partners/5-observations-from-f8-on-messenger-bots-7edcd5bac14f)中提到的，脸书曾经在脸书 chrome(大约 2007 年)中与 FBML 合作过一次，他们给开发者提供了丰富的插件来改善用户体验。这也会发生在这里。无论是 FBML 的小工具还是 react 组件，微应用都将不再是笨重的多选滑块，到处都是重复的文本。

此外，许多人关注机器人和人工智能，但这只是取悦用户的一个因素。像[Fin](http://www.businessinsider.com/andrew-kortina-sam-lessin-found-the-fin-exploration-company-to-create-her-operating-system)/[Operator](http://techcrunch.com/2015/04/07/operator/)/[M](http://www.wired.com/2015/08/facebook-launches-m-new-kind-virtual-assistant/)这样的产品，至少有一部分是由真人驱动的。其他重要的成分是用于处理文本输入的 [NLP](https://www.google.com/url?sa=t&rct=j&q=&esrc=s&source=web&cd=1&cad=rja&uact=8&ved=0ahUKEwi92cj-347MAhVKwGMKHZISDOEQFggcMAA&url=https%3A%2F%2Fen.wikipedia.org%2Fwiki%2FNeuro-linguistic_programming&usg=AFQjCNGIZ1Al8oZ6Oo0uAjj9glUf82OrXw&sig2=fScwaJoXmcXDbqFTojaVVg) 、用于处理音频/照片/视频等输入的 [CV](https://en.wikipedia.org/wiki/Computer_vision) 和[音频到文本的深度学习改进](https://www.technologyreview.com/s/544651/baidus-deep-learning-system-rivals-people-at-speech-recognition/)等等。它不仅仅是文本和人工智能。

在微应用的世界里，你可以利用更丰富的用户界面和多种 SaaS 产品来处理输入，并把更多的时间放在创造足以取悦用户的体验上。

## 脸书广告持续增长

脸书为什么要这么做？如果创业公司能够接触并取悦更多的用户，你会更多地使用他们的广告产品。更高的转化率意味着更快乐的广告客户和更多的初创公司用户。
