# **夏普创业:当 PayPal 发现产品符合市场需求时**

**1000 多亿美元产品 20 周年回顾**

20 年前的这个月，PayPal 发现产品符合市场需求。我清楚地记得那个时刻，因为这是一个至今仍然有效的策略的生动例子。我称之为“急速前进”

一般的初创公司会针对一系列用例推出一系列功能来吸引许多潜在用户，而夏普的初创公司则专注于针对最绝望的客户群的几个杀手级功能。简而言之，它找到了进入市场的突破口。
