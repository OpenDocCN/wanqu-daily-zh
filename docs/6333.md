# 财富的最高形式



财富很容易衡量，但很难估价。

当乔治·范德比尔特(George Vanderbilt)搬进比尔特莫尔(Biltmore)时——这是美国最大的住宅，占地 17.8 万平方英尺——1899 年的一份报纸想知道这有什么意义。

它说，镀金时代，这个国家最富有的人的目标似乎是“不计代价，致力于享乐”但通常他们得到的是相反的结果:“不顾快乐，一味追求消费。”

乔治没有在这座拥有 250 个房间的豪宅里呆多久，他去世时，这座豪宅几乎让他破产。

在比尔特莫尔建造的 20 年前,*纽约日报论坛报*写道“范德比尔特的钱肯定没有给现在的索取者带来幸福”

那不是私下的嫉妒。拥有世界上最大的财富，范德比尔特家族似乎致力于证明金钱买不到幸福。他们更进了一步，表明如果管理不善，金钱实际上可以买到怨恨、不安全感和社会焦虑。它可以批量购买。

金钱可以买到快乐，就像毒品可以带来快乐一样:如果做得对，就难以置信；如果用来掩盖弱点，就危险；如果数量不够，就灾难性。

财富的最高形式有不同的衡量方式。

有几个突出的例子:

**1。控制你的时间和醒来说“我今天可以做任何我想做的事情”的能力**

五岁的富兰克林·罗斯福抱怨说，他的生活被规则所支配。所以他的母亲给了他一天自由的时间——他可以做任何他想做的事情。萨拉·罗斯福在那天的日记中写道:“完全出于自愿，他心满意足地回到了自己的日常生活中。”

因为你想而努力工作和因为别人告诉你必须努力工作，以及如何做和何时做是有区别的。即使你在做同样的工作，按照你自己的方式独立工作会改变一切，就像露营时睡帐篷很有趣，无家可归时很痛苦一样。

对我来说，财富的最高形式是控制你的时间。

财富可以导致时间独立，但它从来没有得到保证。它可以是相反的，因为无论什么创造了财富——无论是公司还是遗产——都创造了对你的时间的要求，与其财务回报成相等的比例。很多首席执行官都属于这一类:他们拥有大量的财富，却没有一刻空闲时间，甚至在需要的时候也没有时间安排控制权，这本身就是一种贫困。

查理·芒格总结道:“我没有致富的打算。我只是想变得独立。”这是一个美好的目标，比净资产更难衡量。

**2。当金钱变得像氧气一样:相对于你的需求来说，它是如此丰富，以至于尽管它是你生活中至关重要的一部分，你也不必去想它。**

在纪录片《凡尔赛女王》中有这样一个场景，一个男人的儿子赚钱能力仅次于他花钱的欲望，导致家族财富在破产边缘萎缩:

> 在我结婚的那天，我父亲发表了一个演讲，他看着我的妻子说:“你这辈子再也不用担心任何事情了。”
> 
> 但是现在我们每天都在担心。

一种更高形式的财富正在避免这种混乱。这不一定和你有多少钱有关。

记住两件事:

多年来我注意到的一件事是，一些最富有的人无时无刻不在想着钱——这很明显，因为这是因果关系。但这是一个重要的观察结果，因为大多数人尽管渴望成为最富有的人之一，但实际上想要的是不同的东西:不必考虑金钱的能力。

这是一种不同的技能，但当你让它发挥作用时，它会很强大。一个人相对于收入的期望值是经过校准的，所以他们甚至不用考虑钱，这个人比一个拥有更多钱但不断想让数字起作用的人拥有更高形式的财富。

**3。允许智力诚实的职业。**

这包括:当你不知道的时候，能够说“我不知道”。能够说出你所在行业的关键事实，而不用担心遭到报复。有能力犯合理的错误，并公开承认错误，没有过度的担心。而不是假装看起来很忙来证明你的薪水。

有高薪职业允许所有这些事情。但是有太多的人不这样做，很多人所谓的“努力工作”和“磨砺”只是想办法掩盖真相。一份让你变得开放和诚实的工作会给你带来难以估量的回报。
