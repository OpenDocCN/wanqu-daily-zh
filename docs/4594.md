开源社区目前发生的事情令人沮丧。尤其是当你意识到许多创业公司和企业*(包括脸书)*的存在都要感谢开源；因为如果他们不得不预先支付高昂的专有软件许可费，他们将无法生存。

开源就是创建社区来共同开发更好的软件。它永远不应该被用作交换人民权利的市场——就像脸书打算做的那样。

脸书正在他们所有的项目中推行一种叫做“BSD +专利”的授权模式，包括广受欢迎的 React。

“BSD +专利”本质上意味着代码是开放的(供每个人查看和使用)，但它的版权归脸书所有。BSD 许可证授予您版权使用许可证。此外，只要你*对他们*好**从不起诉脸书侵犯专利**，他们就会授予你专利许可。

一旦你起诉脸书，**你对 React**——以及你碰巧使用的任何其他脸书‘开源’技术)——**的专利权就自动被撤销**。

## *冰箱图书公司。*

*为了便于说明，假设你是一家冰箱公司“Fridgebook Inc .”，销售智能冰箱。您的冰箱有一个运行您的专有应用程序的屏幕，您使用 React 作为 UI。*



*一夜之间，你听说**脸书决定进军冰箱行业**，他们宣布仅用一周时间就在全球推出他们的新产品:“FB 冰箱”。*

> *假设脸书用“FB 冰箱”公然侵犯了你的一些专利，你能做什么？*

*嗯，你不能马上起诉他们。 *你在面向客户的应用程序上使用 React，记得吗？**

*如果你在迁移到其他东西之前起诉他们(比如 [vue.js](https://vuejs.org/) )，你将立即失去 React 的专利授权，并且突然你自己违约了，**独自一人对抗一家将近 5000 亿美元的公司的非法使用软件的潜在诉讼**。*

*很明显，你不想打断客户服务。*

*因此，如果你想起诉他们，或者至少拥有这样做的任何筹码，**你将需要找到一种解决方案，在创纪录的时间内从 React 迁移出去**。*

*你的处境很尴尬，对吧？这几乎是一种敲诈的情况。**解决办法？首先不使用 React**，并保留主张自己权利的自由。*

# *脸书的解释*

*我最后一次看到，开源哲学围绕着社区，在那里有才华的人贡献他们的一粒沙子，一起构建更好的软件，推动技术更进一步。*

*这就是 Apache 软件基金会、Linux 基金会等的精神。谁是开源领域的**关键参考。***

## ***那么，为什么要将专利引入开源呢？***

*脸书发布了一份官方解释，我用几句话概括一下:*

> *脸书收到大量毫无价值的专利申请。他们浪费了大量资源来对抗他们。所以他们决定利用他们开源项目(如 React)的成功，引入一个*特洛伊木马*来阻止用户对他们提出专利索赔——理论上是没有价值的。他们没有回报这种限制。*

*但是这是最重要的部分。他们声称其他每一家发布开源软件 ***的公司都应该做同样的事情*** 。*

*****更新:*** [*这里我分析一下*](/@raulk/list-of-companies-and-popular-projects-by-the-open-source-licenses-they-use-35a53eaf1c80) *还有哪些大型公司采用这种许可模式。提示:不多，就 2 个。]***

****不幸的是，这是行不通的，最终会导致一个封闭源代码的行业再次出现，原因有几个:****

1.  **这需要市场中最大的玩家达成共识，这些玩家拥有真正的专利库，作为 T2 对抗竞争对手的杠杆。突然间，这些军火库的价值将会变成 0 美元。**
2.  **达成共识是非常不可能的。只要 ***一个流氓公司*** 不加入，其余的就需要保持“他们的守卫/专利武库向上”。**
3.  **如果所有巨头都同意在“BSD +专利”计划下开源，交叉采用将会停止。为什么？如果谷歌在“BSD +专利”下发布 Project X，亚马逊真的很喜欢它，而不是采用它并失去起诉谷歌专利的权利，**他们会离开并自己开发它**。**
4.  **这意味着**社区不会围绕这些产品形成**。社区是开源产品的燃料和动力。**如果没有点燃一个社区的机会，就没有理由开源。****
5.  **最终，随着上述情况一次又一次地发生，巨头们将不再看到开源产品的价值，行业将最终陷入闭源模式。**

# **脸书对开源哲学的不道德使用**

**专利保护想法和发明。在大多数情况下，专利主张案件不是非黑即白——赢或输。**侵权评估复杂且成本高昂。**一场诉讼的提起和追究可能要花费数十万或数百万。你可能有 85%的信心认为 FB 侵犯了你的一项专利，但即使追究它也要花费你很多钱。**

**如果最重要的是，你将需要投资首先迁移到一个不同的前端框架上，**并确保你的所有客户都在使用你的新产品版本*(如果你正在使用 React Native 呢？您的用户可能不会立即升级应用程序！)*** ，在你甚至可以提起诉讼之前，你认为那是开源哲学的一种诚实的、合乎道德的用法吗？**

****底线:****

> **开源不是一种“等价交换”的交易。开源是关于创建社区来共同构建更好的软件。它绝不应该被用作交换权利的市场。**

**事实上，脸书本身就是建立在许可的开源软件的基础上的。**

# **如果你是一家初创公司，为什么你可能不想做出反应**

**如果你正在创业，我假设你和你的投资者都希望在某个时候获得价值*百万美元的退出机会*，对吗？**

**你希望向所有收购者敞开大门，尤其是大型收购者:苹果、微软、谷歌、亚马逊等。**

**这些公司很可能拥有针对脸书的专利库——即使他们没有——如果时机成熟，他们也不想放弃起诉脸书的权利。**

**如果你的产品是建立在 React 之上的，那么**获得你就意味着失去这些权利**，这是他们可能不愿意做的事情。**

> **基本上，一些潜在买家可能对你不感兴趣，如果购买你意味着他们不得不放弃起诉脸书侵犯专利的权利。**

**所以，如果你想保留你的选择…**
