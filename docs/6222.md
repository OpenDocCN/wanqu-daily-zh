# 在我的 Om 上，一个重要的创始人指标

> 原文：<https://om.co/2019/07/31/a-founder-metric-that-matters/?utm_source=wanqu.co&utm_campaign=Wanqu+Daily&utm_medium=website>

随着优步作为一个企业的发展，我们已经看到他们用户体验的几乎每个方面都出现了质的退化，无论是驾驶安全、路线优化还是一般的应用程序到路边的时间。我想知道优步首席执行官达拉·科斯罗萨西是否注意到了这一点。坦率地说，我甚至不知道他是否使用优步，优步 X，优步选择和/或优步池。达拉的前任并不羞于使用他的产品，尽管他经常会惹上麻烦。现实是，嗯，混乱的。不过，如果能看到达拉发布他的季度和每日优步使用仪表板，那就太好了。

事实上，科技行业的所有创始人和首席执行官都应该分享个人指标，显示他们每天或每周使用自己服务的次数。例如，达拉昨天骑了多少次优步？博客平台的头头们一个月发布了多少帖子？做健身 app？展示你的用法！

我知道杰克经常使用他的服务，因为我可以在他的时间表中看到。扎克也是如此，尽管与他的生活较少受政治操作者控制的时候相比，现在少了很多。Airbnb 创始人[是他们平台](https://om.co/2019/03/13/brian-chesky-eats-his-own-dog-food/)的活跃用户，而[以不惜一切代价回避酒店体验而闻名](https://om.co/2019/03/13/brian-chesky-eats-his-own-dog-food/)。看看苹果首席执行官蒂姆·库克(Tim Cook)就知道了，他是苹果手表的拥护者，也是手表作为注重健康的可穿戴设备的活生生的体现。

当创始人和高管是他们产品的积极用户时，这表现在产品的有效性和变化的速度上。我的意思是，如果“大老板”不看，你真的认为任何产品人员会不断尝试改进，而不是稳扎稳打吗？

史蒂夫·乔布斯可能不是每个人都喜欢的那杯茶，但他确实痴迷地使用他的产品。这种 KPI 是有意义的，它反映了这些高管没有忘记谁体验他们的产品。如果你希望人们使用你的产品，你也应该使用它。就像其他人一样。
