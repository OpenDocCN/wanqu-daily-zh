# 数学很难

开发新事物是困难的，但这并不意味着一旦某件事情被开发出来就很难去做。很难想出来但很容易抄出来的东西就是‘数学难’。

许多数学问题要花很长时间才能想出解决的方法。但是，一旦你有了一个方法，通常答案可以很快得到验证，方法也很容易学会。毕达哥拉斯需要聪明才能发明他的定理，但现在孩子们在学校里学习。其他数学难题包括食谱、设计和新想法。

除非毕达哥拉斯能够将他的方法保密，这很难做到，否则他的定理不会帮助他建立一个可防御的公司。甚至小学生也能模仿它。

这就像一种新的用户体验或界面的创业想法——它们很难理解，很难保密(尤其是当你的用户体验到这个秘密的时候),并且很容易复制。

但是如果你能保守这个秘密呢？毕竟，如果你小心的话，“商业秘密”可以保留一段时间。

问题是，你的秘密越成功，别人就会越努力去发现它。你可能真的很聪明，也很努力，但是有很多聪明人会努力工作以获得成功。认为你会有一个完全独特的见解是不现实的，更不用说一个别人想都想不出来的了。

如果这种洞察力是创办一家成功公司所必需的防御标准，从定义上来说，几乎没有人会去创办一家公司。不管你的发明有多聪明，你最终都会在其他方面展开竞争。

数学的难度，也就是解决问题的难度，充其量只能为你赢得时间。多少时间？可悲的是，有些时间与你的成功成反比。你的秘密越有价值，你剩下的时间就越少。

# 努力健身

另一种硬度是做某些事情的内在硬度。有些事情很难做，即使每个人都知道怎么做。这是健美难。

实现一个健美运动员的体格并不复杂，无论无尽的补充剂表明什么。你需要:经常负重锻炼；并且坚持吃富含蛋白质的食物。这两者都不是秘密，理解起来也不复杂。大多数人，不管身材如何，都知道这些东西。问题是做这些事情本身就很难。这需要艰苦的活动，远不如和朋友边喝啤酒边吃汉堡有趣。

最有价值的成就是努力健身。学习一项新技能，变得富有，成为一个好伙伴——这些都需要努力。几乎每个人都或多或少知道如何实现这些目标，但很少有人真正做到。这就是为什么每个行业都有一个补充捷径:尖头吉他；自助书籍；还有带着花哨的卧室技巧的杂志。

健美硬创业公司比比皆是。按需服务、市场、电子商务——它们是 Rocket Internet 投资组合的支柱。他们没有毕达哥拉斯的秘密。这些业务运营起来很痛苦，需要数千人的协调和复杂的操作。

没有秘密，击败其他人是困难的。但是，如果你只是优秀，健美在经济上是不能自我维持的:你需要成为最好的。尽管有微小的差异，胜利者仍然会得到一切。

因此，健美充满了类固醇和技术，以找出微小的优势。健身创业公司也不例外——只是他们的兴奋剂是现金、经验和数千名受过创伤的前银行家和顾问。

你通过努力锻炼身体获得的成功越多，就会有越多的人受到激励去尝试做同样的事情。如果这是一家值得创业的公司，最终回报一定会超过艰辛。一旦情况明朗，你的竞争对手就会带来类固醇。

健美硬度和数学硬度一样，为你争取时间。多少时间？同样，时间的长短与你的成功程度成反比。进展越顺利，你剩下的时间就越少。

# 努力让你开始

两种硬度都是开始的好方法。努力做某事会给你带来成功的机会。但是，一旦你变大了，就必须有别的东西把你留在那里。

可防御并不意味着长期的强硬，即使强硬在开始是有用的。对于最好的公司来说，困难过后，事情会变得容易。

他们的长期优势不需要新的见解或持续的努力:一旦赌注足够高，就很难像这样竞争。他们的优势是独特的资源积累——像谷歌的数据集或苹果的品牌，或规模驱动的产品差异化——像优步的流动性或亚马逊的交付基础设施。你需要滚雪球。

努力做一些事情可以让你在竞争中领先，但这并不是让你坚持下去的原因。
