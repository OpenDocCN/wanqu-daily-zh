# LLC 对 C 公司

有限责任公司和股份有限公司是美国两种主要的公司实体。每种实体类型都有一些特性，这些特性对某些企业来说比其他企业更有利。

我们在这里收集了一些信息，这样你就可以咨询你的专业顾问，做出明智的选择。

## 有限责任公司

有限责任公司是一种根据经营协议组建的公司，经营协议是所有者(称为“成员”)之间的一份合同，规定了公司的经营方式以及经济负担和收益如何在合伙人之间分配。

如何组建有限责任公司的可能性几乎是无穷无尽的，这可能是一件好事，也可能是一件坏事。这使得与有限责任公司的接触具有挑战性，因为人们必须检查经营协议(以及成员之间签署的其他潜在合同)，以了解公司是如何治理的。相比之下，公司更为标准化:它们有共同之处，如代表所有权的股票，由董事会管理，由高级职员处理日常业务，等等。

有限责任公司有一些共同的特征:

*   有限责任公司旨在为创始人提供有限责任，将企业的债务和义务从企业家转移到公司本身。

*   有限责任公司报价**转嫁税**；有限责任公司的所有者通常为企业收入缴纳个人所得税。

## 公司

一个 C 公司是一个实体，它被设计成充当企业的*经营者*和企业的*所有者*之间的抽象层，他们可能参与也可能不参与运营。所有权由股份来追踪，每一份股份对应于企业控制权的一个确定部分和企业经济利益的权利。所有者被称为股东。

许多家喻户晓的公司是 C 公司；一个人可以拥有谷歌的股份，而无需为在那里工作承担任何责任。这种假设，即控制权和所有权可能是通过 C 公司的机制和监管分开流动的。特拉华州拥有高度发达的公司法体系，在发生法律纠纷时具有高度的可预见性。

C 类公司有几个共同的特征:

*   c .公司打算提供**有限责任**；股东一般不对公司的债务和义务承担个人责任。

*   c 公司根据自己的利润被征收公司税(并且有广泛的申报义务)。如果公司向股东派发股息(或者，如果公司向员工支付工资，股东将被单独征税)。

## 公司的标准特征

有限责任公司和股份有限公司都是公司。在美国，政府和你可能希望与之做生意的公司等第三方一般都乐于与这两类公司打交道；在一些国家，情况并非如此(在这些国家，当地的有限责任公司相对于当地的 C 类公司可能处于商业劣势)。

有限责任公司和股份有限公司都旨在限制所有人和高级职员对公司行为和公司债务的责任。

有限责任公司和股份有限公司都可以是合同的一方，可以拥有其他公司(反过来被其他公司拥有)或几乎任何其他资产，可以获得银行服务，并可以经营企业。

## 资产和知识产权

许多兼职项目、小团队、自举企业或还不知道自己长大后想做什么的企业的创始人都会选择有限责任公司。(它们还可以扩展以支持几乎任何规模的企业。例如，Basecamp 是一家有限责任公司。脸书开始时是一家有限责任公司，后来转为股份有限公司。)

有限责任公司非常适合于副业项目的一个原因是，资金和知识产权(IP)可以在有限责任公司成员和有限责任公司本身之间相对自由地流动，通常没有交易发生在有限责任公司时产生的税务后果，而且通常只需最少的仪式。

这在概念上对一家 C 公司是可行的，但它涉及更多的记录保存和(潜在的)棘手的税务考虑，例如，特别是关于知识产权。C 类公司的大多数重大行动也需要一些仪式，如公司的正式决议或股东的投票；有限责任公司经营协议通常授权所有者和/或管理者采取行动。

## 征税

就美国税收而言，有限责任公司被视为转让实体；他们自己并不报税，但是他们的收入在他们的所有者的个人所得税申报表上申报。公司提交他们自己的纳税申报表。(有些令人困惑的是，C 类公司有时可以选择转嫁地位，而 LLC 可以选择像公司一样纳税，但这些都不是它们的默认待遇。)

流经有限责任公司的资金按照有限责任公司所有人的水平征税；流经公司的资金在公司层面和转移到所有者时都要纳税(作为工资或利润分配)。

这些实体的不同税收待遇可能会产生有趣的影响，特别是如果公司正在亏损，正如许多公司在其早期所做的那样。在任何一年发生亏损的 C 类公司通常会将亏损计入未来的纳税年度，并用来抵消未来的利润。有限责任公司的亏损通常可用于抵消同一纳税年度内所有者的收入，例如，雇佣收入。

考虑这样一种情况，一家公司在运营的第一年花费 10，000 美元而没有收入。一家 C 类公司可能不得不将这 10，000 美元的损失推迟到下一年，以获得任何税收优惠。有限责任公司可能被允许减少其所有者的总收入 10，000 美元；这可能导致业主的所得税账单减少数千美元。对于在美国从事技术工作的人来说，这可能会导致他们获得一大笔退款，他们可以用这笔钱来资助企业的发展或用于任何其他目的——这是他们的钱。

## 国际所有者

美国和特拉华州目前都没有对有限责任公司或有限责任公司的所有者提出公民身份或居住要求。也就是说，拥有一家有限责任公司会让非居民或非公民所有者面临非常复杂的税务情况。非居民美国公民可能有义务对其有限责任公司的收入在美国纳税，并且还必须考虑在其当地司法管辖区如何对该收入征税。

例如:考虑一个有两个所有者的有限责任公司，一个在美国，一个在日本的非美国公民。美国可能会对日本的所有者从有限责任公司获得的收入征税。日本的所有者也可能因同样的收入在日本被征税。这大大增加了他们纳税申报的复杂性。

## 投资

与投资有限责任公司相比，专业投资者更倾向于投资 C 类公司。引用 Orrick 法律指南中关于 Stripe Atlas 的话:

> [M]由于收益和损失的传递性质，任何类型的投资者都不会对投资有限责任公司感兴趣(或可能被法律禁止)。

有限责任公司形式的极大灵活性也意味着试图投资于其中的投资者必须进行大量的法律尽职调查，以确保他们购买的是他们预期购买的东西。许多投资者不想做大量昂贵的法律工作作为投资的条件；他们更喜欢按照标准化的条款投资标准化的公司。c 公司更适合这种偏好。

如果一个人作为有限责任公司收到投资要约，许多投资者(包括 Y Combinator)可能会要求他转换为特拉华州的 C 公司，作为在投资之前*接受投资的要求。*

## 员工所有权

有限责任公司和股份有限公司都可以雇佣员工。虽然原则上可以给员工所有权，但不能控制有限责任公司，但这并不简单。c .公司有很好的机制来发行员工股权或股权期权，有很好的税务后果，以及整个科技行业对这种所有权形式的文化和基础设施支持。

与成为有限责任公司的成员相比，员工和顾问可能更愿意接受股权，这可能会在有限责任公司的生命周期内使他们自己的税务状况复杂化(例如，即使他们离开了有限责任公司)。

## 保留退休金的权利

授权是一种机制，公司的创始人或员工通过这种机制逐渐获得所有权。

Stripe Atlas 有限责任公司不包括股权归属计划。虽然公司有明确的方法来区分企业中的合伙人和所有者，但这些概念在本质上是混合在有限责任公司中的。离开有限责任公司的所有者可能需要离开的合伙人和剩余的合伙人就分离条款进行协商(尽管 Stripe Atlas 有限责任公司有一些条款来处理这一过程)。一些公司希望这是一次彻底的决裂，可能会有剩余的一方买断离职合伙人的股权。有些人可能想继续向离职的合伙人支付一部分利润。这些决定很复杂，通常取决于公司的情况和所有者的愿望。

有限责任公司的授权标准也不像科技行业的 C 类公司那样明确；许多有限责任公司是在家庭成员之间成立的(在这种情况下，关系考虑可能会压倒合同安排)，不导致合伙关系解体的离职更为罕见，没有控制权的所有权不会像在 C 类公司中经常出现的那样有价值，C 类公司通常考虑以更大的规模经营。

## 将有限责任公司转变为股份有限公司

有限责任公司通常可以转化为 C 类公司。(原则上，C 类公司可以转变为有限责任公司，但很少这样做。)

由于有限责任公司受成员之间的合同安排的约束，要消除有限责任公司转换过程中的风险，通常需要大量的法律审查，这增加了转换的金钱和时间成本。

Stripe Atlas 有限责任公司有书面的运营协议，以预期可能的转换。开始转换有一个明确的流程。所有权以可以映射到未来 C 公司的股票发行的单位来跟踪。Orrick 在创业公司方面有丰富的经验，他概述了转换过程，并在其法律指南中提供了带注释的模板，Stripe Atlas 用户可以定制转换。

## 有限责任公司在哪里成立？

美国的每个州都可以成立有限责任公司和有限责任公司，不管创始人住在哪里，也不管公司的实际经营地点在哪里。每个州都试图为其有限责任公司和 C 类公司提供产品，使其具有吸引力，以吸引费用和促进当地经济发展。

目前，所有 Stripe Atlas 有限责任公司均在特拉华州成立，可能需要在业主居住或经营的州注册为“外国”有限责任公司。这通常是一个简单的过程，每年只需支付一定的费用。对居住在美国以外的创始人的监管差别很大；请咨询当地会计师或律师或当地政府。

Orrick 的法律指南描述了在特拉华州成立有限责任公司的一些好处:

> 特拉华州有限责任公司的优势在于组建简单、极其灵活(对管理和治理安排的限制很少)，并且更为你将与之互动的各方所熟悉。此外，如果您决定将特拉华州的有限责任公司转变为特拉华州的公司，那么这一过程非常简单和常见，有标准的表格可以处理这一过程的大部分内容。

## 条纹阿特拉斯有限责任公司简介

Stripe Atlas 有限责任公司的一些特征是许多有限责任公司共有的；有些是针对技术创始人的需求定制的。

您应该在签署之前阅读完整的运营协议，因为这是一份法律合同，但这里有一些关于其条款的信息:

*   有限责任公司是根据特拉华州的法律组建的。

*   Stripe Atlas LLC**由经理**管理，允许日常决策和管理责任与所有权(作为成员)区分开来。公司可以有非所有者的经理或没有管理责任的所有者。

*   Stripe Atlas LLC 在其运营协议中包含了一项**知识产权转让**,该协议在 LLC 成立时将已经创建并由成员持有的相关知识产权转让给 LLC。

*   Stripe Atlas LLC 的所有权通过 10，000，000 个单位进行跟踪，其功能类似于股票，代表公司的**简单所有权股份，只有**单一类别的所有者**。** 
*   Stripe Atlas LLC **支持添加新所有者**(经现有所有者一致同意)。

*   Stripe Atlas LLC **支持移除所有者**(经所有所有者一致同意)。

*   Stripe Atlas LLC 包含简化从**转换为特拉华州 C 公司**的过程的语言。

*   Stripe Atlas LLC ***不*包括所有权归属**，原因将在《法律指南》中讨论。

所有 Stripe Atlas LLC 用户都会收到 Orrick 法律指南，该指南提供了有关 LLC、C 类公司、运营协议和 Stripe Atlas 使用的其他模板的详细信息。
