# 自动化使人类劳动力比以往任何时候都更有价值

> 原文：<https://www.vox.com/a/new-economy-future/manual-labor-luxury-good?utm_source=wanqu.co&utm_campaign=Wanqu+Daily&utm_medium=website>

现代经济建立在自动化的基础上，因此很自然地认为未来也将由自动化来定义。似乎每周都有新的 [**研究**](http://www.oecd-ilibrary.org/social-issues-migration-health/the-risk-of-automation-for-jobs-in-oecd-countries_5jlz9h56dvq7-en;jsessionid=4jhdpafin1chj.x-oecd-live-03) 或 [**思考片**](http://www.techrepublic.com/article/ai-will-destroy-entry-level-jobs-but-lead-to-a-basic-income-for-all/) 关于机器人和人工智能的潜在破坏性。

但是，我们对窃取工作的机器人的集体痴迷会导致我们高估自动化的影响——并模糊了经济的重要一点。在许多服务行业，人力劳动是奢侈的标志。

因此，在机器人摧毁制造业工作的同时，对劳动密集型服务的需求也在飙升。

我们可以在周围看到这种迹象。Etsy 正在崛起，这是一个在线市场，其主要卖点是产品不是大规模生产的。人们热衷于使用有机的本地原料的餐馆，这些原料通常来自规模较小、机械化程度较低的农场。小型 [**酒厂**](http://www.wineinstitute.org/resources/statistics/article124) 和 [**酿酒厂**](https://www.brewersassociation.org/insights/4000-breweries/) 的数量呈指数增长，在这些地方，个人接触——通常是参观设施——是卖点的一大部分。

这些都是提高自动化程度在技术上可行的例子，但是公司正在利用他们缺乏自动化作为卖点。他们这样做是因为顾客喜欢与农民、酿酒商和制造他们所购买产品的工匠建立个人联系的感觉。

随着自动化让日常产品变得更便宜、更丰富，人们将越来越多地将支出转向商品和服务，在这方面，与人类供应商的联系被视为一项关键优势。

### 在咖啡店，低效率是奢侈的标志

《华尔街日报》[**2012 年**](http://www.wsj.com/articles/SB10001424052748704164004575548403514060736) 报道称:“由于顾客抱怨这家总部位于西雅图的咖啡连锁店将咖啡制作艺术简化为机械化的过程，还带有流水线的浪漫色彩，星巴克的咖啡师被告知停止同时制作多种饮料。”。

明尼苏达州的一名星巴克咖啡师抱怨说，新规定“在某些情况下使制作饮料的时间增加了一倍。”这不仅对底线不利；它还会产生更长、更慢的队伍，给顾客带来不便。

但是星巴克的管理层明白他们生意中的一些重要的事情:人们去星巴克不仅仅是为了喝杯咖啡——毕竟，有很多更便宜更快捷的方式可以买到咖啡。人们去星巴克是因为他们喜欢这种体验。这种体验有一个重要的表演维度——顾客希望感觉他们的咖啡师在准备他们的杯子时投入了个人的注意力。

星巴克也不是附近最高档的咖啡店。严重的咖啡势利者拒绝喝这种东西，更喜欢有异国情调的公平贸易咖啡豆的独立咖啡店。在这些商店制作咖啡的过程往往比在星巴克更为精细和劳动密集型。由于这些商店通常较小，没有星巴克品牌吸引顾客，咖啡师往往服务的顾客较少——往往导致价格较高。

但是很多咖啡爱好者不在乎。他们认为精心制作的手工咖啡比大众市场的好，他们愿意为此支付额外费用。这有助于解释为什么人们制作咖啡仍然有很大的市场，即使咖啡机变得越来越复杂[](http://dailytekk.com/2015/09/21/coffee-tech-the-10-coolest-and-best-new-coffee-makers-on-the-planet/)**。**

 **### 劳动密集型服务业是美国经济的未来

咖啡远不是美国最重要的产业。但是在美国经济的大部分领域也可以看到类似的趋势。

看到这一点的一个方法是看一下美国劳工部对 2014 年至 2024 年间美国增长最快的职业的预测[](http://www.bls.gov/ooh/fastest-growing.htm)**。一堆的位置被治疗师和护理员占据:物理治疗师和他们的助手和助手，职业治疗助手和助手，家庭健康助手。** 

 **在某些情况下，日益增长的需求是由对个人护理需求更大的老龄化人口推动的。但这还不是故事的全部。护士助产士、遗传咨询师和医学超声波诊断师等职业也有望实现强劲增长。他们主要是针对怀孕的母亲，政府统计数字 并没有预测未来十年出生率会快速增长。

从某种意义上说，自动化这些工作的许多方面是可能的。与其让治疗师上门，不如有人给你发一盘演示治疗技术的视频。一个网站可以向病人提问并推荐练习。如果这些资源不够，病人可以通过视频会议应用程序获得一些个人建议。

对于需要治疗护理的人来说，这种半自动的治疗服务可能比什么都没有要好。但这比和一个全神贯注于你的人面对面交流要糟糕得多。事实上，谈论这种工作的“自动化”似乎是一种错误。由应用程序甚至机器人提供的“治疗”是一种不同的服务，就像自动售货机提供的咖啡与人类咖啡师准备的一杯咖啡是不同的服务一样。

另一个例子:人们对[大规模开放在线课程](https://en.wikipedia.org/wiki/Massive_open_online_course)和其他基于互联网的学习平台非常兴奋。让世界上的任何人都能接触到世界上最好的大学的信息的前景无疑是令人兴奋的。

但 MOOCs 不太可能对传统的四年制大学构成严重威胁。与教授和其他学生面对面的互动提供了很多 YouTube 视频无法捕捉到的价值。大学生有机会加入俱乐部和运动队，建立社交网络，谈恋爱，并受益于个人辅导。大学生活的这些方面可能无法自动化，也没有人愿意这样做。

随着时间的推移，技术进步正在稳步消除制造过程中的低效。但在与服务相关的行业，与他人交谈的“低效率”往往是一个关键的好处。因此，随着美国继续变得更加富裕，我们应该预计这些服务将在经济活动中占据越来越大的份额。
