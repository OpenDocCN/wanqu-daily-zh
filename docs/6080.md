# 单飞成功|凯文·伯克

> 原文：<https://kev.inburke.com/kevin/going-solo-successfully/?utm_source=wanqu.co&utm_campaign=Wanqu+Daily&utm_medium=website>



三年前，我辞掉了工作，开始做全职顾问。效果很好。我开始花更多的时间做我喜欢做的事情，我已经能够为客户提供优秀的产品。我想复习一些成功开展咨询业务的技巧。

*   **多收费**——人人都这么说，千真万确。我开始收取接近`my full time salary / 12`的月租费。这不是一个好主意，因为你有你的雇主不再支付的管理费用-医疗保健可能是最大的一项，你没有带薪休假，合同之间可能有无薪停工期，而且公司可能不会支付你。你需要收取更多的费用才能达到收支平衡。

    我每次都害怕“你的费率是多少”的对话，它们并没有变得更容易。在我报出我的价格之前，我重读了一遍[高科技雇员反垄断案](https://en.wikipedia.org/wiki/High-Tech_Employee_Antitrust_Litigation)的细节来激励自己——它提醒我，我正在与一家毫不在乎的公司谈判，我是唯一一个会为自己辩护的人。如果你认为你不需要额外的钱-无论如何都要得到它，然后在年底捐给慈善机构更多/买 CD/放入股市/交给政府。亚马逊刚刚赚了 110 亿美元，缴税 0 美元；你将比亚马逊的高管多花一美元。

    如果你不确定收费多少，给每个新客户的报价要比上一个多。你的报价通常是你质量的一个信号，所以当你要价更高时，需求并没有真正下降。

    如果你和一个客户一起工作，他们对你的工作非常满意，并且想延长你的合同，考虑要求更高的价格。“现在你知道我有多好了”等。

*   **拿到钱**——签订的合同，完成的工作，在钱打到你的银行账户之前都不重要。我是吃了苦头才知道的。如果一家公司倒闭了，你的发票就一文不值了。你可以保留公司的知识产权，但这可能也毫无价值。你可以起诉，但最终你会赢得一个判决，你可以从一家没有任何报酬给你的公司那里收钱。

    试着尽可能多的预付合同款——我通常要求预付一半或更多。如果一家公司提供 30 英镑，问问你是否可以缩短到 5 英镑或 10 英镑，或者提前提交你的发票。按时提交发票——这是一个代价高昂的错误，等到你意识到它的重要性时，已经为时太晚了。

    尽你所能地努力弄清楚公司的财务状况——如果你能在媒体上做足功课，或者向你的主要联系人提问，比如他们在烧多少钱，你还有几个月的时间。如果一家公司不愿意提供这些信息，这是一个危险信号，他们可能无法支付你工资。

    如果你看到*任何*危险信号——公司想缩短合同，人们开始离开，公司突然削减津贴——告诉你的联系人你需要预付工资，否则你将不再工作。如果他们推迟付款，他们可能根本没有钱付给你。这是一个糟糕的情况，但减少你的损失比给一个实际上付不起工资的人工作要好。

*   **不要按小时收费** -我从来没有真正这样做过，所以我不能说它有多糟糕，但不要这样做。你不希望客户在下午 3 点就把你打发走，突然你失去了原本指望的 3 个小时。每周收费。

*   找一个律师——找一个律师来审查你签署的每一份合同。通读一遍，把有关的事情标记给律师。他们会建议语言。把语言发给公司。你这样做并不是在为难你，公司一直都是这样做的。律师费用昂贵，预计每小时支付超过 400 美元，合同审查可能需要 30-60 分钟。这钱花得值。

    一个值得努力争取的条款是责任限制。你不希望出现这样的情况:由于你犯下的一个错误，造成了 200 万美元的损失，或者一个高价值的客户离开了公司，而公司突然盯上了你拥有的一切。同样，该公司可能希望防止你试图起诉他们，对你的声誉、未来业务等造成高额损害。将双方的总责任限制在合同规模或合同规模的倍数可能是一个好的举措。

*   **注册一家公司**——向你聘请的律师咨询你想成为什么样的公司。一般来说，你越“像公司”，公司就越难拿走你的个人资产。我没有雇员或股东，所以我有一个单一成员有限责任公司，出于税收目的不予考虑——阅读[国税局的描述](https://www.irs.gov/businesses/small-businesses-self-employed/single-member-limited-liability-companies)。有时，当我告诉企业或尝试注册购买时，他们会搞不清楚这意味着什么。尽管如此，它还是很适合我。它可能不适合你——我不是律师，你应该和律师谈谈，学会权衡，看看什么对你的业务有意义。

*   确保与公司签订合同 -你签订的合同应该是你正在合作的客户和你的公司*之间的合同，而不是客户和你个人之间的*。与你的律师讨论这个问题。

*   找一个会计 -作为一个小企业，很多东西都是可以免税的-比如家庭办公室、客户旅行，即使只是在城市的另一边-你想确保你能得到 35%的折扣。会计会帮你做这件事。

*   推销自己——不一定是广告或赞助，而是:所有和你全职工作过的人都应该知道他们现在可以雇用你了。如果他们没有，那就联系人们，让他们知道。建立一个网站，工程师可以把它发给他们的老板。[我的网站并不花哨，但很有效](https://burke.services)。订名片——vista print 是垃圾，【moo.com】订。如果你有一个网站或开源项目，在广告的底部放一个说明，你可以被雇佣，就像这篇文章底部的那个。

*   **为所有事情设立独立账户** -为你的企业设立独立账户。在你的个人账户上办一张商业信用卡或单独的返现卡。我没有为企业注册的支票账户，但我开了第二个支票账户，我称之为“企业账户”。客户把钱存入那个账户，我从那个账户中支付商业信用卡。我甚至有一张单独的信用卡，用于商务旅行。

    这有两个原因。这使得税务会计变得容易多了。我知道商务卡上的每笔交易都是工作旅行，可以报销；7 月 27 日下午 5 点 34 分，当我向 SamTrans 支付 2 美元时，我不必努力回忆我在做什么。

    第二，如果你没有为生意做好记录——如果你把私人生活和生意中的资金“混在一起”——这就让客户更容易追踪你的个人资产，这就是所谓的“穿透面纱”独立账户(以及转账纪律！)让辩称你的业务收入和支出以及个人收入和支出是分开的变得容易得多，即使你不一定有法律结构来支持它们。

    我还为我合作的每家公司设立了一个新的 Github 账户。这避免了电子邮件发送到错误地方的任何问题，或者需要授予/撤销公司使用的任何第三方工具的权限。我使用 [`github.com/kevinburke/swish`](https://github.com/kevinburke/swish) 在我的 Github 账户之间交换 SSH 设置:

```
    $ cat $(which notion-github)
    #!/usr/bin/env bash
 ${GOPATH}/bin/swish --identity-file ${HOME}/.ssh/github_notion_ed25519 --user kevinburkenotion 
```

*   **平衡多个客户:**如果你能做到这一点，或者做一些类似费用预付的事情，那就太好了。我发现转换环境真的很难，所以我一次和一个客户一起工作，并把它当作一份全职工作。做对你有用的事。

*   **回馈让你成功的工具**——我[每年拿出一部分收入](https://burke.services/2017/donations.html)来支持帮助我工作的软件工具——iterm 2、Vim、Go、Postgres、Node.js、Python、nginx，以及其他各种开源项目。你也应该考虑这样做。(如果你是一个开源维护者正在阅读这篇文章，请告诉我如何支付你！！)
