纳德拉的微软正在成功地交付和销售一种新的愿景，使其成为人工智能云计算新时代的先锋。

它在基础设施上投资了令人瞠目的金额，使其 Azure 云计算基础设施成为亚马逊网络服务在这个利润丰厚、增长迅速的市场领域的主要竞争对手。

它还在基于云的 Office 365 生产力软件和 Dynamics 365 企业产品方面取得了重大进展。

在 2014 年升任首席执行官后，塞特亚·纳德拉的两侧是他的两位前任比尔盖茨和史蒂夫鲍尔默。





与此同时，这家曾在反垄断审判中被拖上法庭的公司，正公开与大牌竞争对手合作，而且——或许最令人惊讶的是——它又在制造真正酷的科技产品了。

它的 Surface 电脑得到了好评，并被积极地与苹果的电脑相比较。它的[混合现实耳机 HoloLens](http://www.afr.com/technology/microsofts-hololens-now-augmenting-australias-reality-20161012-gs0i7w) 引起了市场的兴趣，也卖出了大量的 Xbox One 娱乐主机。

随着越来越多的科技观察家公开挠头说他们突然对微软比对苹果设备更感兴趣，这种转变是显著的。

纳德拉慷慨地将他的时代描绘成进化，而不是革命，并且对他的前任只有善意的话语。

## 寻找相关性

与此同时，在近年来狂欢之前经历过暴风雨的高管们形容他是一个几乎没有自我的领导人，他解放了这个星球上最有才华的一些技术人才，并指引他们走上了一条共同的道路。

纳德拉说:“在一个寿命甚至都不是目标的行业，我真的认为一切都是关于相关性和影响。”“我感觉很好，我们目前的技术发展势头良好，但这都是暂时的。我最关注的是我们是否忠于自己的目标。”

任何人只要在微软位于华盛顿州雷蒙德市的美国总部附近呆过一段时间，或者听过纳德拉在一次会议上的演讲，就完全知道那是什么目的。

一句厚颜无耻的夸大其词的公司口号令人生厌:“我们的使命是让地球上的每一个人和每一个组织都有能力实现更多。”

微软全球通信总经理蒂姆·奥布莱恩表示，史蒂夫·鲍尔默已经实施了一些文化变革，而这些变革在塞特亚·纳德拉的领导下加速了。





这肯定是冠冕堂皇的，但它似乎也是公司最高层真诚的意思。

在纳德拉的领导下，一切有价值的东西都必须来自微软内部的态度已经被摒弃。人们已经认识到，数百万人会希望使用其竞争对手制造的产品，它正在大力打造一条道路，作为更广泛的科技生态系统的主导部分。

该公司曾经是一个有围墙的花园，小心翼翼地保护着自己的技术，现在它与 Adobe 等主要科技公司建立了合作伙伴关系，致力于在苹果设备上提供服务，并且——对传统观念来说是惊人的——大力拥抱开源。

曾经被鲍尔默描述为癌症的 Linux 操作系统现在在其 Azure cloud 中得到支持，去年微软超过了脸书和谷歌等公司，成为开源代码在线交换 GitHub 的最大贡献者。

## 追随鲍尔默

纳德拉于 1992 年加入微软，并在成为首席执行官之前领导云计算和企业集团，他不是许多人预期的在鲍尔默之后进来解决问题的强硬局外人。

虽然他的性格明显不同于他喜欢社交的前任，但他断然拒绝了鲍尔默的个性在现代阻碍微软发展的观点。他说他欠比尔·盖茨和史蒂夫·鲍尔默很多，因为他们给了他额外的责任，并利用他对他们组织的了解以不同的方式经营。

“我是一个完美的内部人士。我是在微软长大的。纳德拉说:“我成年后的生活就是微软。“我是史蒂夫·鲍尔默和比尔·盖茨的微软的产物，我为此感到骄傲。我并不是想与众不同；当我成为首席执行官时，我希望能够根据我自己来塑造公司。"

谈到他成为首席执行官后发给员工的第一封电子邮件，纳德拉说他对公司的方向从一开始就很明确，并且一直保持不变。

即将离任的微软澳大利亚公司总经理皮普·马洛说，塞特亚·纳德拉非常平易近人，并且信守承诺。路易·杜维斯





他在信中警告员工，他们所在的行业不尊重传统，只尊重创新。他写道，他们的工作是确保微软在移动和云优先的世界中茁壮成长。

这向该组织提出了挑战，要求它走出舒适区，以其在输给苹果(Apple)和亚马逊(Amazon)等公司的领域的成就来衡量自己。纳德拉作为首席执行官的首次公开露面是期待已久的在苹果 iPad 上运行的 Office 应用程序的发布，这是一个象征性的举动，在新旧微软之间划出了一条清晰的界限。

虽然鲍尔默已经批准推出这些应用程序，但从外部来看，这被视为一个公司的突破，该公司此前通过冷落苹果来伤害自己的利益。

## 分享焦点

此次发布的方式也因其不同于以往由首席执行官主导的公告而引人注目。尽管这是纳德拉第一次公开露面，但他竭力确保这不是萨提亚秀。他退居二线，让聚光灯在很大程度上落在公司的后起之秀之一、Office 产品营销总经理朱莉娅·怀特(Julia White)身上，她因自己的演示获得了好评如潮。

怀特当时已经在该公司工作了 13 年，后来成为该公司至关重要的 Azure 云和安全部门的公司副总裁。

“我想象过和史蒂夫一起为 iPad 发布 Office，那会是什么样子。但是在那里，当萨提亚第一次公开露面时，我就像是‘这是一个全新的地方’，”怀特告诉老板。

“就我们展示的内容、我们展示的方式、活动的基调而言，这是一次非常不同的经历，一切都不同。”

怀特表示，纳德拉的领导极大地改变了微软的内部文化。她表示，他经常引用管理专家彼得德鲁克(Peter Drucker)的话，不断谈论要有一个增长的心态，他关于创造一种积极的文化来激励创新的言论是真实的。

“这是非常真实的，人们从外部感受到公司的变化只是因为它确实在发生，”怀特说。

“他是一位以价值观为基础的领导人，这是一种与以前非常不同的方式。作为一个领导团队，在萨提亚担任首席执行官的最初几个月，我们花了大量时间来弄清楚我们的使命是什么，定义我们的抱负是什么，现在它正在得到贯彻。”

纳德拉说，这种成长心态代表了鼓励员工在思考新想法时承担风险的愿望。

他说，他不担心如何应对竞争对手公司发布的热门产品，而是希望他的员工思考公司可以做些什么来开辟新的道路。

## 变化中的竞争文化

在盖茨和鲍尔默治下蓬勃发展的竞争文化需要被一种人们寻求更好地合作，而不是被视为山大王的文化所取代。

“我们必须从无所不知到无所不知，这对我很重要，”纳德拉说。“这不是关于新的教条，而是关于能够给团队成员喘息的空间，以便能够发挥他们的最佳水平，能够变得脆弱，并且不必摆出一副什么都知道的样子，而是要保持好奇和学习。”

微软内部竞争文化的核心是绩效管理系统，所有员工都是通过这个系统来评判的。

它就像一条高中评分曲线，被应用于整个公司的团队。团队中的一些人被列为表现最好的人，并得到相应的晋升或报酬，而其他人——不管他们有多好——相比之下被贴上了灾难的标签，并经常被解雇。

长期担任微软高管的蒂姆·奥布莱恩(Tim O'Brien)是全球通信部的总经理，他告诉*老板*说，谴责*虚荣* *公平*的文章抨击了微软内部的狗咬狗文化，表达了公司内部许多人的感受。

他说，鲍尔默听取了反馈意见，并致力于一个新的微软愿景，以前相互竞争的部门朝着一个共同的目标努力。

这包括采用基于团队合作而非个人成功的新绩效评估系统。

纳德拉的提升见证了微软推动的一个显著加速，奥布莱恩说，鲍尔默后来承认需要更换领导人，以允许员工真正推进变革。

“以前的系统有点像如果你在一群被狮子追赶的人中，你不需要最快，你只需要比最慢的人快，”奥布莱恩说。

## 展示变化

他表示，纳德拉和他的领导团队需要通过行动证明，以前的那种行为不再是公司内部所需要的。他说，随后，让一艘超级油轮改向这一看似不可能的任务以相对顺利的方式成为可能。

“我开始看到它消失了，功能障碍、争斗、琐碎和一些你在许多企业环境中看到的疯狂消失了，”奥布莱恩说。

“然后是协作、合作和信任，人们可以与其他团队分享他们的新想法，而不会影响他们的绩效评估和奖金。”

然而，事情并非一帆风顺，纳德拉需要做出一些重大而有争议的决定。

2015 年，他对鲍尔默卸任前收购的诺基亚手机业务进行了 76 亿美元的减记，从而取消了鲍尔默作为首席执行官的最后一项重大举措。

他认为，凭借广受好评但无利可图的设备成为 iPhone 的竞争对手是不值得的，此后他承诺，只有当手机业务能提供显著的新东西时，才会重返该行业。

“无论我们做什么，我们都会回到我们的使命，问自己，‘我们能做一些独特的、与众不同的、最重要的是对我们的客户有价值的事情吗？’”纳德拉说。

“我认为这是一个很好的过滤器，永远不要被公平竞争所驱动。”

## 交付成果

他承认，在不久的将来，还需要做出更多艰难的决定，但他很高兴自己赢得了员工、客户和市场的善意来做出这些决定。

总部位于旧金山的瑞银(UBS)分析师布伦特·希尔(Brent Thill)表示，纳德拉在任期早期转向重新定位云世界(不考虑设备)是一个至关重要的决定。尤其是因为他成功地为自己的变革注入了紧迫感，这吸引了一些最优秀、最聪明的人才将微软视为首选雇主。

当然，科技行业所有公司的典型风险依然存在。竞争对手总是可以突然出现，团队可能会出现代价高昂的错误，纳德拉必须警惕地确保当前的势头不会丧失。但就目前而言，积极的情绪是有形的。

“微软的股票在鲍尔默首席执行官时代保持平稳，我们相信这一变化是释放公司隐藏价值所必需的，”希尔说。

“纳德拉做出了一些艰难的决定，并在商业云上加倍下注，这有助于巩固与大型企业的关系。亚马逊是领导者，但微软现在被认为是紧随其后的第二名，很少有其他公司表现出竞争的规模或产品质量。”

在澳大利亚，微软准备进入自己的新时代，12 月份有一个令人惊讶的消息:长期在当地工作的董事总经理 Pip Marlow 将于 3 月份加入金融服务公司 Suncorp。

新的澳大利亚董事总经理史蒂文·沃拉尔将需要发展自己与纳德拉的关系，但马洛说他会发现这很容易建立联系。

“萨提亚的伟大之处在于可及性。她说:“当他说你可以打电话给他，或者他很乐意帮忙时，他是真心实意的。

“作为一名首席执行官，他有着独特的定位，能够解决系统中的任何摩擦，并认真对待这一责任。因此，每当我就某件事给他发短信时，他总是自己回复，然后坚持到底，做出改变。”

## 持续增长

与此同时，纳德拉热衷于强调，该公司收到的善意和积极的头条新闻只是暂时的重要。主要责任是确保微软长期走在正确的道路上。

面对苹果、亚马逊网络服务(Amazon Web Services)和谷歌(Google)等极其富有的竞争对手不可避免的竞争，如果微软想要利用自己的复兴，它需要让员工大军朝着同一个方向前进，专注于自己的使命。

尽管日程安排令人难以置信的紧张，而且需要跟上令人眼花缭乱的技术发展，纳德拉说，不断学习和做出改变的愿望给了他足够的精力来完成这项工作。

纳德拉说:“我一直在思考计算可以在人们的生活中做些什么，然后微软告诉我，哦，哇，这可以对社会和经济产生深远的影响。”“对我来说，最好的能量来源是走出去，接触客户，了解他们使用技术的新方式，感受我们所做工作的相关性。这些会议让车轮转动起来，这样就有足够的能量来运转。”
