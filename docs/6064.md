# 发送给所有错误- AVC

> 原文：<https://avc.com/2019/01/the-send-to-all-mistake/?utm_source=wanqu.co&utm_campaign=Wanqu+Daily&utm_medium=website>

我相信我以前写过这个，但我看到它如此频繁，我觉得有必要再写一次。

创业者、风险投资人和创业生态系统中的其他人经常发送电子邮件向我们公司的所有(或大多数)合伙人介绍一家公司。那封邮件是发给我们所有人的，而不是我们中的任何一个人。

结果是，我们没有一个人在介绍中感觉到所有权，尽管我们通常知道谁应该回复，但这可能会导致电子邮件在一段时间或更长时间内得不到回复。

另一方面，如果一封电子邮件发送给一个伙伴，并可能抄送给其他人，那么收件人会觉得有责任回复，并且邮件通常会得到回复。

我经常给忙碌的人发电子邮件。我学到的是，我需要直接写给他们，亲自写便条，这样很明显是我自己写的，然后抄送给某人(通常是他们的助理，但也经常是同事)，以确保他们看到它。

电子邮件是一种极具挑战性的媒介，在使用它的时候，你必须非常小心，以优化回复的机会。

向所有人发送电子邮件通常不是一种有效的优化形式。
