# 教学如何让你成为更好的程序员

> 原文：<http://venturebeat.com/2014/09/14/how-teaching-can-make-you-a-better-programmer/?utm_source=wanqu.co&utm_campaign=Wanqu+Daily&utm_medium=website>

在旧金山学习如何编程已经变得和学习如何做 DJ 或者吃 4 美元的烤面包一样时髦。从投资银行家到艺术家，每个人都想增强一项技能，在这个城市的初创企业推动的经济中，这项技能现在似乎与知道如何阅读一样必要。编码学校吹嘘毕业生的就业率高达 98%,起薪高达 11 万美元，学习如何编码的回报是立竿见影的。

但这对我们这些已经在科技行业的人来说意味着什么呢？我认为这是一个伟大的趋势，但在许多方面我很羡慕。对于现有的工程师来说，加深我们对代码的理解需要晚上和周末的自学，并花费许多时间努力为开源项目做出贡献。新生可以享受集中学习的乐趣，并享受立即付诸实践的好处。对于我们这些经验丰富的工程师来说，经常很难进入思考而不是实施的模式，如果没有适当的指导，更难知道你需要学习什么。

对于 web 开发人员来说，这个问题更加严重，因为新技术的数量每天都在增长，很难弄清楚它们是如何组合在一起的。

有句老话说，学习一件事的最好方法是把它教给别人。我不认为所有科目都是这样，但从我在大会上教授 web 开发的经验来看，这绝对是真的。

七周前，我踏上了为大会的[前端 Web 开发](https://generalassemb.ly/education/front-end-web-development)项目教 20 名成人学生的旅程。当我和他们一起结束我的学期时，他们带着一套全新的技能离开，这将使他们的职业生涯突飞猛进，我可以全心全意地说，这次经历使我作为一名工程师做出贡献的能力突飞猛进。

以下是我从学生那里学到的 3 条重要经验:

## 我开始意识到自己的偏见和假设

我最初是用 Java 开始学习计算机科学的。然后我发展到爪哇语，然后 SmallTalk。我在 Scheme 干过一段时间，但我的主要工作是面向对象编程语言。JavaScript 是一个完全不同的世界。我努力解释为什么你会想要一个匿名函数。毕竟，语法更加混乱，你可能会失去你的功能。最初，我只能想出一个借口，有时你只用一次函数。

在几乎不相信我的解释之后，我不得不研究并内化匿名函数是函数式编程之美的一部分。他们给你更多的灵活性。您可以将函数作为参数传递给其他函数，并以更具声明性的方式编写代码。通过教 JavaScript，我终于了解了 JavaScript 的强大之处，并且能够以更有效的方式使用它。

## 有些事情我们认为是理所当然的

我从概念上知道有一个调用栈，但我没有想太多。它的唯一目的是向我展示运行时错误可能源自何处。当我向我的学生展示一个例子，一个函数返回一个不同的函数，这个函数引用了初始函数的一个参数，他们模仿他们的头爆炸。

即使在对控制流和函数也是值进行了无数次解释之后，学生们还是不喜欢这些材料。最后，当我的助教在黑板上画了一个堆栈，并演示了函数在被调用时如何被放入调用堆栈，以及在执行完毕时如何弹出时，啊哈的时刻到来了。这是我很久以前就已经内化了，但很久没有想过的事情。

给我的学生上课让我思考事物实际上是如何工作的，以及当我运行一个程序时，我的计算机里发生了什么。每当我解释一个新概念时，我会立即得到反馈，了解我的解释是否有意义，并巩固我自己对这个概念的理解。

## 承认你不知道没关系

技术领域有很多自我。很多年前，我从麻省理工学院毕业，获得了计算机科学的数学学位。我 13 岁开始编程。显然，我对运行时环境的心理模型应该是可靠的。毕竟，我是站在全班同学面前，试图传播我丰富的知识的人。但是，成为一名实践者和真正掌握事物背后的理论是有很大区别的。当你教别人并帮助他们建立联系时，你需要在所有抽象背后的理论上有一个坚实的基础。

我花了整个晚上和下午重新学习编程背后的理论，这样我就可以向我的学生解释了。在我开始教学之前，我会掩饰自己浅薄的理解，并通过思考“当然，我可以实现一个网站——你还需要什么？”

所以你可能会想，更深入的理解能让你在编码中得到什么？你以前不是一个完美的功能练习者吗？有了新的巩固的理解，我现在可以更快(更准确)地通读源代码并构建一个心智模型。我能够与我的合作者进行更有成效、更有意义的对话。我以前从未考虑过为大型开源项目做贡献，因为我被吓倒了，不知道从哪里开始。现在，我发现自己喜欢阅读源代码，我想象在接下来的几个月里，我将第一次致力于 IronRouter(与 Meteor 相关的主要路由器项目，一个 JavaScript 框架)。

我目前正在为软件工程师开发一个教育平台，以提高他们的技术能力。其他每个领域都有培训项目来提升现有员工的水平，无论是高管 MBA 项目还是领导力培训。偶尔，开发人员可能会被派去参加一个会议，但深度学习通常不会在那里发生。我正在创建一个平台，让开发人员能够利用自己的时间，通过一种新的媒介来学习和理解新概念。

通过用兼职教学来补充我的产品开发，我已经能够将我的理论付诸实践，并且成果显著。毫不夸张地说，世界上最美好的感觉之一是，当你看着一个漂亮的网站，心想:“该死的，我真希望我也能做出来”，然后意识到这是一个从你这里学习 web 开发的学生做的。

所以出去吧。学点东西。教书。
