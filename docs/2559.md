# 互联网经济

我们生活在一个捆绑销售的时代。五大消费科技公司——谷歌、苹果、脸书、亚马逊和微软——已经远远超出了它们原有的产品线，进入了各种硬件、软件和服务领域，这些领域相互重叠，相互竞争。但是他们的收入和利润仍然严重依赖于他们无法控制的外部技术。形象化这些外部依赖性的一种方法是考虑典型的互联网会话的路径，从用户到一些产生收入的行为，然后(在某些情况下)再回到用户:



当评估一家互联网公司的战略地位(其利润的可防御能力)时，你需要考虑:1)公司如何产生收入和利润，2)整个循环，而不仅仅是公司拥有产品的层次。

例如，亚马逊是谷歌核心搜索业务的主要威胁，这似乎有悖常理。但你可以通过跟踪资金循环来了解这一点:谷歌收入的很大一部分[来自对可以在亚马逊上购买的东西的搜索查询，亚马逊上的购买体验(从最初的购买意图到消费/拆箱)明显好于你通过谷歌搜索找到的大多数非亚马逊电子商务网站上的购买体验。过了一段时间，购物者学会跳过谷歌，直接去亚马逊。](http://www.wordstream.com/articles/google-earnings)

把互联网经济圈想象成一个模型火车轨道。你前面的位置可以改变你周围的交通方向。位置后，你可以建立新的轨道绕过你。新技术的出现(起初看起来像玩具一样，没有威胁)创造了全新的轨迹，使得以前的轨迹变得过时。

在循环的每一层都有有趣的发展(还有许多更小的分支循环没有在上面的图表中描述)，但在任何给定的时间，某些层都是行业的闪点。最近最突出的战斗是在移动设备和操作系统之间。这场战斗似乎已经结束，Android 软件和 iOS 设备取得了胜利。未来可能的爆发点包括:

**物流自动化。**今天的物流网络由船只、飞机、卡车、仓库和人员组成。未来的网络将包括更多的自动化，从机器人仓库到自动驾驶汽车、卡车、无人机和[送货机器人](http://fortune.com/2016/04/06/dispatch-carry-delivery-robot/)。这种转变将分阶段发生，取决于特定商品和客户的经济状况，以及地理和监管因素。亚马逊当然在物流方面有巨大优势。谷歌曾多次尝试进入物流领域，但在 T4 收效甚微。按需拼车和送货初创公司可以在这方面发挥有趣的作用。物流层对电子商务至关重要，而电子商务反过来又对搜索货币化至关重要。随着电子商务继续从传统零售业手中夺取市场份额，亚马逊在物流领域的主导地位使其拥有非常强大的战略护城河。

**网络 vs 应用**。移动网络[正在](http://cdixon.org/2014/04/07/the-decline-of-the-mobile-web/) [衰落:用户在移动设备上花费的时间越来越多，花在应用程序上的时间也越来越多，而不是网络浏览器。苹果加入了这场战斗的应用程序方面(例如，允许 Safari 中的广告拦截器，鼓励应用程序在网站上安装](http://daringfireball.net/2014/04/rethinking_what_we_mean_by_mobile_web)[智能横幅](https://developer.apple.com/library/ios/documentation/AppleApplications/Reference/SafariWebContent/PromotingAppswithAppBanners/PromotingAppswithAppBanners.html))。脸书也站在了应用的一边(例如，鼓励出版商使用即时文章而不是网页浏览)。谷歌当然需要一个充满活力的网络来保持其搜索引擎的有用性，因此加入了这场战斗的网络方面(例如[惩罚有间隙应用广告的网站](http://techcrunch.com/2015/09/01/death-to-app-install-interstitials/)，开发减少网站加载时间的[技术](https://www.ampproject.org/))。现实的危险不是网络消失了，而是它被边缘化了，大部分可盈利的互联网活动发生在应用程序或其他界面，如语音或消息机器人。这种转变可能会对依赖非原生广告等旧商业模式的网络出版商产生重大影响，并可能使小型创业公司更难超越利基用例。

视频:从电视到移动设备。互联网公司押注视频消费将继续从电视转向移动设备。希望这不仅能创造引人注目的用户体验，还能打开目前花费在电视上的数百亿美元的广告渠道。

> “我认为视频是一个大趋势，几乎和手机一样大。”— [马克·扎克伯格](https://twitter.com/cdixon/status/706198805922902018)

过去十年，互联网赢得了收获购买意向的广告市场(过去出现在报纸和黄页上的广告)，其中大部分奖金流向了谷歌。未来十年的问题是，谁将赢得产生购买意愿的广告市场(迄今为止，赢家是脸书，其次是谷歌)。最有可能的是，这将取决于谁控制着视频广告的用户流量。今天，最大的视频平台是脸书和 YouTube，但预计视频将嵌入几乎每一项互联网服务，就像过去十年互联网如何从以文本为主过渡到以图像为主一样。

**语音:烘焙搜索进入 OS。Siri、Google Now 和 Alexa 等语音机器人将类似搜索的功能直接嵌入到操作系统中。今天，语音界面的质量还不足以取代大多数活动的视觉计算界面。然而，人工智能正在[快速提升](/software-is-eating-the-world/what-s-next-in-computing-e54b870b80cc#.kyn1qnbvj)。在不久的将来，语音机器人应该能够处理更加微妙和互动的对话。**

亚马逊的[愿景](https://developer.amazon.com/public/solutions/alexa/alexa-voice-service)在这里是最雄心勃勃的:在每一个可能的设备中嵌入语音服务，从而降低设备、操作系统和应用层的重要性(并非巧合，这些也是亚马逊最薄弱的层)。但所有大型科技公司都在语音和人工智能领域投入巨资。正如谷歌首席执行官桑德尔·皮帅最近[所说](https://googleblog.blogspot.com/2016/04/this-years-founders-letter.html):

> 下一个重大步骤将是“设备”这个概念逐渐消失。随着时间的推移，计算机本身——无论其外形如何——将成为帮助您度过一天的智能助手。我们将从移动优先转向人工智能优先的世界。

这将意味着人工智能接口——在大多数情况下意味着语音接口——可能成为互联网经济循环的主路由器，使许多其他层可以互换或无关紧要。语音在今天是一个新奇的事物，但是在科技领域，下一个大事件往往是从语音开始的。
