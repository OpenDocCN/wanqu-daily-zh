# 初创公司创始人应该问天使投资人和风险投资人的问题，但很少这样做

我总是很惊讶，也许只有 1/50 的创始人会恰当地面试他们的潜在天使投资人或风险投资人。这是一笔双向交易。



这里有一堆你应该问的问题。

*   在过去的六个月里，公司/天使投资人做了多少投资？
    *【如果是风投公司或天使网络】*在过去的 6 个月里，你自己领导过多少家公司？
*   …您目前正在投资吗？如果是，支票的大小是多少？
*   对于这种交易规模，我是否应该知道任何超出行业规范的条款、成本或怪癖？*例如，如果他们分摊法律费用，没有利息费用等*
*   最近的 3 次是什么时候，多少，他们是什么，他们占了百分之几
    你自己决定投资还是等其他投资者先投资再跟进？* *见脚注*
*   *【如果是风投公司或天使网络】*基金里谁做决策？
*   *【还有如果是风投公司或者天使网络】*有没有投资委员会，什么时候开会？
*   你的大部分交易流来自哪里？
*   你的支票金额是多少，你的投资组合中有多少是他们跟进的，你希望有多少是他们跟进的？
*   最后一笔投资从第一次接触到存入银行花了多长时间？
*   *【如果你通过票据融资】*你最近一次投资可转换票据是什么时候？
*   *【如果是风投公司或天使网络】*你认为谁是你最接近的竞争对手？
*   *【如果是天使】*作为天使投资人，你最佩服谁，为什么？
*   *【如果一个 VC】*你还真的很佩服其他哪家风投公司/哪些合伙人，为什么？
*   *【如果是风投公司或天使集团】*你公司里最了解这个行业的资深合伙人是谁？
*   在这一特定领域，您关注的指标是什么？**
*   ..他或她是否有权决定投资？
*   您的视点的下一步是什么？
*   在接下来的一年里，你*【和你的公司】*能为我们的业务增加什么价值？

这些应该在第一次见面或电话中问。

如果打电话的是风险投资公司的合伙人，利用这个初次电话把大部分注意力集中在他们身上，当然是在做了一个吸引人但简短的推销之后。这降低了合作伙伴为其市场数据库对您的业务进行数据挖掘的成功率，或者偶尔降低了他们在竞争对手投资之前进行数据挖掘的成功率。在通话结束时，询问您何时可以与他们之前提到的普通合伙人会面。

请在评论中添加我错过的任何内容，我确信我可能会错过。

*这一点很重要，因为“你需要一个首席投资者”，或者仅仅是一般的拖延，是许多投资者用来挫败创始人的手段，要么是因为他们真的是旅鼠，要么是因为他们没有礼貌说不。创始人必须记住，作为投资者等待往往没有什么坏处——每天都感觉像是在消除风险，当投资者可能错了，错过下一个谷歌时，为什么要拒绝创始人呢？..至少一些投资者是这么想的。事实上，创始人唯一能承受的真正压力是 FOMO 或价格上涨；这意味着你应该做好充分的准备，准备好所有的文件和资料，对你的目标投资者进行资格预审(并确保你有足够多的目标投资者！)不择手段地得到热情的介绍，然后尽可能在短时间内完成所有这些会议，并结束所有的旅鼠会议；动力决定一切。哦，还有再看看这个:[*【http://paulgraham.com/fr.html】*](http://paulgraham.com/fr.html)*
