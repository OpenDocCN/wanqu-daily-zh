# 关于信任、创始人争吵和让事情重回正轨的 6 个步骤

关系是建立在信任的基础上的，它们需要努力才能保持活力和健康。

在他的书《信任的边缘》中，David Horsager 概述了他认为在一段关系中产生信任的八大支柱。八大支柱是清晰、同情、性格、能力、承诺、联系、贡献和一致性。如果其中任何一个开始动摇，一段关系会很快开始分崩离析，并可能导致永久性的伤害。

在初创企业的早期阶段，创始人的角色有时一开始就不明确(对于主要的联合创始人来说更是如此)，难怪公司倒闭的许多原因都与创始人纠纷和联合创始人之间失去信任有关。

*注意:在这篇文章中，重点不是有明确 CEO 的公司是否比那些从一开始就共同努力的公司更好。早期的公司组织结构是一个非常不稳定的东西，可能完全是另一篇博客文章的主题。*

那么，你能主动做些什么呢？

**1。围绕一套共同的价值观创造一种文化，并以此为生活准则。**共同的价值观和建立在此基础上的强大品牌使决策变得更容易，因为你有一个人人都认同的“宪法”,你的选择以此为基础。

**2。明确定义角色和职责，这样事情就不会被遗漏**，并确保每个人都相信每个角色都交给了最有能力处理它的人。你可能会发现有助于概述这一过程的一个组织模型被称为[责任分配矩阵](http://en.wikipedia.org/wiki/Responsibility_assignment_matrix)(我更喜欢被称为 RASCI 的那个)，通过它你为一个项目分配人的责任，但你允许其他人被咨询、通知和支持，也许你也允许其他人负责。

**3。让每个人对自己有助于实现团队目标的部分负责**—有些人喜欢在一天开始前在团队面前召开全体会议，强调他们正在做的事情。

**4。经常沟通，即使有时很难做到。**虽然自然地，很多话题都源于与工作相关的事情，但很多压力也可能来自工作之外的领域。通过在一周内花些时间赶上进度，你有时可以了解人们反应的外部原因。尽管这并不能成为他们的借口，但有时对某个问题的认识可以引发一场关于如何共同应对该问题的有意义的讨论。

此外，请记住，随着公司的发展，需要新的工作类型，创始人个人的情况可能会发生变化，并会影响联合创始人角色所需的努力量。这种变化会进一步放大任何潜在的问题，所以要经常沟通，以确保你能快速解决这些问题。

**5。如果创始人之间的事情失败，要有一个计划。**如果创始人之间的失败导致你不得不分道扬镳，如果你有一个预先商定的方式来处理股权分割和知识产权所有权，对所有人来说都会更容易。作为起点，您可以随意使用[创始人合作协议](http://seedsummit.org/founders-collaboration-agreement/)来处理这些问题。

**6。如果事情变糟了，迅速采取行动**恢复信任或必要时进行内部重组。

如果信任被破坏，两位联合创始人之间的关系确实走下坡路，你需要调整意图。扪心自问，你是否愿意接受以下结果。

**选项 1**–完成和解，重建创始人之间的信任

重建信任并不容易。承认错误和失败，并带着不再犯错的严肃态度为此道歉，这需要勇气。

因为重建信任需要时间，所以双方都必须有意愿和精力来解决导致信任被破坏的问题，然后想出可行的解决方案(从上面概述的六个行动开始)来避免这些问题再次发生。然而，如果其中一方被冒犯到无法挽回，那么最好考虑下面的替代方案。

**选项 2**—一位创始人担任领导职务，另一位(们)担任更具操作性的角色
**选项 3**—两位创始人共同同意给公司带来一位新领导
**选项 4**—冒犯(或被冒犯)的创始人离开公司
**选项 5**—两位创始人共同同意关闭公司

当然，最后一个选择是最不受欢迎的，但至少在讨论开始时，各方都清楚最后的选择是什么。我还从上面的列表中省略了其他股东的任何形式的法律干预行动，这不是因为它们在法律上是不可能的(取决于你的股东协议的规定)，而是因为除非创始人在董事会，否则最好不要这样做。

总之，保持信任需要努力，但通过尽早投资，随着时间的推移，你可以保证所有创始方之间的工作关系更加有效和协作。
