# 后悔金手铐马修里昂

> 原文：<http://lyonheart.us/articles/regretting-the-golden-handcuffs/?utm_source=wanqu.co&utm_campaign=Wanqu+Daily&utm_medium=website>





这份工作令人振奋。这是我第一次真正体验到只做产品而不是做顾问，这是我第一次参与到游戏中。它提供了一个我不会感到被遮蔽的空间，并给了我一个关于我能力的新视角:我迎接了许多技术挑战，在塑造产品设计方面发挥了重要作用，并帮助建立了一个伟大的团队。我对我的工作充满热情。

事后看来，问题显而易见。我们用一家供应商的尖端技术开发了一款产品，寄希望于平台*协同*，吸引早期用户，免费给他们太多东西。供应商放弃了他们的技术，我们慷慨的免费服务吸引了那些很难转化为付费客户的人。工程团队花了我们大部分时间与滥用服务作斗争，随着产品愿景的动摇，人们离开去了其他公司。当创始人宣布我们正在寻找买家以继续运营时，我并不感到惊讶。

不久后我写了一封辞职信，但从未上交。我开始感到筋疲力尽:对工作不感兴趣，对公司的情况充满怨恨。几个朋友鼓励我离开，休息几个月，清醒一下头脑，找份新工作。我妻子和我正在买房子，我们把它搁置了。我真的不想留下来，但我是最后一个留下来的原始员工，我在创造我们自己的大部分知识产权方面发挥了很大作用。我很希望有人会想把我留在身边。

有几个逛街的人。工程团队的大多数人离开了已经变得压抑、高压力的环境，去创办自己的公司或自由职业者。我在那里并不开心，但我没有离开，创始人鼓励我去度假。当我回来的时候，我发现我们有一个感兴趣的买家。

尽职调查后大约一周，我们得到了消息。他们想继续进行收购。买家想把我们的产品整合到他们的战略中，哇，T2，看看这种协同效应！最棒的是，我们不必公开承认失败。我们有一个成功的*故事*。我的普通股期权一文不值，但不知何故，它们会转化为挽留提议——众所周知的将我留在身边的金手铐。





科技行业认为他们已经想通了一切。我们有这些系统告诉我们人们是如何对某事做出贡献的，这是一个 T2 式的精英管理，所以自然每个人都得到了公平的报酬——不要介意 T4 关于 T6 谈论报酬的巨大禁忌，我们说过这已经是一个精英管理了吗？[不要理会窗帘后面的那个人](https://www.youtube.com/watch?v=YWyCCJ6B2WE)，开心点，我们付钱给你是为了[做你喜欢做的事情](http://www.businessinsider.com/steve-jobs-destroyed-the-follow-your-passion-myth-2015-3?op=1)！虽然在公司外谈论你的薪酬可能是受保护的商业秘密，但在同事间谈论它是受联邦保护的言论。不要介意你的非正式内部结构对外人来说是不可见的，对外人来说唯一重要的是组织结构图。

这是拐弯抹角地说我的提议不是我所期望的。当然不是——我只能继续讲述成功的故事。

不要误会我的意思——工资是一个不错的提升——但我已经在低于市场水平的情况下工作了几年。正如一位朋友所说，留任奖金是一笔不错的遣散费。新的雇佣合同非常苛刻，以至于我雇佣了一名专门研究雇佣法的律师。我帮助建立的团队对他们的提议非常不满，以至于我们都聚在一起坐下来，开诚布公地讨论其他人认为我们的价值:不多。

律师是我花过的最好的 600 美元之一。从那以后，我在谈判中运用了我从中学到的东西。

</aside>

其他人都要养家糊口。如果不是这样，我会把他们都赶走，去找更好的工作。我感到内疚，因为我鼓励他们中的一些人呆了这么久。我们有一种预感，在我们第一次支付之前的三四个月的整合期将会是痛苦的。没有人对签约有好的感觉，但是当你不得不签约或者失去工作的时候，当你没有能力离开的时候，你真的没有任何谈判的力量。

我也知道，无论我们为集成制定了什么目标，如果我能帮助塑造它们，保护它们免受范围蔓延的影响，并使事情保持在正轨上，就有更好的机会实现它们。我想我可以做些好事，帮助每个人得到更好的待遇。

我的工程师的思想寻求优化谈判过程，这是一个庞大的官僚机构像毒药一样回避的东西。在他们眼里，集体谈判就是共产主义。我与我的律师合作，达成了一份对他们的律师和我本人都友好的合同，并与我的团队分享了这份合同。我去掉了最讨厌的部分——一项严重到两年内禁止使用电脑的竞业禁止，以及对我先前知识产权的保护。我无法为我正在进行的技术副业项目获得保护——他们认为这些是兼职，但我认为改进就是改进，我可以花几个月时间学习西班牙语和吉他。所以我说:如果我的同事都得到同样的待遇，我也会签字。

各种各样的人暗中威胁要小心，因为技术是一个小镇，我不应该烧任何桥梁，所以我知道这个策略是有效的。我指出，我所支持的一些人真的可以走到街对面，获得更好的补偿，甚至包括一整年的留任奖金，但他们对此置若罔闻。他们试图用我将产品带到企业下一个层次的机会的故事来赢得我的个人信任。这是一场奇怪的凝视比赛，看谁先眨眼。

剩下的团队签了名，虽然对事情并不完全满意，但觉得他们可以在需要几个月才能完全融入的新环境中生活。至少，这将为他们争取时间找到其他工作，同时继续领薪水，为家人买保险。

至于我自己，我知道我不能签。我能感觉到。留下来对我来说不是正确的事情。我打算拒绝这个提议，离开，去找一些承包工作。金钱在短期内可能没有那么好，但至少我可以远离环境，可以把一切抛在脑后，按照自己的节奏前进。

这是故事应该结束的地方，也是我后悔的地方:我还是签了。

一些甜言蜜语，合同项目的最后让步，以及一次恰到好处的加薪，成功地打消了我离开的念头。我算了一下:离整合支出还有三到四个月。我会在我的简历上得到一份体面的工作。我会帮助我团队的其他成员在集成目标的范围内前进。在我没有动力的时候，这可能比出去找新工作更容易赚钱。





我努力过好自己的生活，让自己没有遗憾。说我犯了错误并从中吸取了教训。以我的知识和感觉，我做了当时我能做的最好的决定。这个故事我做不到。我知道我不应该签约，我深深地感受到这一点，这引发了身体上的反应，但我不顾一切地去做了。

在科技行业有一些关于倦怠的讨论，但没有应有的多。我已经记不清有多少朋友和熟人在离开一份特别紧张的工作后，会休假几个月，只是为了清醒一下头脑。[倦怠是一种鲜为人知的疾病，类似于临床抑郁症](https://en.wikipedia.org/wiki/Burnout_(psychology))，职业压力助长了它，整合收购的压力是一场盛宴。

我团队中一些薪酬较低的成员在收购消息公布后立即利用这段时间找工作。他们中的一个真的穿过街道去获得更好的整体补偿。我试着低下头，把注意力集中在目标、随后的回报和我随后的自由上。

在签署前大约一个月的时候，事情看起来好像要崩溃了，好像我们要错过目标了。无论如何，我渴望知道真相。前一周，我们已经购买了出国旅行数月的机票，现在正指望实现这个目标来资助这次旅行。我开始安排一些承包工作，屏住呼吸，但由于许多重要人物在整合的成功中有利害关系，这是没有必要的。多亏了一些挥手和一些顺利交谈，我们回到了目标。

我没有足够的精力去追求我的副业，去利用来之不易的权利。当我不工作的时候，我根本不想靠近电脑。我去远足和拍照，我画画，我酿造啤酒，我学习基本的西班牙语。我写了一些关于我在初创公司经历的糟糕小说，还有一封我从未寄出的长达 8 页的辞职信，来证明我为什么不开心。

通过一系列小奇迹，我们在目标上得到了认可。虽然这是一种解脱，但这并不是结束——从技术上讲，根据合同，他们必须向我们付款。在薪资和保险等基本问题出现后，没有人愿意冒险去争取他们应得的东西。最好等到钱到了再发辞职信。人们对这是下一个工资周期还是下一个工资周期有些困惑，他们如何扣缴税款，似乎没有人知道任何事情。我们所能做的就是等待。

我永远不会忘记我们签字后的那个发工资日的早晨。我醒得有点早，对我的银行存款余额感到惊喜。*事情发生了*。我在我的团队的后台发布了一个笑脸，还煮了咖啡。我写了一封两句话的辞职信，走进办公室，享受着可能是我一年中第一个真正快乐的日子，然后辞职了。

在我的一生中，只有另外一次，当我们最终关闭我们的第一个家时，我感到了同样的宽慰。

</aside>

自从交易完成后，我还没有和创始人说过话，他说了一些没有感情或上下文的话。传达事实的话语。因为每个人都期望他说些什么。我走出门外，离开了一份工作，这份工作提供了我不知道自己有能力应对的挑战，是我人生中最紧张时期之一的背景，是我愿意立刻再次共事的人的聚会场所。我剩下的大部分同事都已经有了其他工作，并在此后不久发出了通知。

在我回家的路上，我收到了一条匿名短信，里面有一张投资意向书的照片。当我得知那些为我留下来尽最大努力的人正在像强盗一样亲热时，我感到很受打击。我差点把手机扔进河里。

接下来的一周，我们要离开这个国家，把所有的东西都储存起来。我们离开了我成年后住过最久的公寓，那是我们买房子前的临时妥协，然后我们登上了飞机。多年来，我第一次没有被别人的目标所拖累。





这也不是故事的结尾。我从收购中赚取发薪日的时间结束了，但我仍然要为接下来的七个月付出压力和疲惫的代价。这一点很重要，值得重复: [*倦怠是一种鲜为人知的疾病，其症状类似于临床抑郁症*](https://en.wikipedia.org/wiki/Burnout_(psychology)) 。继续做一些导致问题的事情只会使问题复杂化。

出国，从身体上把自己从半陌生的环境中分离出来，学习一种新的语言和新的习俗，这对康复的开始是有好处的。这让我有了其他的关注点，因为我根本没有精力去追求一些技术创意，而几个月前，我想把这些创意变成产品。在我期望找到动力的地方，却有一种空虚、痛苦，一种一切都是徒劳的感觉。

所以我学了西班牙语。我深入研究了拍摄照片的过程。我黑进了一个愚蠢的 DSL。我没有体重秤，但是在没有压力和吃得更好之间，我的腰带降了三级。我在海滩上呆了一个月，结交了新朋友，并从肉店老板那里得知，我花四美元一磅买的美味牛排来自路边吃草的奶牛。当我没有精力去编码的时候，几个月后我[任组长](/articles/making-time-for-quality)一些[的心思](/articles/refining-software)在[产品开发](https://www.youtube.com/watch?v=bEZcodengwk)上。

我们回国后，我有一点旧病复发，既回到美国，又与科技界人士交谈，并在白板上做代码采访。压力使我和妻子的关系变得紧张，我永远感激她给了我疗伤的空间。我花了一些时间思考我想从工作中得到什么，意识到我不再想与我有专业经验的技术团队一起工作，我想做一些有意义的事情。

我不完全确定我是从什么时候开始不再感到倦怠的。大约在这个故事开始后的 15 个月，我在一个项目中获得了一些承包工作，我对这个项目非常感兴趣。我不再感到空虚，痛苦，我知道我变得更好了。

我过于狭隘地关注支出，以至于没有看到更广泛的情感成本。最后还是不值得。我最好不要签约，不要给自己增加额外的压力，不要让问题变得更糟。我应该听从自己的直觉拒绝的。





我从朋友和熟人那里听说过类似的故事，关于一份需要太多个人牺牲的工作的持久影响。他们中的许多人需要几个月的时间来恢复，或者转向新的、同样不健康的反弹工作，以逃离他们所处的有毒环境。最近《T2 纽约时报》关于亚马逊文化的文章深深打动了我，因为它描绘了那些走向病态极端的环境的潜在理想。

我不是唯一注意到的人。[人们在](https://medium.com/life-learning/work-hard-live-well-ead679cb506d)谈论[找到平衡](https://medium.com/@aq/finding-balance-7388ce748051)，[讲述他们的故事](https://medium.com/@jcheiffetz/i-had-a-baby-and-cancer-when-i-worked-at-amazon-this-is-my-story-9eba5eef2976)。《时代周刊》描述的亚马逊衡量标准驱动的绩效文化的可悲之处在于,[研究](http://ericdodds.com/research-and-the-realities-of-time-worked/)显示[过度工作的人表现更差](https://hbr.org/2015/08/the-research-is-clear-long-hours-backfire-for-people-and-for-companies)。这些环境的强度和项目的范围帮助人们感觉他们是某件大事的一部分，某件值得为之做出牺牲的事情。我们称这种感觉为激情。它会让人上瘾，会影响判断，但它不一定是完全的，也不应该把人推向伤害。小心激情和那些会利用它来对付你的人。
