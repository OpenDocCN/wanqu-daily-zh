# 让我们停止诽谤我们公司的前任吧

最近有一个硅谷趋势一直困扰着我。打开 Twitter，你会看到数百个技术 bios，上面写着:

【公司】工程师。前苹果，前谷歌，前麻省理工学院

作为一名营销人员，我理解这种吸引力。协会是增加可信度和权威性的有力工具。但是这种自我认同存在问题和危险。

首先，它可能会被误解。

不久前,《哈佛深红报》发表了一篇文章，讲述了哈佛校友在公开自己背景时面临的困难(被称为“投下氢弹”)。如果他们提起自己的母校，他们就有可能把自己人格的其余部分与哈佛的所有联想混淆起来。但如果他们故意隐瞒学校被抓，那就显得居高临下了。所以校友们踮着脚尖走在两者之间；适当的时候提及他们的背景，但不要给人留下第一印象。

这似乎也是精英技术 bios 的一个好方法。这可能对这个人有利，因为委婉地低调表现出自信和谦逊，而高调有时会表现出不安全感。

然而，除了社交礼仪之外，还有两个更重要的原因让我们应该停止宣传血统的做法。

第一个与决策有关。如果有人在大公司工作过，那么他们可能容易受到过度自信偏见的影响。过分关注过去的联想和胜利会引发自动思维泡沫:“我当然是对的。我以前是对的，我是前某某人。”这可能会导致对反对观点不屑一顾或变得自满。

《权力的游戏》粉丝可以看到丹妮莉丝·暴风女的相似之处，她经常这样描述自己:“坦格利安家族的第一个名字，未燃烧者，安达尔人和第一人的女王，大草海的卡丽熙，锁链的破坏者，龙的母亲。”虽然这些头衔中的每一个都是她应得的，但她沉迷于这些头衔的习惯让她对许多危险视而不见。

我们应该停止这种做法的第二个原因是文化。

科技行业存在多样性问题，过多谈论前任只会给公司添砖加瓦。认同前任会让我们偏向其他前任，我们会和他们谈得更多，给他们更多指导，并最终雇佣他们。如果我们变得有影响力，那么崇拜我们的人可能会看到我们的优秀道路，并认为他们永远不会成功——他们去了错误的学校或有了错误的第一份工作。那就可惜了，因为创新思维总是来源于发散。

“前任”现象渗透到科技生态系统的其他领域。每周，我都会阅读一篇新闻报道或 Kickstarter 上一个关于苹果或微软前员工创办的公司或产品的推介。我想我在某个地方见过一件前谷歌的帽衫。

对于报道故事的作者或制作销售文案的企业家来说，它为初创企业增加了一个熟悉的纽带，但听起来也很空洞。当然，还有更深入、更有趣的产品或创始人角度有待发掘，尤其是在这个大型科技公司雇佣数百万员工的时代？

我们都知道有些公司只雇佣某些前任和校友。

整件事的讽刺之处在于前任总是有原因的。人们离开公司是因为那些公司不再满足他们的需求。从这个角度来看，一份前任名单变得不像是庆祝胜利，更像是一首回忆失败恋情的蓝调歌曲。

回到我们朋友的推特上:

【公司】工程师。前苹果，前谷歌，前麻省理工学院

让我们隐藏血统，忘记前任，让世界更深入地了解这个人的生活，这样他们就不会显得一里深一寸宽。

【公司】工程师。鸡尾酒爱好者，骄傲的柯基犬主人，巨人队球迷，妈妈，前素食主义者。

听起来像是我想一起出去玩的人！
