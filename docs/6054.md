思想实验:亚马逊将如何进入风险投资业务？

#### **使用来自 AWS 的数据为投资决策提供信息**

亚马逊可以利用来自 AWS(亚马逊网络服务)的专有数据。亚马逊的优势在于，大多数最优秀的科技初创企业都是建立在它的服务之上的。亚马逊有很多关于这些公司花费多少，他们使用什么服务，他们使用什么技术等等的信息。

AWS 的数据可能极具预测性，并为亚马逊提供公司快速增长或达到拐点的早期迹象。它可以使用这些数据作为对一家公司更好的尽职调查……例如，这些数据可以帮助确定哪些公司声称他们拥有“人工智能”是真实的，哪些只是营销。

#### 亚马逊拥有真正的投资优势。

利用这些数据投资上市公司可能不合法，因为这可能被视为内幕信息。但是将它用于私人公司是亚马逊可以做的事情。

当然，亚马逊担心的是他们的一些 AWS 客户会变得疯狂并转向 Azure(这是进入风投行业的最大风险)…但这是可以管理的。亚马逊可以只使用 AWS 法案中的信息(而不必看到任何真正的商业秘密信息)来初步选择他们可能希望重点投资的公司。然后，如果公司同意，Amazon VC 团队可以查看服务器日志等。

#### **这就引出了第二件事:“你的利润就是我的机会”**

亚马逊可以像在所有其他业务中一样赢得风投交易:价格和便利。

在价格上，亚马逊可以提供比传统投资者更好的条款，因为传统投资者需要支付高额管理费和运费。亚马逊不需要这么做，也不需要、不想或者能够(因为利益冲突)占据董事会席位。因此，它将有更多的杠杆作用…尤其是在数据越来越重要的后期融资阶段。

虽然亚马逊可能会尝试收购股权，但它也可以只专注于债务(这是一种它已经熟悉的产品——见下文)。

风险投资公司扣除费用(管理费和套利)后的回报历来很低。但如果亚马逊真的专注于投资，它可能会获得极高的实际回报。

#### **极度便利:获得扩张资本的最简单方式**

想象一下，登录 AWS，看到一份条款清单。只要点击这里，同意这些简单的条款，我们将电汇 1000 万美元给你。不到 5 分钟。是的，这看起来很疯狂。但这是可能的。

如今，亚马逊向其商家提供贷款(这是一项非常好的业务)。Square 还向其商户提供贷款。亚马逊和 Square 都利用其专有数据向他们确信会偿还的企业发放贷款。这些贷款表现非常好。Square Capital 被誉为一项了不起的生意。他们可以这样做，因为他们有独特的数据…他们可以给出一个有吸引力的价格(较低的利率)，并使它更方便(如一键获取的能力)。亚马逊没有理由不能向 AWS 客户提供贷款。

亚马逊可以创造一种产品，只需点击一下鼠标，就能以极具吸引力的条款为公司提供资金。作为现金流的额外奖励，亚马逊甚至不需要给这些公司汇钱。相反，它可以给公司 AWS 积分。如果一家公司每月在 AWS 上花费 50 万美元，并相信未来会继续这样做(许多技术公司都是如此)，那么获得 1000 万美元的 AWS 信用与获得 1000 万美元的硬美元几乎是一回事。

#### **其他成功的科技公司创办风险投资公司的例子**

亚马逊不会是第一家成功创办风险投资公司的大型科技公司。谷歌和 Salesforce 都有非常大(而且，我听说，非常成功)的风险投资(Salesforce 有数十亿美元，谷歌有数百亿美元)。苹果没有风投公司(尽管它也有巨大的数据优势)。尽管运营一家风投公司似乎有悖于苹果的精神，但亚马逊喜欢挑战新行业，并利用其专有数据为自己谋利。

#### **债务可能比股权更容易产生初始产品**

AWS 资助私营公司的第一批受害者不太可能是传统的风险投资公司。更有可能是风险债务公司。这可能会严重伤害一些传统的债务提供者(如西方技术合作伙伴)和一些新的积极参与者(如 TPG、大型对冲基金和其他新的贷款人)。

#### **预测:亚马逊不会创办风投公司**

如果亚马逊不那么雄心勃勃，它会进入风险投资业务领域。亚马逊不成立风险投资部门的唯一原因正是它可以成立的原因:因为风险投资行业规模很小，收益虽然数十亿美元，但可能不值得亚马逊付出努力。

另外:亚马逊可能担心这会损害他们的 AWS 业务。当然，许多人对我的 Twitter 试用气球的回应都相信这一点:

总结:虽然亚马逊不太可能挑战现有的风险资本家和风险贷款人，但这是一个非常有趣的思想实验，看看它如何挑战。
