# 机器学习竞赛实际上是一场数据竞赛

希望让人工智能成为差异化优势的组织需要利用替代数据集——他们可能不得不自己创建的数据集。

[Megan Beck and Barry Libert](#article-authors) <time class="article-header__date pub-date-dimension" datetime="2018-12-14T11:27:00-05:00" pubdate=""><abbr class="published updated" title="2018-12-14T11:27:00-05:00">December 14, 2018</abbr></time> Reading Time: 6 min 

机器学习——或者人工智能，如果你喜欢的话——已经成为一种商品。竞相同时定义和实现机器学习的公司惊讶地发现，实现用于让机器智能处理数据集或问题的算法是容易的部分。从谷歌的 TensorFlow 开源机器学习框架到微软的 Azure Machine Learning 和亚马逊的 SageMaker，有一批强大的即插即用解决方案可以轻松完成繁重的编程工作。

然而，没有被商品化的是数据。相反，数据正在成为机器学习竞赛中的关键区分因素。这是因为好的数据并不常见。

### 有用的数据:既有价值又稀有

数据正在成为一个差异化因素，因为许多公司没有他们需要的数据。虽然几十年来，公司一直使用公认的会计原则以系统的方式衡量自己，但这种衡量长期以来一直专注于实物和金融资产——事物和金钱。2013 年，诺贝尔奖甚至颁发给了资本资产定价，强化了这些早已确立的优先事项。

但是今天最有价值的公司从事软件和网络交易，而不仅仅是实物商品和资本资产。在过去的 40 年里，资产重心已经完全翻转，从 1975 年 [83%的有形资产主导市场到 2015 年](http://www.oceantomo.com/intangible-asset-market-value-study/)84%的无形资产。今天的企业巨头不再制造咖啡壶和销售洗衣机，而是提供应用程序和连接人们。这种转变在我们衡量的东西和真正推动价值的东西之间造成了严重的不匹配。

结果是有用的数据很少。市场价值和账面价值之间的差距越来越大。由于这种差距，各公司正在竞相将机器学习应用于重要的商业决策，甚至[取代他们一些昂贵的顾问](https://hbr.org/2018/01/robo-advisers-are-coming-to-consulting-and-corporate-strategy)，却意识到他们需要的数据甚至还不存在。本质上，新的人工智能系统被要求在同样的旧材料上应用新的技术。

就像人一样，机器学习系统不会在任何主题上变得智能，直到它被教会。为了变得聪明，机器需要比人类多得多的数据——尽管，当然，它们读取数据的速度要快得多。
