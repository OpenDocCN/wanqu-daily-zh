# “WordPress 不仅仅是一个博客平台”



自从 WordPress 第一次出现以来，我就一直在用它来创建博客。这些年来，我为许多不同类型的人设计和建立了博客。个人、音乐家、艺术家、爱好者、小公司、大公司，甚至财富 500 强公司。

然而，我使用 WordPress 的时间越长，我在创建博客时遇到的问题就越多。你看，WordPress 变了。最初是一个不起眼的博客平台，旨在为大众提供数字出版，现在已经发展成为一个成熟的网站 CMS。

这本身并不是一件坏事。我仍然认为 WordPress 是一个了不起的平台，事实上我认为它是最好的 CMS。但这不是我写博客想要的。到处都有太多的东西，太多的混乱，太多(太多)的选项阻碍了我真正想做的事情:发布内容。

WordPress 是免费的，易于使用的，开源的，可扩展的，可主题化的，自我托管的，对所有人开放的。这是它目前的优势所在，而且与 Tumblr、ExpressionEngine、Svbtle、Typepad 或 Squarespace 等平台相比，它将永远胜出。但它也在某些方面失败了。

WordPress 不再有目的了。它不再是**代表**任何特定的东西，而是代表你可以想象用它做的任何事情。它的内部开发方式也发生了巨大的变化，从为发布平台创建和发展功能，到为人们创建和发展自己的平台。自定义帖子类型，自定义分类，内嵌主题编辑器，高级菜单系统，多站点，多用户，多一切。使用一些主题和插件，WordPress 可以变成任何东西，从一个 bug 追踪器，到一个电子商务商店，再到一个全面的社交网络。

## 根据任何人的定义，WordPress 不再是“仅仅是一个博客平台”

Introducing... Just a blogging platform



Ghost 是我对 WordPress-lite fork 的理想主义和虚构的概念。它有一个目的，也只有一个目的:为大众提供数字出版。让有比我更重要的想法和事情要说的人能够在网上发布内容——快速、轻松、美观、高效。Ghost 是关于打破 WordPress 最初所做的同样的障碍。

就像一个幽灵作家，幽灵，是你的幽灵出版商。它以匿名方式完成这项艰巨的工作，将你的内容放到网上，这样你就可以专注于最重要的事情:你的想法。你的内容。





### 您可能希望使用的仪表板

作为内容发布者，我们花了多少时间在浏览器标签之间穿梭，查看我们博客的多种营销和分析渠道？太多了。

Ghost 会有一个你确实想使用的仪表板。一个你每次登录时都想看到的关于你的博客的所有相关内容。你的流量水平，你的热门内容，你的社交媒体数据，你的作者的表现。完全定制它来满足您的需求，拖放部件到您自己的优先级别-或建立自己的。把它想象成你的 iPhone 主屏幕。





### 通过实际**看到**帖子来管理帖子

十多年来，电子邮件客户一直使用拆分视图，因为这很有意义。

Ghost 在管理您的内容时会有一个拆分视图，这样您就可以从管理面板浏览您的内容，而不必编辑它。这个列表会告诉你你所关心的事情，快速而简单。文章的标题、类别、发表日期和阅读次数。没有另一个页面加载，你可以在决定下一步做什么之前预览每一篇文章。对于有多个作者的博客来说，这是一大进步。





### 一个新的书写屏幕实际上对写博客有意义

![Ghost Editor Screen](../Images/1a50dfe2a1832b64691ac42bc86bb301.png)

有两件事破坏了我写作时的思路。首先是每 20 秒钟就要从键盘切换到鼠标，以便将基本格式应用到我的想法中。第二，必须不厌其烦地手工编写 HTML 来达到同样的效果。两者都没有意义。两者都是仅有的选择。

Ghost 将会有一个新的系统，再次使用拆分视图。左边是降价，右边是预览。您专注于编写，在编写过程中应用基本格式，并在右侧看到内容将如何出现在您的站点上的实时更新预览。没有更多的点击，没有更多的括号和斜线。

到达这样一个点，你知道你以后会想要插入一个图像，但是不想现在就停止写作？没问题。只需在新段落中键入“(图像)”。Ghost 会将图像上传器加载到您的预览区域。稍后在图像中拖放。



## 还有什么**还有**？

#### 选择更少

![options](../Images/b1eaefa4d200c332039378ef77f2c3e8.png)

WordPress 的座右铭是“决定，而不是选项”——然而仍然有太多的选项，太多的设置，太多的事情，你在管理用户界面上有不必要的控制。像管理配色方案、quickpress、press this、post-via-email、远程发布、内嵌主题编辑、媒体编辑等等。很多人都没用过的东西。幽灵会摆脱这一切。

#### 没有评论

![comments](../Images/a5312f20edf75d3b6393d0a0f3ea72b7.png)

Ghost 没有本地评论系统。当然，许多人仍然想在他们的博客上发表评论，这不是问题。

Ghost 将与以构建复杂评论平台为生的公司进行简单的整合。比如 Disqus。通过节省时间和精力，我们可以专注于对 Ghost 来说最重要的事情:出版。

#### 没有学习曲线

![code](../Images/d3bd2b5d066d6aca90582cd6fbc18aea.png)

想为 Ghost 开发，但你是 WordPress 开发者？酷毙了。这将是一个 WordPress 分叉，所以它将使用完全相同的主题和插件结构。你现有的知识和经验是可以转移和使用的——马上。

毫无疑问，随着时间的推移，这种情况会发生变化，但核心将保持不变。

#### 合理的兼容性

![compatibility](../Images/7db2ecbf3be7291d66db4fb15eb4e9e3.png)

WordPress 在向后兼容性方面是惊人的，也是可怕的。你可以在当前版本的 WordPress 上安装 5 年没有更新的插件和主题，大多数时候，它们仍然可以工作。这对用户来说相当不可思议，说实话，这没有任何意义。过于关注向后兼容性会阻碍未来的发展。它会引起肿胀。

Ghost 将与主要版本有分界点，允许核心开发人员从代码库中删除旧代码，并发展平台以允许其改进。没人指望为 OSX 10.4 版老虎编写的应用能在 OSX 10.8 版山狮上运行。

幽灵也一样。

#### 需要更少的插件

![plugins](../Images/064bd83676537fd744db1637720b8416.png)

WordPress 的最大优势可以说是插件，但是如果你去掉所有非博客功能的插件，你会剩下一套相当一致的插件，每个人都用它们来实现同样的事情:搜索引擎优化、相关帖子、谷歌分析、Twitter/脸书/Instagram 集成、内嵌拼写和语法检查、联系人表单、编号分页、备份。这些东西不需要是插件。数以百万计的下载已经证明这些东西应该是核心平台本身的一部分。我们不会让你每次安装都下载 Joost de Valk 的 WordPress SEO 插件——我们会让 Joost 把他的代码直接提交给 core。我们会把证明对其代码有压倒性需求的第三方开发者视为合作伙伴。简单。

#### 没有法律的公司部门

![free](../Images/4e270318ef3e084654218dccf0d71ac1.png)

人们经常说 WordPress 在演讲中是免费的，而不是在啤酒中。

Ghost 在 Mozilla 中是免费的，而在 Automattic 中不是。Ghost 将 100%作为非盈利组织运营。

做出的每个决定都是为了改进软件，而不是底线。



### 还有一件事:**开放**发展

Ghost 将被公开开发，它将鼓励贡献。到目前为止，开源项目通常有一个令人难以置信的高门槛，这是如此复杂和令人费解，只有顶级开发人员才有希望参与进来。

Ghost 将促进具有不同技能的人的开放和简单的贡献，以帮助平台的成长和发展。因为设计师和开发人员一起解决问题总是会产生更好的最终结果。

Ghost 将通过与用户、设计者、开发者以及在 ghost 基础上建立业务的公司交流来开发。它会邀请那些在这方面做得最多的人来帮助它成长和提高。它会接受并鼓励新的想法。
