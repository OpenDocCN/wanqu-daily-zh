# NDA 过期了——让我们透露一个奇怪的创业公司——特伦斯·伊登的博客

> 原文：<https://shkspr.mobi/blog/2021/07/nda-expired-lets-spill-the-beans-on-a-weird-startup/?utm_source=wanqu.co&utm_campaign=Wanqu+Daily&utm_medium=website>

很久以前，当我还很年轻，而你更年轻的时候...

伦敦遍地都是科技创业公司。我在经营自己的咨询公司。从一家企业跑到另一家企业，试图在这个新奇的“移动互联网”领域找到一份专家的工作。我工作过的一些公司非常棒。其中一些破产了。有些是...仅仅...恶！

前几天，在翻一箱文件时，我发现了一份我签名的旧 NDA。公司早就解散了，NDA 几年前就到期了——所以我认为我可以谈论这个问题。

一家公司——姑且称之为“Fronk”——邀请我来咨询他们正在启动的一个开发者倡导项目。你必须记住，这是此类倡议的早期阶段。DevRel(现在似乎是这么叫的)过去是由产品经理在会议上与其他经理交谈组成的。除了偶尔的披萨驱动的黑客马拉松之外，很少有针对个人开发者的。

这是他们的说辞——我保证我没有夸大其词。我在苏格拉底对话中呈现给你们。

“我们想雇佣像你这样的人去其他公司面试。”

你想付我钱去参加工作面试吗？为什么？

“在面试中，你要宣传我们客户的产品。"

你的客户？

“假设 AWS 想要出售更多的 InfiniDash 许可证。他们付钱给我们，让我们把消息传达给大公司和初创企业。”

怎么会？

“比如说，在面试中，你被问到处理困难局面的时候。我们希望您谈谈使用 InfiniDash 如何让您的团队生活变得更加轻松。”

好...

“或者，如果他们要求举一个例子来说服高层领导，比如说你提出了 InfiniDash，它有多容易操作，所有高管都很喜欢它。诸如此类的事情。”

没错。但是我对 InfiniDash 没有经验。这难道不是一个问题吗？

“我们只会让你去那些还没有使用我们客户产品的公司面试。所以他们不会测试你。我们还将为您支付产品认证费用。"

你怎么给我弄到这些采访？

“哦，我们与大量招聘顾问合作。他们为每个接受面试的体面候选人支付报酬，所以他们给我们提成。我们从我们的技术客户那里得到报酬。是双赢！”

嗯...如果他们给我这份工作呢？

“绝对不能让你接受。告诉他们你已经在别处担任了另一个角色。”

这是...怎么说来着...道德？

“我们的投资者是这样认为的！你不会在简历上撒谎的。你不会接受你不胜任的工作。你不会得到佣金的。每次面试我们都会付给你固定的费用。如果你愿意，你甚至可以把它放在你的其他合同上。”

我没有接受这份工作。我非常担心他们商业模式的道德性和合法性。我也不认为这会有用。显然，他们的投资者也不知道。Fronk 混了几个月——我偶尔会在会议上遇到他们——但后来他们似乎消失了。

可是...

每隔一段时间，我会采访一位候选人，他开始滔滔不绝地讲述以今天的 JavaScript 风格重写一切如何真正帮助了他们的上一家公司。或者他们的老板对这种很酷的新技术能做什么印象深刻。或者为什么他们不能*永远不能*在任何不使用这个特定代码编辑器的地方工作。

这让我想知道弗兰克是否还活着...？
