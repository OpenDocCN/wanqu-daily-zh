# 一位才华横溢的年轻程序员的毁灭性衰落

现在他不工作了，李经常打盹。克里斯汀怀孕七个月了，他们同意在孩子出生后，李将是一个全职爸爸，至少在他想出下一步该做什么之前。与此同时，他们将依靠自己的积蓄和克里斯汀在一家广告技术公司的新工作收入生活。

然而，李的行为变得更加怪异。他每周有几个晚上独自在家看电视。他整天戴着他的帽子，每天，把它拉得越来越低。当克里斯汀分娩时，他在两天的折磨中睡了大部分时间，先是在家睡觉，然后在医院继续午睡。醒来后，他不顾克里斯汀的意愿，坚持不让她接受硬膜外麻醉，这引发了与一名医生的激烈争论。他们的儿子出生后，克里斯汀的妈妈说，医生把她拉到一边，说她从来没有见过一个准爸爸有这样的反应。克里斯汀后来质问他的行为，他向她保证，“我会做得更好。”

在为人父母的最初几个月里，他失败了。他睡了很多午觉。有时她会给他做晚饭，他会拒绝并点一份玉米煎饼。“我当时就想，这是怎么回事？”克里斯汀说。"一切都感觉如此陌生和失控。"

他对他们的儿子不感兴趣，这让她心烦意乱，她决定上演一幕正常育儿的时刻。如果她不能说服他和他们的孩子在一起，她会满足于这种幻想。当李躺在沙发上时，她把婴儿递给他，并抓起手机记录下这一幕。“你站着，你真可爱！”他一边把孩子抱在胸前，一边轻声说道。“你在笑，在出声！”他宠爱他的孩子不到一分钟，然后把他交还给克里斯汀。

她一直试图探究他在想什么，而他一直在回答:“我会做得更好。”他重复的回答让她觉得像机器人一样。这似乎和他现在触摸他们散步时经过的每一棵树的方式一模一样。“我想在内心深处我知道有些事情不对劲，”克里斯汀说。她认为他可能在手术后患上了创伤后应激障碍，或者正与抑郁症作斗争。她一直要求他和她一起去看心理咨询师。最后，当她准备回去工作时，她威胁说如果他不回去，她就离开他。李同意了。

在夫妻治疗环节中，克里斯汀公开哭了，并谈到她的丈夫似乎并不关心他们的新宝宝。“李只是一片空白，”她回忆道，她想知道为什么他不伸手安慰她。突然，他站起来，宣布他忘记归还治疗师办公室的浴室钥匙，然后溜出房间去把钥匙放回去，几分钟后又回来了。

当她的产假结束时，克里斯汀雇了一个保姆并回去工作，但她的警报越来越响。当李整天躺在床上的时候，她开始预约她能想到的每一个专家。“所以我哄他起床，让他上车，确保我的儿子和他的保姆在一起，以某种方式掩护我自己的工作，”然后让他从一个约会到另一个约会。“就这样过了三个月。”

2017 年 3 月中旬，克里斯汀和李去神经科医生那里得到了核磁共振的结果。对 Kristin 来说，似乎神经科医生最初对她的担忧持怀疑态度。李年轻、健康、健谈。
