# iPad 操作系统——asym co

> 原文：<http://www.asymco.com/2019/06/13/the-ipad-operating-system/?utm_source=wanqu.co&utm_campaign=Wanqu+Daily&utm_medium=website>

WWDC 2019 包括一个庞大的发布列表，很难总结。焦点总是在软件更新上，但今年的列表似乎比往常更加详尽。

我认为最重要的是:

1.  WatchOS 现在包括直接访问应用商店。这继续了手表与 iPhone 的分离，其轨迹仍然是可预测的。
2.  听觉健康/噪音应用程序/周期和趋势管理扩展了 iPhone 的健康维护“要做的工作”。
3.  AR & Swift 通过 Reality Kit 和 Reality Composer 以及 Swift UI 扩展平台。
4.  MacOS 通过新的 Apple Music 应用程序、Sidecar iPad 集成、可访问性和“查找我的”Mac 离线信标进行改进。
5.  iPad 和 Mac 开始与 Project Catalyst 共享应用程序。
6.  新的 Mac Pro 和新的 XDR 显示器是高端性能计算市场的巨大飞跃，苹果在这一市场已经落后了几年。
7.  登录苹果是在线身份管理的一个巨大且急需的更新。一件大事。
8.  iPadOS 作为第五代苹果操作系统推出。苹果没有像 Project Catalyst 建议的那样合并 iPad 和 Mac，而是根据它们的差异而不是共性来保留和认可模块化的外形。

如果有一件事是我认为最重要的，那就是 iPadOS 的分拆。我不太清楚这将如何改变 iPad 的命运，但通过宣布自己是一个不同于 iOS 的平台，它标志着 iPad 可以朝着一个新的方向快速发展。iPad 的前景和问题一直是，它是伟大的硬件，却被无法利用它的软件所拖累。

核心应用、界面和连接都受到了限制，这不一定是件坏事。约束造就了一个系统，而 iPhone 的约束让它变得伟大。没有手写笔，没有菜单系统，没有窗口意味着 UX 必须大大简化。

但显然，如果不对这些限制进行重新评估，iPad 就无法发展。上一次发布的 iPad Pro 暗示了即将到来的事情:它有一个全尺寸键盘和一个非常强大的处理器和屏幕。

因此，我们看到了一些新功能，如同一应用程序的拆分视图、文件夹、文档共享、USB 和 SD 驱动器支持、Zip/Unzip、桌面级 Safari 针对触摸进行了优化。下载管理器，自定义字体，新的文本编辑手势(复制/粘贴/浮动键盘)。铅笔延迟改进。所有这些变化都是面向“生产力”或专业用途的。

然而，iPad 不是 Mac。它将保持独立，面向 Mac 的非消费者。事实上，iPad 用户比 Mac 用户多三倍，随着生产力的提高，iPad 用户群很可能会进一步扩大。

iPad 尚未被证明是 iPhone 或 PC 杀手。只是介于两者之间。它不是碗，也不是游泳池。这是一个浴缸。是的，浴缸永远不会像碗一样普遍，但它们有自己的用途，不会被更好的水容器淘汰。

这里的操作系统的想法是有点自负。就内核和核心 API 而言，所有苹果操作系统之间有着巨大的共同点。但是苹果所谓的操作系统不仅仅是核心代码，还包括对开发者的定位。通过给 iPadOS 品牌化，该公司向开发者发出信号，让他们换个角度思考 iPad。

在很大程度上，他们已经做到了。为 iPad 编码总是需要不同的设计方法。但现在，或许苹果正在明确划定界限，强化这种区别。

回到上面的亮点列表，没有一个是“全垒打”，再次强化了公司已经没有大创意的想法。但是击出 8 支安打会产生和两支全垒打一样的结果。这方面的持续改进不容忽视。关于[打磨石头](https://www.engadget.com/2011/11/11/steve-jobss-story-of-the-stones/)直到它们变成宝石，有一些事情要说。
