# **指标对比经验**



Photo by [ChocolateFrogs](https://www.flickr.com/photos/chocolatefrogs/)



几十年前，如果你想建立一个伟大的经历，你会深吸一口气，闭上眼睛，对着你的直觉祈祷。

但是当我们的祖先学会驾驭火的力量时，我们也学会了驾驭测量和分析的力量。我们不再在黑暗中摸索，想知道真的有人在看我每两天发出的那些电子邮件吗？和*我想知道有多少人试用了我们推出的新功能。*我们只需深入我们的数字宝库，就能找到答案。

唉，尽管有了新发现的曙光，存在主义的阴影依然存在。我听到他们在办公室的大厅里争论，在下班后喝酒时嘀咕，在错误的评论线程上输入所有大写字母:

*   我们这样做只是为了衡量标准吗？
*   我们如何在提高数字和做有意义的事情之间取得平衡？
*   还有我个人最喜欢的:*你是那种数据驱动的，还是你其实很在乎用户体验？*

哇哦！带电的文字和热辣的指责！

想要就指标和良好体验进行富有成效的对话吗？以下是我学到的东西。

# 首先，不要把事情框定为“度量与经验”T3】

除了让人想起大预算电影中两个热情的超级英雄之间的对抗，将事情框定为“衡量标准与经验”完全不是开始对话的正确方式。

这就像说“碳水化合物与健康饮食”如果你想在关于营养的讨论中以可信度开始，就不要以此作为开场白。

能够测量东西让你洞察人们在你的产品中做什么。除非你喜欢生活在岩石下，否则拥有更多信息是一件好事。当然，你需要能够分辨出哪些信息是重要的，哪些是不重要的，但是争辩说*拥有更多信息是不好的*这整个概念实际上是站不住脚的。Metrics 不是坏人。

此外，如果你做了一些人们认为有价值的事情，那应该*完全符合让你的成功指标上升*。如果人们使用你的产品的方式没有任何改变，你真的不能说你做了更好的东西。相反，如果你做了改变，人们开始少用你的产品，我不在乎你做了什么，但证据相当清楚，你把事情搞砸了。

最后，指标如此有价值的第三个原因是，它们*有助于将团队团结在清晰和有形的事物周围，他们可以让自己对*负责。从纯粹的后勤角度来看，很难让 50 个人来执行像“创造一个超级惊人的体验！”当然，在你充满力量的演讲后，每个人都可能会挥舞着拳头大喊，“是的！一次超级惊艳的体验！这就是我们所追求的！”但是到了星期一，A 队兴奋地出现了，说“看看这个！我们做了一次超级惊艳的体验！”B 组的反应是“嗯，不，那实际上是废话”，会发生什么？谁是对的？你是如何一致而清晰地定义什么是“超级惊人的体验？”

解决这个问题的一个方法是等级制度。你可以指定公司里的一个特定的人(或一系列人)来评判什么是高质量的，什么不是高质量的。如果你喜欢更少的层级，另一个方法是定义一个可衡量的目标:“一个超高质量的体验意味着 50%尝试过这个功能的人会在一周内回来并再次使用它。”现在，A 队、B 队和其他每个人都清楚地知道他们的目标是什么，以及他们每天离目标有多近。

因此，总而言之，指标是有用的，在高层次上，指标和用户体验并没有陷入永恒的斗争。确保你不会说暗示他们是这样的话。

# 当然，坏事是以“改进度量”的名义做的

就像你可以吃太多充满果冻的甜甜圈，给碳水化合物一个坏名声，度量标准也可以用来证明糟糕的决定。这是可能发生的，因为并不是所有你能衡量的事情都值得去衡量或影响。这种情况也可能发生，因为您不能通过查看单个指标来了解全部情况。通常，您需要一套指标来很好地了解实际发生的情况。

如果你碰巧选择了错误的东西来衡量，并试图移动，你可能最终会做一些实际上对人们的体验有害的事情。观察一些例子:

a)“本来这个故事的点击率是 2%。当我们做出这一改变时，点击率上升到了 5%万岁！”

*有什么问题？*一个故事的点击率并不能充分告诉我们体验是否真的更好。如果我把我网站上的所有链接都改成“点击这里赚 250 美元”会怎么样？当然，点击率会上升！但最终，人们会意识到我实际上并没有给他们 250 美元，然后他们会很生气。他们会停止点击我的链接，卸载我的应用程序，并在应用程序商店给我一星评价，同时写下 FUUUUUUU——全部大写。我的生意失败了，我的生活糟透了。结束了。

b)“人们过去在我的应用上花 5 分钟。现在，在推出我的最新功能后，他们只花 3 分钟。啊哦。”

*有什么问题？*在你的应用上花费的时间实际上是你追踪的一个重要指标吗？看情况。如果你是一个内容目的地，那么很可能是的——你存在的原因是给人们提供阅读/观看/聆听的好东西，所以他们花的时间越多，他们就越有可能发现你提供的有价值的东西。如果你的应用程序是一个工具，比如帮助人们签署数字文件，那就不是。你最好跟踪一些其他指标，比如你的应用程序实际上被用来帮助人们签署文件的次数。事实上，人们可能喜欢这样的事实，现在他们可以更快地签署他们的文档，随着时间的推移，这应该与更多地使用您的服务相关联。

c)“最初，在伊利诺伊州有更多的人使用我们的猫-照片-meme 应用程序，但现在在俄亥俄州有更多的人使用它。”

*有什么问题*？很酷的故事，兄弟。没关系，也不是你需要关注或试图影响的指标。

# 当然，有些重要的事情我们无法轻易或准确地衡量。

如果我们能读懂用户的想法，那么理论上我们就能为他们设计完美的体验。不幸的是，我们并不都是吉恩·格雷，所以我们用我们能衡量的东西来试着做出有根据的猜测，来判断人们关心什么。在这个时代，我们可以衡量的东西有其局限性，记住这一点很重要。仅仅看人们在你的产品中做了什么并不能告诉你:

*   人们对你的产品或其任何特性的喜爱、厌恶或漠不关心的程度
*   随着时间的推移，变化是增加还是减少了人们对你产品的信任
*   你的产品被认为有多简单易用
*   与市场上的其他类似产品相比，人们如何看待你的产品
*   人们最想改变、增加或修正的是什么
*   人们会如何使用你的产品久而久之

有些事情你可以通过定性研究或调查和民意测验来获得，但这些方法都不完美(还记得英国退出欧盟预测民意测验吗？)即使在你可以准确衡量广泛情绪(如品牌信任)的情况下，也很难知道具体的变化产生了什么影响(例如，那个标志或视觉重新设计是否改善了人们对我的品牌的看法？)

不能准确地度量上述内容意味着存在度量标准让我们失望的情况。观察以下示例:

1.  **了解复杂性的成本**:每次你给你的应用程序添加一个新功能，你跟踪的指标很可能会变成正数(毕竟，在没有人使用 X 之前，现在有更多的人使用 X，人们似乎也没有减少使用 Y 或 Z，所以总体来说这感觉像是一个胜利。)然而，如果你不断地添加功能，在某个时候，你最终会得到一个被认为是混乱和臃肿的产品。然后，突然，一些闪亮的新竞争对手将获得快速牵引，因为每个人都喜欢“我爱 Q！只是*这么简单罢了。”选择的悖论和认知复杂性的代价是真实的。我们只是还没有想出如何精确地测量它们。*
2.  **了解品牌的力量**:当苹果或耐克推出一款新产品时，很多人会倾向于购买它，即使他们没有做调查，因为他们过去对这个品牌有过很棒的体验。如果一个叫 Pear 或 Sike 的新贵带着一个类似的产品上市，情况就不一样了。在高层次上，我们都知道并理解这一点。但是，很难把品牌的力量量化，变成一个可以每天追踪的数字。很难知道一家公司做出的所有数千个决定如何影响该品牌，以及权衡这些权衡的成本和收益是什么。
3.  大赌注的力量:没有指标能告诉你赢得未来需要什么样的大胆举措。想象一下 2008 年，智能手机刚刚开始出现的时候。如果你看一下你网站的指标，你会发现很小一部分流量来自智能手机。你可能已经得出结论，非常实际，你真的不应该投资太多的建设移动，因为它是你的观众这么小的一部分。今天，我们意识到那些在移动领域下了大赌注并获得巨大回报的人的远见卓识。没有对当前行为的检查能准确地告诉你需要往哪个方向跳。战略性的长期规划仍然需要和以前一样的东西:相信你的直觉。

# **良好度量标准卫生的一些经验法则:**

这些是我在寻求变得越来越自律于良好的目标设定和衡量策略的过程中最大的收获。

*   要评估产品与市场的契合度，就要看忠诚度。不要看使用你的产品或功能的绝对人数(这可能会被你的推广力度等因素扭曲。)留存率与你的产品是否有价值最相关，因为它告诉你尝试过它的人是否足够喜欢它，愿意回来再次使用它。
*   为了优化增长，了解你的漏斗。为了让人们成为你产品的固定用户，他们必须通过一系列障碍。首先，他们必须知道你的产品。第二，他们必须有足够的兴趣去查看。第三，他们必须转换(下载一个应用程序，填写表格，确认电子邮件等。)第四，他们必须在你的产品中做足够多的工作，以理解为什么它可能在他们的生活中有价值。第五，他们必须记得回来。每走一步，你都会失去一些人。如果你能跟踪和测量这个损失率，你就能开始计算出在哪里集中你的努力来减少漏斗的泄漏。
*   **找出哪些指标真正重要，并关注那些**。很容易进入跟踪一切的状态(因为你可以)，你有一个装满数字的仪表板，感觉它们都应该是绿色的。认识到大多数事情都不重要，只有一小部分真正重要。不要浪费时间谈论不重要的东西，也不要让一些不太重要的指标上升或下降。
*   要找出跟踪的最佳指标，使用魔术棒技术。问你自己:“如果我可以挥动魔杖，了解我在世界上的用户的任何信息，我最想知道什么来告诉我我的应用程序是否会成功？”即使你的答案不是你实际可以衡量的(“我的应用程序是否建议了我的用户认为有价值的推荐？”)，这是一个有用的工作起点。(“好吧……所以我不能问每个用户简历是否有价值……但如果有价值，我可能会看到他们更多地保存或分享简历，他们可能会花更多的时间阅读简历，等等，等等。”)
*   **不要在不了解指标目标的情况下就接受它**。我怎么强调都不为过:你和你的团队达成一致的目标将会对你的工作产生巨大的影响，所以一定要相信它们。不要接受表面价值的指标目标。问为什么。思考它们是否有意义，以及它们会激励什么行为。有没有这样的情况，有些事情看起来像是一个好的决策，但是指标却不动？相反，有没有这样的情况，你可以想象度量标准上升了很多，但不相信产品真的更好？如果是这样，另一个指标(或一组指标)会更好地跟踪真正重要的东西吗？
*   **以怀疑的态度看待数据，提出对策**。如果数据向你展示了看起来不错的结果，问问你自己:“我还能看到什么让我相信这些结果并不像看起来那么好？”这些被称为对策，每个成功指标都应该有一些对策。(比如不看点击率不看快速反弹次数，不看一个产品的销售数字不看有多少退货或者取消等等。)最好是多疑地解读数据，这样你就能迅速发现自己的错误并调整策略。不要陷入确认偏见的陷阱，你只是在寻找证明你的直觉是正确的信号。
*   使用定性研究来找出原因。告诉你人们做了什么的定量数据最好与让你洞察人们感受的定性研究相结合。进行可用性测试，利用焦点小组，并进行调查，以了解你所看到的行为背后的*为什么*。
