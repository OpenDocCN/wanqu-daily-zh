# 生活在不完全互联网的美丽泡沫中

> 原文：<https://www.buzzfeed.com/annehelenpetersen/the-not-quite-internet?utm_source=wanqu.co&utm_campaign=Wanqu+Daily&utm_medium=website>

在大学的第一学期，有一个家伙占据了我大部分的思想。他住在楼上，是整个宿舍里唯一的一间。他有着栗色的头发和攀岩者肌肉发达的手臂。当我到达校园的时候，他是我第一个看到的人，第一个真正看到的人。他在帮某人在墙上挂挂毯。当然:这是 1999 年。

他的宿舍里有一长墙的 CD，他向我介绍了钢琴的配乐，我从连接 200 人宿舍中每个音乐库的共享以太网上下载了这些配乐。他从不接宿舍电话。他的 AOL 即时通讯用户名是他的首字母(ATM)加上他的出生年份(80)。他从不聊天，但总是在线，他的黄点用不活跃来戏弄我。

引起他注意的真正方法是通过电子邮件，用我们在校园里发送电子邮件的方式:远程登录，这是我一直想要的，但只是断断续续地收到。它黑白分明，极其简单，只有一个字母的删除、转发和回复命令。你不能附上照片，至少不能没有一个伟大的待办事项；你不能用粗体、斜体或下划线。最好的特征是一个广为流传的秘密:手指。

你可以在 telnet 主屏幕上输入命令，然后输入六个字母的用户名(姓的前四个字母+名的首字母+中间名的首字母)。然后屏幕会提供一个害相思病的大学新生可能想要的最好/最糟糕的东西:这个人最后一次登录他们的电子邮件账户的日期、时间和地点。

实际上:跟踪。实际操作:*只是想知道*是否有人打开了你发送的那封精心制作但非常随意的三句话邮件。(通常:还没有。)但是手指不仅仅是用来迷恋的；你可以用它来确定某人在哪个计算机实验室，或者确定某人是否在休息后回到校园，或者，正如我的朋友在大三时所做的那样，确定你约会的那个人实际上并不仍然住在泰国的海滩上，而是使用蒙大拿州农村某处的 IP 地址！

手指是当前技术的先驱，如短信、阅读收据和 GPS 跟踪，这些技术现在完全构建了我们的日常交互。但是，1999 年是一个没有着落的地方，只有足够的数字工具来影响——但不能改变——我们对关系、互动或友谊的模拟理解。
