# 90:9:1——技术不断创造的奇怪比率|技术|卫报

> 原文：<https://www.theguardian.com/technology/2015/dec/12/ratio-technology-mozilla-firefox-os-90-9-1?utm_source=wanqu.co&utm_campaign=Wanqu+Daily&utm_medium=website>

操作系统、浏览器和搜索引擎有什么共同点？关键球员之间的比例似乎是 90:9:1。一个玩家占优；然后其他人得到最少的份额。

以移动操作系统为例:本周，Mozilla 基金会[终止了 Firefox 操作系统](http://techcrunch.com/2015/12/08/mozilla-will-stop-developing-and-selling-firefox-os-smartphones/)——这款移动操作系统本可以用基于 HTML 的应用取代本地应用——这是移动操作系统大战中最后的垂死挣扎。现在有三个主要平台——谷歌的 Android、苹果的 iOS 和微软的 Windows Phone——全球出货量目前分别以大约 85:14:1 的比例在[运行。](http://www.idc.com/prodserv/smartphone-os-market-share.jsp)

现在看看桌面操作系统的销售:在最近一个季度，微软的 Windows、苹果的 Mac OSX 和可能安装 Linux 的“自建”机器之间的比率大约为 91:8:1。

这很奇怪地让人想起了[“1%法则”](https://en.wikipedia.org/wiki/1%25_rule_(Internet_culture))——早在 2006 年观察到的一条经验法则，它指出，如果你有一个 100 人的在线互动小组，那么一个人将产生一些内容，九个将提供反馈，90 个将简单地消费它。(研究广泛证实了这一原则。)

在个人电脑和智能手机领域，90:9:1 的比例每次都是在早期阶段的寒武纪大爆发之后出现的，因为不同的参与者争相利用并试图占据主导地位。

在个人电脑上，有 DR-DOS、MS-DOS、CP/M、GEM、Amiga，以及后来的 BeOS 和 Linux(仅举几例)。

智能手机上有 Symbian、Meego、Windows Mobile、Asha、BlackBerry 7、BlackBerry 10(现在实际上已经死了，因为 BlackBerry 已经转向 Android)，当然还有 Firefox OS，以及 Jolla(看起来快要死了)和三星的 Tizen(还没有)。

在这两种情况下，爆炸都是由组件的日益商品化推动的，但因为无论是在 PC 还是智能手机上维护操作系统和应用生态系统都是一项昂贵的业务，所以出现了一次洗牌。例如，黑莓放弃了 BB10，因为首席执行官陈志强暗示，为不同的芯片编写驱动程序成本太高；通过转向安卓，这一挑战被转移到了谷歌。

现在看看搜索引擎。根据 Statcounter 的数据，在欧洲，谷歌获得了近 90%的桌面搜索。微软的必应获得 7.3%。雅虎以 3.5%的份额排在最后，剩下的 1.2%归一大批“其他人”所有。根据同一数据集，移动领域的用户中，94%是谷歌，3%是雅虎，2%是必应，0.3%是“其他”。

浏览器怎么样？在 2002 年至 2003 年间，微软的 ie 浏览器占据了主导地位:随着网景的衰落，在苹果的 Safari 和 Firefox(浏览器)分别于 2003 年和 2004 年到来之前，它的各种版本以 95%的份额达到顶峰。

火狐成为“9%”的玩家，并迅速取得进展，在 2009 年 11 月达到顶峰，约占浏览器使用量的 24%。曾经有段时间，新版火狐浏览器的发布在[成了互联网事件](http://news.bbc.co.uk/1/hi/technology/7457503.stm)。

但是就在 ie 浏览器没落的时候，谷歌 Chrome 于 2008 年 9 月问世，并迅速成为一个重要的玩家。根据 Statcounter 的数据，现在桌面浏览器市场正在发生剧变；Chrome 正在接管，而 Firefox 和 Internet Explorer 正在萎缩。

你越是在网络环境中寻找它，一旦市场成熟，某种 90:9:1 的比例似乎就越频繁地出现。当然也有变化:例如，iPhone 用户比经验法则预测的略多，Mac 用户略少，而浏览器市场的巨大变化是因为 Chrome 有一个几乎可以与微软 Windows 相媲美的推广平台:它是唯一一个在谷歌首页做广告的产品。

但是为什么浏览器和搜索引擎要接近这个比例呢？它们不需要商品制造；它们的部署和维护成本对消费者来说是隐藏的，消费者只需要使用它们。但它们相对容易建立，使用它们的网络效应将有助于那些做得好的人。

值得注意的是，所有这些产品都是网络世界的一部分，每一个都是相互排斥的活动——也就是说，你一次只能使用一个。没有人(在一定程度上)会同时购买或使用两部智能手机。几乎没有人使用两个桌面操作系统。一次只能在一个搜索引擎上进行搜索。您一次只能使用一个浏览器。

有一个值得注意的技术硬件领域 90:9:1 显然不适用:游戏机。在上一代游戏机(任天堂 Wii v PlayStation 3 v Xbox 360)中，总销量分别为 1.012 亿台、8590 万台和 8490 万台——几乎是平分秋色。当代人继续偏离这一规则。一种可能的解释是，游戏机从未有过“商品化”的时刻:每台游戏机只有一家制造商，而个人电脑和智能手机却有多家制造商。因此，只要竞争对手的平台能拿出令人信服的内容，游戏机就永远不会看到围绕这个或那个平台的整合。(世嘉在这方面的无能导致了它作为硬件供应商的没落。)

还有一个领域这个比例不成立:社交网络。你可以属于并使用脸书、推特、Instagram、Snapchat 和其他几十个网站，并向每个网站发送相同的内容。在信息领域，你可以使用 WhatsApp 和 BBM，谷歌视频聊天工具和 Skype。这些网络和应用程序的用户数量并不遵循这一规则，即使是日常用户也是如此。

在非排他性方面，社交网络似乎与那些基于任务的产品——搜索引擎、浏览器、个人电脑和智能手机——非常不同。

那么 90:9:1 对于产品来说是不是一个无可争议的定律呢？这还有待观察；但它可能是未来互斥活动的指南。“出租车”公司(优步、Lyft、Hailo)会走同样的路吗？还是机器学习算法部署？几年后也许值得再来看看。
