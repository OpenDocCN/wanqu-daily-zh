### 审查制度

* * *

审查总是伴随着政府的痕迹。

在世界上的许多国家，政府倾向于阻止他们的公民访问互联网上的某些域/IP。有人说，“它是为了保持文化和道德的健康！”。他们封锁色情内容。即使在美国，我们也可以发现某些域名被屏蔽，无法访问，比如那些包含 CP 的域名。

然而，在伊朗(和大多数其他国家)情况并非如此！

他们屏蔽了很多东西。我们无法访问英国广播公司、福克斯新闻频道或美国之音等媒体网站，Twitter 或脸书等社交媒体网站，Telegram、微信、Kik 或 SnapChat 等信使，或 YouTube 等服务，甚至一些 SourceForge 子域名——你可能不相信！

**为什么？**因为极权政府就是这样生活的。虽然他们出售 VPN 和代理，他们监视他们的人，他们很容易地消除社会中不需要的元素。你可能永远也不会知道你几天前刚拜访过的朋友发生了什么事！

过滤/网络审查是一个非常有利可图的行业！对政府和私营公司来说都是如此，因为他们得到的报酬比这里典型的 IT 公司多得多！

有时，你会看到他们对一些服务进行大量过滤，例如电报，以推广他们自己的服务！有时，他们会对某项服务说些花里胡哨的话，例如:“这是以色列的！”我是说，那又怎样？在与伊拉克的战争中(1980-1988)，你们没有使用过以色列的武器吗？那时候不算事，现在算了！

有时候，它们会破坏 SSL/TLS！我最近看到类似的事情:[哈萨克拦截 HTTPS 交通](https://www.reddit.com/r/privacy/comments/cfglcu/update_kazakhstan_intercepting_https_traffic/)，正式 MITM 化他们的公民！

作为一名开发人员，什么影响了我？要知道，他们的系统并不完美。有时候你会失去一些必须的服务，比如 GitHub！为什么？因为过滤程序由于意外或错误配置而断开了与 GitHub 的连接！有时，您会想到完全中断的 SSL 握手！有时，SSL 握手可能会持续很长时间，并在结束时中断！有时，CloudFlare 会因为 TCP 数据包损坏而拒绝服务！

当你要学习新的东西时，这是一件痛苦的事情；没有 YouTube，没有 Reddit，没有 Medium，痛苦地在网上冲浪等等。

我不说低质量上网，像家里 WiFi 200 kb/s！

### 制裁

* * *

如果你关注世界新闻，你肯定听说过因伊朗核计划而对伊朗实施的制裁。虽然它不仅仅局限于物理和核的东西。许多公司服从了美国对伊朗的制裁。

没有人真正关心人民会发生什么。人民一文不值。他们就是这么认为的。你说，伊朗政府和国际机构，还有联合国。

由于缺少药品，人们正在死去。人们在挨饿。经济体系正在崩溃，政客和他们的孩子都在国外！他们都不知道街上发生了什么。

我们这些 IT 男会怎么样？ *制裁！*

我们错过了什么？ *福斯！*我们正在错过*免费开源软件！*你看到了吗？

几个月前，懈怠队决定加入制裁。他们只是删除了每一个他们发现是伊朗人的用户！没有事先通知！许多人在 Slack 上丢失了他们的数据，但没有人打算做任何事情！他们有一些伊朗用户在国外生活了很多年，甚至很久都没有去过伊朗了，但是他们的账户和其他人一起被删除了！在[T](https://twitter.com/imaN_Khoshabi/status/1075648863732752384)[w](https://twitter.com/blockchaindildo/status/1075834524649619460)[I](https://twitter.com/MaziR1989/status/1075660298835714048)[T](https://twitter.com/GamerGeekNews/status/1075849114972086272)[T](https://twitter.com/verge/status/1075976412224475141)[e](https://twitter.com/jabdi/status/1076146349719150595)[r](https://twitter.com/lrz/status/1075852522814926850)上有很多人在抱怨这件事。[A](https://twitter.com/tha_rami/status/1076924288639340545)T16】nT18】dT20】eT22】vT24】eT26】nT28】m[o](https://twitter.com/thehill/status/1075825437417312258)[r](https://twitter.com/kfvahedi/status/1076073199333449728)[e](https://twitter.com/__dc0d__/status/1076524955129524224)。(我猜你[得到了](https://twitter.com/search?q=slack%20iran%20since%3A2018-12-18%20until%3A2019-01-01)的想法。)

由于经济银行制裁，我们不能轻易拥有万事达卡/维萨卡。因此，我们不能创建 AWS 帐户，我们不能在亚马逊/Ebay 上购买任何东西，我们不能拥有谷歌商店控制台，我们不能使用(几乎)任何企业服务。

[Android dev](https://developer.android.com) 返回 HTTP 403， [Docker docs](https://doc.docker.com) 返回 HTTP 403， [bintray](https://bintray.com) 返回 HTTP 403，[Schema.org](https://schema.org)返回 HTTP 403， [GitLab](https://gitlab.com) 返回 HTTP 403 等等。([有一长串可用的](https://gist.github.com/alibo/dfd7c258bcc44a0e8c9f7c5bfd3bd2c3#file-iran-sanctions_blocker-sites-csv))

### 开发体验

* * *

你可能不知道我在说什么。想象一下，你应该用你对公司一无所知的新技术来建造一些东西。第一步是找到技术文档，并尝试找出如何让它工作。

在谷歌搜索它的名字后，你会发现许多相关链接，包括官方文档的链接。你点击链接，突然弹出一个令人讨厌、丑陋、愚蠢的页面，上面有一个大字“你被美国制裁，我们无法为你服务”。

你回到谷歌搜索结果，试图找到别的东西。你在那里看到 YouTube 和 Medium 链接，但你知道它们在你居住的地方被审查或不可用，你也不能使用它们。

一个指向 StackOverflow 问题的链接吸引了你的注意力，你点击了它。这个问题是关于某种可能先进的技术，而你根本不知道他们在说什么！你没有选择去谷歌搜索结果，第二页。第 2 页到第 10 万页没有相关链接！

你去找你的老板，告诉他们，“这项技术在这里行不通。找点别的或者取消项目”。第二天，你在别的地方找新的职位！

### 我们如何生存？

* * *

我们必须绕过制裁和审查。

#### HTTP 代理

* * *

代理是(目前几乎)可行的解决方案之一。

找到一个合适的代理并不容易，使用任何代理都不安全，而且代理并不能涵盖一切。

有一个名为 [FOD](https://github.com/freedomofdevelopers/fod) 的个人社区资助代理服务器，只接受某些[某些域](https://raw.githubusercontent.com/freedomofdevelopers/fod/master/domains)，拒绝其他任何域。对每个人来说，配置他们系统以使用那些特定域的代理不是一件容易的事情。而且他们并没有覆盖*所有的*域名，每当有人发现一些域名没有被覆盖并通知服务器所有者时，这个列表就会变得越来越长。另一个限制是，这个代理不包括审查服务。

#### DNS 代理

* * *

有一个由[沙里夫理工大学](http://www.en.sharif.edu/)运营的 [DNS 代理](https://shecan.ir)，它只能绕过制裁。但是既然是政府推荐的，听起来就不像是一个安全的选择！在首页，他们有教程，让用户在操作系统上设置他们的 DNS 服务器指向代理服务器，这意味着代理服务器将解决你所有的 DNS 查询！我个人不喜欢第三方(这是政府推荐的)，窥探我所有的 DNS 查询。我不会把我的 DNS 服务器从`1.1.1.1`换成他们的！

#### 公共 vpn

* * *

不是安全的，而是可行的解决方案。

免费和付费 VPN 大多由政府推动。他们确实会窥探每一个请求，并调查他们发现的任何可疑的事情。明显危及安全和隐私。

任何其他非政府 VPN 被政府禁止，你需要每周寻找 2-3 次新的工作 VPN！

#### 专用 vpn

* * *

安全和工作，但昂贵的解决方案。

有一些你可以使用的私人 VPN，或者你甚至可以运行你自己的，它们是完全安全和隐私友好的，但是它们也很贵！不是所有人都能购买/服务私有 VPN。

#### TOR 项目

* * *

最可靠但不是最好的解决方案。

TOR 是一个不可禁止的隐私解决方案，它显然绕过了制裁和审查。但是它有一个很大的问题，不是所有的服务器都喜欢从 TOR 获取流量。例如，当你通过 TOR 访问它的服务器时，CloudFlare 会很烦人。谷歌让你解决许多难题。由于 TOR 奇怪的 TCP 流量，一些服务器根本不提供任何服务。

此外，伊朗政府试图限制连接 TOR，虽然他们永远无法完全阻止 TOR(除非他们完全阻止外国服务器！)，但是它们会阻止您直接连接到 TOR 网络。obfs 桥来了！然而，你需要先去 bridge.torproject.org。

#### 我如何生存

* * *

我混合使用以上所有方法！

我已经配置了 [bind/named](https://bind9.net) 通过 [shecan](https://shecan.ir) 和 [privoxy](https://privoxy.org) 代理几个特定的域查询，通过 [FOD](https://github.com/freedomofdevelopers/fod) 到 FOD 隧道所有支持的域，其他的通过 TOR。

我也使用 GitHub gists 在信任的人之间保存和传播 TOR 桥梁。

* * *

我只是想写一写，做人们每天做的事情而不去想它有多难！我打赌你无法想象没有 YouTube 的互联网。您从未经历过在没有事先通知的情况下突然丢失数据的情况！你无法想象在重重审查和制裁下生存有多痛苦。你不知道等待 VPN 连接超过 10 分钟，然后被拒绝是什么感觉！

令人痛心的事实是“这一切的发生仅仅是因为我们生活在伊朗，没有人关心那里的人民。连人都不是！”
